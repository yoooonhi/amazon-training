# 高级4.2 SD 再营销

> **高级4：高阶广告与流量组合**
> **本课目标**：掌握 Sponsored Display（SD）的再营销受众设置、转化窗口选择、lookback 配置，用比 DSP 门槛低得多的方式做"看过没买"的召回。

## 这节课解决什么问题

上节课你学了 DSP 能做再营销（召回看过没买的人），但 DSP 预算门槛太高，多数中小品牌用不起。好消息是——**SD（Sponsored Display）也内置了再营销功能**，而且门槛低得多（小预算就能跑），是中小品牌做"召回"的高性价比工具。

问题在于：很多卖家用了 SD，但只把它当"投商品/投类目"的展示广告用，完全没碰再营销受众功能。或者开了再营销但转化窗口、lookback 设置一塌糊涂——要么窗口太短（7天）抓不到长决策周期买家，要么太长（90天）烧了一堆不再相关的人，效果出不来就放弃。

这节课讲清 SD 再营销的受众设置、转化窗口（conversion window）、回溯期（lookback）这三个核心配置怎么选，以及它和 DSP 再营销的分工。这是中小品牌做"召回"的实战课。

## 核心知识

### SD 广告的三种定向方式

先搞清 SD 不只有再营销，它有三种定向方式：

```
SD 三种定向：

1. 商品定向（Contextual / Product Targeting）
   → 投放到「特定商品/品类」的详情页
   → 类似SP的商品定向
   → 触达「在看相关商品」的人

2. 受众定向（Audience Targeting）—— 本课重点
   → 投给「特定行为特征」的人
   → 分两类：
     a. 再营销受众（Retargeting）：和你互动过的人
     b. 兴趣受众（Lifestyle/Interest）：没互动但有相关兴趣

3. （部分市场）类目 + 受众组合
```

> 关键认知：**SD 的再营销受众定向，是中小品牌做「召回」的高性价比工具。** 它不像 DSP 要数万美金月预算，SD 可以小预算起步，是 DSP 再营销的"平价替代+先行验证"。

### SD 再营销受众的两大类

```
SD 受众定向 → 再营销受众，分两大类：

1. 浏览再营销（Views Remarketing）
   → 触达「过去X天内看过你Listing详情页」但没买的人
   → 召回「有过兴趣但没转化」的人

2. 购买再营销（Purchases Remarketing）—— 更精准
   → 触达「过去X天内买过你产品」的人
   → 做复购/交叉销售（推新品/配件/补充装）

→ 两类目的不同：
  浏览再营销：召回流失，推动首次转化
  购买再营销：复购和交叉销售，提升LTV
```

> 实操要点：**新手做 SD 再营销，优先做「浏览再营销」**——召回看过没买的人，这批人转化率最高（已经表达过兴趣）。购买再营销适合做复购品类（耗材/配件/补充装）。

### 核心配置一：转化窗口（Conversion Window）

转化窗口是亚马逊追踪"点击广告后多久内的转化算这次广告的功劳"：

```
转化窗口选项（SD通常有这些档）：
  - 7天
  - 14天
  - 30天

含义：
  7天窗口：买家点击你SD广告后7天内买的，算这次广告转化
  14天窗口：14天内买的都算
  30天窗口：30天内买的都算

选择逻辑：
  - 决策周期短的产品（日用品/低价）→ 7天或14天够
  - 决策周期长的产品（高客单/耐用品）→ 30天更准
  - 评估广告真实效果要选合适窗口，太短会低估长决策产品的效果
```

> 红线提醒：**转化窗口选错会误判广告效果。** 高客单耐用品用7天窗口，会以为广告"不出单"——其实买家还在考虑，30天后才买。所以高客单产品要看30天甚至更长的归因数据。

### 核心配置二：Lookback（回溯期）

Lookback 是再营销里**最关键的设置**——它决定"多久之前浏览/购买过的人，还会被你再营销触达"：

```
Lookback 含义（以浏览再营销为例）：
  Lookback 14天 = 触达「过去14天内看过你Listing但没买」的人
  Lookback 30天 = 触达「过去30天内看过但没买」的人
  Lookback 90天 = 触达「过去90天内看过但没买」的人

Lookback 长短的权衡：
  短Lookback（7-14天）：
    + 受众新鲜，意向强，转化率高
    - 受众池小，量少
    适合：决策周期短、求高转化

  长Lookback（60-90天）：
    + 受众池大，量大
    - 受众陈旧，意向弱，转化率低
    - 可能烧钱在「已经买了别人的/不再感兴趣」的人
    适合：决策周期长、求规模

  推荐：
    新手起步用 14-30天（平衡转化和规模）
    按产品决策周期调整（短决策用短、长决策用长）
```

> 关键纪律：**Lookback 不是越长越好。** 90天 lookback 看着受众池大，但很多人早就不感兴趣或买了竞品，烧钱转化低。先用中等 lookback（14-30天）测出效果，再按数据调整。

### SD 再营销的完整设置流程

```
SD 再营销活动设置SOP：

1. 选目标
   → 召回浏览未买（浏览再营销）还是复购（购买再营销）

2. 选ASIN范围
   → 哪些ASIN的浏览者/购买者纳入受众
   → 通常选核心转化好的ASIN（受众质量高）

3. 设置Lookback
   → 起步14-30天，按决策周期调

4. 设置转化窗口
   → 短决策7-14天，长决策30天

5. 出价和预算
   → 再营销受众通常比冷启动CPC高（意向强）
   → 但ROI更好，因为受众质量高
   → 小预算起步测，再放量

6. 广告素材
   → 再营销广告可定制文案（「看过没买？给优惠」）
   → 自定义 banner/标题

7. 监控和优化
   → 看再营销受众的转化率（应高于冷启动）
   → 调lookback（转化差就缩短）
   → 加预算（效果好就放量）
```

### SD 再营销 vs DSP 再营销：分工

| 维度 | SD 再营销 | DSP 再营销 |
|---|---|---|
| 门槛 | 低（小预算起步） | 高（月预算数万美金起） |
| 受众类型 | 浏览/购买再营销 | 更丰富（含lookalike/兴趣） |
| 广告位 | 站内为主 | 站内+站外（Twitch等） |
| 素材定制 | 有限 | 高度定制 |
| 适合阶段 | 中小品牌/验证阶段 | 成熟品牌/规模扩张 |

> 关键认知：**SD 再营销是 DSP 再营销的「先行验证 + 平价替代」。** 中小品牌先用 SD 再营销验证"召回有效"，跑通了有规模再升级到 DSP 做更大受众池和站外触达。

### SD 再营销的常见踩坑

```
踩坑1：没开再营销，只用商品定向
  → 错过召回看过没买的人（最高ROI受众）
  → 正解：至少开浏览再营销测试

踩坑2：Lookback太长（90天）烧钱
  → 受众陈旧转化低
  → 正解：起步14-30天，按数据调

踩坑3：用ACOS苛求再营销
  → 再营销受众贵但转化好，短期ACOS可能不低
  → 正解：看LTV和总转化，不只看单次ACOS

踩坑4：受众池太小硬跑
  → 流量低的产品浏览者少，再营销受众池小
  → 正解：受众池小先扩流量（SP先做起来），再营销才有米下锅

踩坑5：素材不针对再营销场景
  → 用通用素材，没体现「召回」的定制信息
  → 正解：再营销素材针对「看过没买」定制（优惠/补充信息）
```

> 红线提醒：**再营销要有"米"下锅——你的 Listing 必须先有流量。** 浏览者少，再营销受众池就小，跑不出量。所以 SD 再营销是"流量做起来之后"的优化工具，不是冷启动工具。

## 实操要点

- **SD 再营销是中小品牌做召回的高性价比工具**：门槛比 DSP 低得多，小预算起步，是 DSP 再营销的先行验证。
- **新手优先做浏览再营销**：召回看过没买的人，这批人转化率最高，购买再营销留给复购品类。
- **Lookback 起步 14-30 天**：不是越长越好，90天烧陈旧受众转化低，按产品决策周期调。
- **转化窗口按决策周期选**：短决策7-14天，长决策30天，选错会误判广告效果。
- **再营销要有流量基础**：浏览者少受众池就小，先把 SP 流量做起来再营销才有米下锅。

## 动手练习

为你一个有稳定流量的核心 ASIN 设计 SD 再营销方案：(1) 决定做浏览再营销还是购买再营销（看产品是首次转化型还是复购型）；(2) 设置 lookback（按决策周期，起步14-30天）和转化窗口；(3) 写一段针对"看过没买"场景的广告素材文案；(4) 定监控指标和优化触发条件。重点练的是**用配置和场景做再营销，而不是开了就不管**。
