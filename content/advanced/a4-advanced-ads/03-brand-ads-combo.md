# 高级4.3 品牌广告组合

> **高级4：高阶广告与流量组合**
> **本课目标**：能用 SB（Sponsored Brands）、SBv（Sponsored Brands Video）、SD（Sponsored Display）做全漏斗配合，根据品牌不同阶段设计不同的品牌广告策略。

## 这节课解决什么问题

中级课你接触过品牌广告（SB/SBv），知道它们长什么样。但很多卖家用品牌广告的方式是"各自为战"——SB 投一组、SBv 投一组、SD 投一组，互相不配合，也不知道哪个该在上层做认知、哪个该在下层做转化，预算分配拍脑袋。

问题在于：**品牌广告的真正威力在于「组合」。** SB/SBv/SD 各自在漏斗里承担不同角色，组合起来才能形成从认知到转化的完整链条。单打独斗每个都打折，组合起来才是全漏斗打法。

这节课讲清 SB/SBv/SD 三种品牌广告的定位、全漏斗配合方式、不同阶段的品牌广告策略。它是高级4.4"全漏斗流量打法"的前置课——把单个广告类型搞清角色，才能拼成完整漏斗。

## 核心知识

### 三种品牌广告的定位

先搞清 SB、SBv、SD 各自的角色和广告位：

| 广告类型 | 形式 | 主要广告位 | 漏斗角色 | 强项 |
|---|---|---|---|---|
| **SB（Sponsored Brands）** | 品牌标题+logo+产品集/自定义图 | 搜索结果顶部/中部/底部 | 上层认知 + 下层收割 | 品牌曝光、品牌词防御 |
| **SBv（Sponsored Brands Video）** | 短视频广告 | 搜索结果中（自动播放） | 中层考虑 | 视觉冲击、产品演示、高CTR |
| **SD（Sponsored Display）** | 展示广告（含再营销） | 详情页/搜索页/站内外 | 中层考虑 + 下层再营销 | 触达竞品详情页、召回流失 |

> 关键认知：**SB/SBv/SD 不是「同一种广告的三个版本」，是漏斗不同位置的武器。** SB 偏认知和品牌词，SBv 用视频打动考虑阶段买家，SD 抢竞品流量和召回流失。组合才是全漏斗。

### SB（Sponsored Brands）：品牌曝光的主力

```
SB 的核心价值：

1. 品牌曝光（搜索结果顶部黄金位）
   → 买家搜某个词，你的品牌+产品集最先出现
   → 建立「这个品类有我」的认知

2. 品牌词防御
   → 投自己的品牌词，防止竞品蹭
   → 竞品投你品牌词时，你的SB守住顶部

3. 品类词卡位
   → 投品类通用词（如「yoga mat」）
   → 在品类搜索的顶部占位

4. 送流量到Store（品牌旗舰店）
   → SB可链接到你的Store，做品牌深度展示

SB 的三种形式：
  - Product Collection：产品集（多个ASIN）
  - Store Spotlight：指向旗舰店
  - Custom Image：自定义图片（品牌创意）
```

> 实操要点：**SB 是「品牌存在感」的广告。** 买家搜品类词看到你反复出现，会形成"这是这个品类的主要品牌"的认知。这对品牌建设很关键，但要注意 SB 不只看直接出单，还要看品牌词搜索提升。

### SBv（Sponsored Brands Video）：考虑阶段的利器

```
SBv 的核心价值：

1. 视频自动播放（搜索结果中部）
   → 比静态图片更有冲击力
   → 适合产品演示（怎么用、效果如何）

2. 高CTR（视频天然吸引眼球）
   → 通常比静态SB的点击率高

3. 考虑阶段推动
   → 视频展示产品卖点/使用场景
   → 推动买家从「知道」到「想要」

SBv 适合的产品：
  ✓ 有视觉演示价值的产品（怎么组装、效果对比）
  ✓ 功能复杂需要展示的产品
  ✓ 强体验类（美妆/家居/运动）
  ✗ 纯功能/低视觉差异的产品（视频价值小）
```

> 实操要点：**SBv 不是「所有产品都要做」。** 它对有视觉演示价值的产品效果最好。如果你的产品是"看不出区别的标品"，视频的边际价值有限，先把 SB 做好。

### SD（Sponsored Display）：抢竞品流量 + 召回

```
SD 的核心价值（两个方向）：

1. 商品定向：抢竞品详情页流量
   → 你的广告出现在「竞品Listing详情页」
   → 把在看竞品的买家拉过来
   → 适合「我有差异化优势」的产品

2. 受众定向（再营销）：召回流失（高级4.2讲过）
   → 召回看过没买的人

SD 的独特优势：
  → 能投到竞品详情页（SP也能，但SD展示位不同）
  → 再营销能力（中小品牌的召回工具）
  → 触达「不在搜索」的人（详情页浏览者）
```

### 全漏斗配合：SB/SBv/SD 各在哪个位置

```
全漏斗视图（品牌广告组合）：

┌─────────────────────────────────────┐
│ 上层：认知（Awareness）             │
│   SB 品牌词/品类词 → 品牌曝光       │
│   （+ DSP 站外认知，高级4.1）       │
├─────────────────────────────────────┤
│ 中层：考虑（Consideration）         │
│   SBv 视频 → 产品演示、推动想要    │
│   SD 商品定向 → 抢竞品流量          │
├─────────────────────────────────────┤
│ 下层：转化（Conversion）+ 召回      │
│   SP 搜索广告 → 收割搜索需求        │
│   SD 再营销 → 召回看过没买          │
└─────────────────────────────────────┘

配合逻辑：
  - 上层SB让买家「认识你」
  - 中层SBv/SD让买家「考虑你」
  - 下层SP/SD再营销让买家「买你」
  - 全程配合形成完整链路
```

> 关键认知：**单一广告只能覆盖漏斗一段，组合才能全程覆盖。** 只投 SP 的卖家，永远只能收割"已经在搜的下层流量"，上层认知和中层考虑全空着，量做不大。SB/SBv/SD 补齐了上层和中层。

### 不同阶段的品牌广告策略

```
阶段1：新品/新品牌冷启动
  → 重点：SP（收割已有搜索流量，先活下来）
  → 品牌广告：少量SB（品牌词+核心品类词）
  → SBv/SD：暂缓（流量基础不够，受众池小）
  → 逻辑：先证明能转化，再做品牌广告

阶段2：成长期（有稳定销量+评论积累）
  → SP：主力，持续优化
  → 加SB：品类词卡位 + 品牌词防御
  → 试SBv：核心产品做视频（有演示价值的）
  → 试SD：商品定向抢竞品流量
  → 逻辑：开始建品牌存在感，扩中层流量

阶段3：成熟期（品牌有一定搜索量+多SKU）
  → SP：持续收割
  → SB全开：品牌词+品类词+Store引流
  → SBv矩阵：多个有演示价值的SKU
  → SD全开：商品定向+再营销双线
  → 加DSP：站外认知+大受众池再营销
  → 逻辑：全漏斗打通，飞轮转动

阶段4：品牌领导者（强品牌搜索量）
  → 全漏斗+品牌建设
  → DSP做大规模认知广告
  → 品牌词自然流量成主要来源
```

> 实操纪律：**品牌广告要「按阶段加」，不是一上来全开。** 新品期就铺全套品牌广告，预算分散效果差。先 SP 站稳，再逐步加 SB→SBv→SD→DSP，每个阶段加一个，验证有效再加下一个。

### 品牌广告的预算分配原则

```
预算分配参考比例（按阶段）：

冷启动期：
  SP 90% / SB 10%（SB只投品牌词）

成长期：
  SP 70% / SB 15% / SBv 10% / SD 5%

成熟期：
  SP 50% / SB 20% / SBv 15% / SD 15%
  （有DSP的再分配DSP份额）

注意：
  - 这是参考，按实际效果调
  - 品牌广告的效果要看「飞轮」（品牌搜索提升），不只单次ACOS
  - 品牌广告的ACOS通常高于SP（因为承担认知任务）
  - 别用SP的ACOS标准苛求品牌广告
```

> 红线提醒：**别用 SP 的 ACOS 标准砍品牌广告。** SB/SBv 在上层做认知，单次 ACOS 可能高，但它们拉动品牌搜索和自然流量。只看单次 ACOS 会把真正有效的品牌建设砍掉。要看整体飞轮。

### 品牌广告组合的协同效应

```
协同效应示例：

场景：买家搜「ergonomic office chair」
  1. SB出现在顶部 → 买家看到你的品牌（认知）
  2. 往下滑看到SBv视频 → 看到产品演示（考虑）
  3. 点进详情页没买，离开
  4. 浏览其他网站时，SD再营销广告出现 → 召回
  5. 买家最终搜你的品牌词 → SP品牌词广告+自然位接住（转化）

→ 单次搜词，被你的4个广告触达4次
→ 这就是「组合」的威力，单一广告做不到
```

## 实操要点

- **SB/SBv/SD 是漏斗不同位置的武器**：SB 偏上层认知和品牌词，SBv 中层考虑用视频，SD 抢竞品+召回，组合才是全漏斗。
- **按阶段加品牌广告**：冷启动先 SP，成长加 SB→SBv→SD，成熟全开+DSP，不一上来全铺。
- **SBv 只给有视觉演示价值的产品**：标品/看不出区别的产品视频价值小，先把 SB 做好。
- **SD 做两件事**：商品定向抢竞品详情页流量 + 受众定向召回看过没买的人。
- **别用 SP 的 ACOS 标准砍品牌广告**：它们承担认知任务拉动飞轮，要看整体品牌搜索提升不只单次 ACOS。
