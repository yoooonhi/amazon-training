# 高级4.4 全漏斗流量打法设计（实操）

> **高级4：高阶广告与流量组合**
> **本课目标**：把 SP/SB/SD/DSP 四种广告工具，按"认知→考虑→转化→忠诚"漏斗设计成一套配合的流量打法，输出一份可执行的全漏斗广告作战方案。

## 这节课解决什么问题

前三课你分别学了 DSP、SD 再营销、品牌广告组合。但实战中决定整体广告策略时，很多卖家还是"每种广告分开投"——SP 一组人管、SB/SBv 另一组、SD 再一组、DSP 找代投，互相不通气，预算分散，谁也不知道整体漏斗哪里漏。

问题在于：**亚马逊的广告不是一个工具的事，是全漏斗的系统工程。** 买家从"不认识你"到"买你"再到"复购"，中间经过认知、考虑、转化、忠诚多个阶段。SP 只管转化，SB/SBv/DSP 管认知和考虑，SD 再营销管召回——必须当成一个整体设计，才能让流量在漏斗里顺畅流转，而不是每层都漏掉一半。

这节课是整个高级4模块的综合实操：把 SP/SB/SD/DSP 四种工具，按漏斗设计成一套配合的打法。学完你能输出一份"目标→漏斗→工具配合→预算分配→指标监控"的完整全漏斗作战方案。

## 核心知识

### 全漏斗模型：认知→考虑→转化→忠诚

```
完整漏斗的四层：

┌────────────────────────────────────────┐
│  ① 认知层（Awareness）                 │
│  买家状态：不认识你 / 不知道有这个需求  │
│  目标：建立品牌/品类认知               │
│  工具：DSP（站外）+ SB（品类词曝光）   │
│  指标：Reach、品牌搜索提升              │
├────────────────────────────────────────┤
│  ② 考虑层（Consideration）             │
│  买家状态：知道你，在对比考虑          │
│  目标：推动从「知道」到「想要」        │
│  工具：SBv（视频演示）+ SD（抢竞品）   │
│  指标：详情页浏览、加购率               │
├────────────────────────────────────────┤
│  ③ 转化层（Conversion）                │
│  买家状态：想买，在搜索/比价           │
│  目标：收割搜索需求，出单              │
│  工具：SP（搜索广告，主力）            │
│  指标：ACOS、ROAS、销量                 │
├────────────────────────────────────────┤
│  ④ 忠诚层（Loyalty/Retention）         │
│  买家状态：买过你                       │
│  目标：复购 + 交叉销售 + 防流失        │
│  工具：SD 购买再营销 + DSP 再营销      │
│  指标：复购率、LTV、新客比例            │
└────────────────────────────────────────┘
```

> 核心理念：**全漏斗的本质是「让流量在每层都不漏」。** 单一广告只覆盖一层（SP 多在转化层），其他层全空着。全漏斗打法用不同工具覆盖每层，让买家从认知到复购全程被你触达。

### 每层漏斗的工具配合详解

#### ① 认知层：让不认识你的人认识你

```
认知层的任务：建立「这个品类有你这个品牌」的印象

主工具：
  - DSP 站外展示广告：触达不在亚马逊搜索的人（站外网站/Twitch）
  - SB 品类词广告：买家搜品类词时，你的品牌出现在顶部

配合逻辑：
  - DSP 在站外创造「听说过」的认知
  - SB 在站内搜索时强化「这个品牌在」的印象
  - 买家还没需求时被DSP触达，有需求搜索时被SB接住

关键指标（不是ACOS！）：
  - Reach（触达人数）
  - 品牌搜索量提升（认知是否建立）
  - 新客比例（New-to-brand）
```

#### ② 考虑层：让认识你的人想要你

```
考虑层的任务：推动买家从「知道」到「想要」

主工具：
  - SBv 视频：产品演示、卖点冲击，激发欲望
  - SD 商品定向：出现在竞品详情页，对比中拉人

配合逻辑：
  - SBv 用视频展示「为什么选我」（演示/效果/差异化）
  - SD 投到竞品详情页，买家对比时看到你
  - 买家从「知道」被推到「想要」

关键指标：
  - 详情页浏览量（DPV）
  - 加购率
  - SBv 视频完播率/CTR
```

#### ③ 转化层：收割搜索需求出单

```
转化层的任务：把想买的人变成订单（最熟悉的SP主场）

主工具：
  - SP 搜索广告（主力）：核心词/长尾/竞品ASIN定向
  - SB 品牌词防御（守住自己的品牌词搜索）

配合逻辑：
  - 上两层（认知+考虑）建立的兴趣，最终转化为搜索行为
  - SP 在搜索时收割，把需求变成订单
  - SB 品牌词守住（防止竞品蹭你的品牌搜索）

关键指标：
  - ACOS / ROAS
  - 转化率
  - 销量
```

#### ④ 忠诚层：复购 + 交叉销售 + 防流失

```
忠诚层的任务：让买过的人再买、买更多、不流失

主工具：
  - SD 购买再营销：触达买过的人，推新品/配件/补充装
  - DSP 再营销：大规模召回买过的人（站外触达）

配合逻辑：
  - 买过你产品的人是最高价值受众
  - SD/DSP 购买再营销推新品/配件/复购装
  - 提升LTV（客户终身价值）

关键指标：
  - 复购率
  - LTV
  - 交叉销售率
```

### 全漏斗的协同飞轮

```
飞轮效应：全漏斗配合最终让整个系统越转越快

1. 认知层（DSP/SB）创造品牌认知
   → 品牌搜索量提升

2. 品牌搜索量提升 → SP 品牌词广告和自然位流量增大
   → SP的ACOS下降（品牌词转化高且便宜）

3. 自然流量增大 → 排名提升
   → 更多自然订单（不花钱的流量）

4. 订单增多 → Best Seller Rank提升
   → 自然流量进一步增大（飞轮加速）

5. 买过的人 → 忠诚层再营销复购
   → LTV提升，单客价值增大

→ 整体飞轮：认知投资 → 品牌搜索提升 → 自然流量增大 → 飞轮加速
→ 这是单投SP永远做不到的（SP只在转化层，上层空着飞轮转不起来）
```

> 关键认知：**全漏斗打法的最终价值不是「多几种广告」，而是「让飞轮转起来」。** 飞轮一旦转起来，你的自然流量、品牌词流量、复购会持续增长，广告依赖度下降。这才是高级运营和初级运营的本质差距。

### 全漏斗作战方案模板（实操输出）

```
《XX品牌 全漏斗流量作战方案》

一、整体目标
  - 阶段目标（如：3个月品牌搜索量+50%、整体ACOS降到X%）
  - 销量/利润目标

二、漏斗层级与工具配置
  ┌──┬──────────┬──────────────┬──────────┬────────┐
  │层│ 目标     │ 工具         │ 预算占比 │ 指标   │
  ├──┼──────────┼──────────────┼──────────┼────────┤
  │① │认知建立  │ DSP+SB品类词 │ 15-20%   │ Reach  │
  │② │推动考虑  │ SBv+SD商品   │ 20-25%   │ DPV    │
  │③ │收割转化  │ SP主力       │ 40-50%   │ ACOS   │
  │④ │复购忠诚  │ SD/DSP再营销 │ 10-15%   │ LTV    │
  └──┴──────────┴──────────────┴──────────┴────────┘

三、各工具具体设置
  SP：核心词/长尾/竞品ASIN定向分配
  SB：品牌词+品类词+Store引流
  SBv：哪些SKU的视频，投什么词
  SD：商品定向（竞品ASIN）+ 再营销（浏览/购买）
  DSP：站外认知 + 再营销（如有预算）

四、协同设计
  - 买家旅程：从认知到复购的触达链条
  - 数据监控：各层指标和飞轮信号（品牌搜索/自然流量）

五、预算与节奏
  - 月广告总预算：$____
  - 各工具预算分配（按上表占比）
  - 调整节奏（双周/月度复盘调比例）

六、监控与优化
  - 各层关键指标（不止ACOS）
  - 飞轮信号：品牌搜索量、自然流量占比、复购率
  - 优化触发条件
```

### 全漏斗打法的阶段适配

不是所有卖家都该做全套全漏斗，按阶段匹配：

```
阶段A：单品/小卖（月广告<$3000）
  → 重点：转化层SP（80%）
  → 试水：少量SB品牌词（10%）+ SD商品定向（10%）
  → 不做：DSP（预算撑不起）、SBv（先验证）
  → 目标：先活下来证明能转化

阶段B：成长期品牌（月广告$3000-$30000）
  → 转化层SP（50-60%）
  → 加考虑层：SB品类词+SBv+SD商品定向（25-30%）
  → 加忠诚层：SD再营销（10-15%）
  → 不做：DSP（先验证SD再营销）
  → 目标：扩中层流量，建品牌存在感

阶段C：成熟品牌（月广告>$30000）
  → 全漏斗全开
  → 加认知层：DSP站外（15-20%）
  → 转化层SP降到40-50%（飞轮起来自然流量补）
  → 忠诚层：SD+DSP再营销双线
  → 目标：飞轮转动，广告依赖度下降

阶段D：品牌领导者
  → 全漏斗 + 大规模DSP品牌广告
  → 自然+品牌词流量成主要来源
  → 广告做认知和新客，转化靠自然
```

> 实操纪律：**全漏斗不是「同时全开」，是「按阶段逐步搭建」。** 每加一层，先验证有效再加下一层。预算不够硬上全漏斗，每层都跑不出量，等于没做。

### 全漏斗的监控指标体系

```
全漏斗要看的不只是ACOS，是各层指标 + 飞轮信号：

各层指标：
  ① 认知层：Reach、品牌搜索量提升、新客比例
  ② 考虑层：详情页浏览、加购率、SBv完播率
  ③ 转化层：ACOS、ROAS、转化率、销量
  ④ 忠诚层：复购率、LTV、交叉销售率

飞轮信号（最重要的整体健康指标）：
  - 品牌搜索量是否持续增长（认知在建立）
  - 自然流量占比是否提升（飞轮在转）
  - 整体ACOS是否随时间下降（效率在提升）
  - 新客比例（认知广告在拉新）

⚠️ 警惕：只看ACOS会误判全漏斗效果
  → 认知层广告ACOS高是正常的
  → 要看整体飞轮是否在改善
```

> 红线提醒：**全漏斗打法的成功标志是「飞轮转动」——自然流量和品牌搜索增长，广告依赖度下降。** 如果投了很久全漏斗，自然流量没涨、品牌搜索没涨、整体ACOS没降，说明飞轮没转起来，要排查哪层漏了。

## 实操要点

- **全漏斗是系统工程不是单一工具**：认知/考虑/转化/忠诚四层，每层用不同工具覆盖，让流量不漏。
- **按阶段逐步搭建全漏斗**：小卖先转化层，成长加考虑+忠诚，成熟才加DSP认知层，预算不够别硬上全套。
- **飞轮转动是成功标志**：品牌搜索增长+自然流量提升+ACOS下降，飞轮转起来广告依赖度下降。
- **分漏斗层看指标**：认知看Reach/品牌搜索，考虑看DPV/加购，转化看ACOS，忠诚看复购/LTV，别只盯ACOS。
- **排查"哪层漏了"**：飞轮不转就排查——认知没建立？考虑没推动？转化没收割？忠诚没复购？针对性补强。
