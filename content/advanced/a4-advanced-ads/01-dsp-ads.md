# 高级4.1 DSP 广告入门

> **高级4：高阶广告与流量组合**
> **本课目标**：搞清 DSP（需求方平台）广告与 SP/SB 的本质区别（展示 vs 搜索），理解受众定向逻辑，判断自己的品牌到了什么阶段该上 DSP、预算量级多少。

## 这节课解决什么问题

初级和中级课你把 SP（Sponsored Products）打磨得很熟，能控 ACOS、能扩词、能优化搜索词。但走到一定阶段你会发现——**SP 的天花板到了**。搜索广告只能触达"已经在搜的人"，而你的潜在买家大部分时间并不在亚马逊搜索框前；你想做品牌认知、想触达"看过没买的人"、想在站外影响他们再拉回亚马逊，SP 完全做不到。

很多卖家听说"DSP 能做大品牌广告"，就找亚马逊代投 DSP，结果预算砸了几万美金没什么效果，因为根本不懂 DSP 的逻辑、自己的阶段不对、预算量级不够、受众设置混乱。

这节课讲清 DSP 的本质——它和 SP 是**两种完全不同的广告物种**（展示广告 vs 搜索广告），它的受众定向逻辑、预算量级、适用阶段。它是高级广告模块的地基，搞不懂 DSP 的本质，后面的全漏斗打法就是空中楼阁。

## 核心知识

### DSP vs SP/SB：两种不同的广告物种

这是最关键的认知。很多人把 DSP 当成"高级版 SP"，完全错了：

```
SP/SB（搜索广告）的本质：
  - 买家「主动搜索」某个词
  → 你的广告出现在搜索结果里
  → 捕获「已经有明确需求」的人
  → 是「需求收割」

DSP（展示广告）的本质：
  - 不管买家在不在搜
  → 你的广告主动「展示」给目标受众看
  → 触达「可能还没明确需求」或「看过没买」的人
  → 是「需求创造 + 需求召回」

类比：
  SP = 实体店里等顾客上门找你（收割现有需求）
  DSP = 在大街上发传单、做品牌广告把人拉进店（创造和召回需求）
```

| 维度 | SP/SB（搜索） | DSP（展示） |
|---|---|---|
| 触发方式 | 买家搜索关键词 | 系统主动展示给受众 |
| 广告位 | 搜索结果页、商品详情页 | 亚马逊站内+站外（首页、Twitch、第三方网站） |
| 受众状态 | 有明确需求（在搜） | 任意状态（认知/考虑/看过没买） |
| 目的 | 收割搜索需求 | 创造认知、召回流失、再营销 |
| 计费 | CPC（按点击） | CPM（按千次展示） |
| 定向 | 关键词/商品/类目 | 受众（行为/兴趣/lookalike） |

> 关键认知：**DSP 不是 SP 的升级版，是另一种广告物种。** SP 解决"搜索流量怎么抢"，DSP 解决"非搜索流量怎么创造和召回"。两者解决不同问题，是互补不是替代。

### DSP 的受众定向：核心能力

DSP 最强大的是**受众定向**——不靠关键词，靠"人"的特征和行为来触达：

```
DSP 受众定向的主要类型：

1. 再营销（Retargeting）
   → 触达「看过你的Listing/加购未买」的人
   → 召回最有可能转化的人（ROI最高）

2. 类目/兴趣受众
   → 触达「最近在看某品类」的人
   → 比如卖瑜伽垫，触达最近浏览健身器材的人

3. Lookalike（相似人群扩展）
   → 基于你现有买家，找「相似特征」的新人群
   → 扩大潜在客户池

4. 自定义受众
   → 基于你指定的ASIN/行为定义受众
   → 比如「买过竞品A的人」

5. 人口统计
   → 年龄/性别/收入等基础定向
```

> 关键认知：**DSP 的核心价值是「再营销」——召回看过没买的人。** 这是 DSP 投放最容易见效的部分，因为这些人已经表达过兴趣，转化率远高于冷启动受众。新手投 DSP 优先把再营销做透。

### DSP 的广告位：站内 + 站外

DSP 的展示位比 SP 广得多，不只局限在亚马逊站内：

```
DSP 广告位覆盖：

站内（亚马逊自有流量）：
  - 亚马逊首页 banner
  - 商品详情页的展示位
  - 搜索结果页的展示位（非搜索广告位）
  - 感谢页/购物车页

站外（亚马逊广告网络）：
  - Twitch（亚马逊的直播平台）
  - IMDb 等亚马逊自有媒体
  - 第三方网站/App（通过亚马逊广告网络）
  - 移动端展示

→ 这意味着 DSP 能触达「不在亚马逊搜索的潜在买家」
→ 这是 SP 做不到的
```

### DSP 的计费与预算量级

DSP 的计费方式和 SP 完全不同，这决定了它的预算门槛：

```
DSP 计费：CPM（Cost Per Mille，千次展示费用）
  → 你按「展示次数」付费，不按点击
  → 通常 CPM $5-$25 不等（按受众竞争程度）

预算量级（这是DSP的高门槛）：
  → DSP 通常有最低投放门槛
  → 通过亚马逊官方 DSP 管理：月最低预算约 $35,000起
  → 通过认证服务商（DSP代理商）：门槛较低但仍需可观预算
  
对比 SP：
  → SP 可以 $1/天起步
  → DSP 是「品牌级」投放，月预算以万美金计

结论：DSP 不是小卖家的工具，是有规模的品牌玩家才用得起
```

> 红线提醒：**DSP 不是「试试看」的广告。** 它的预算门槛高（官方管理月预算数万美金起），预算不够硬上 DSP，效果出不来还浪费钱。小卖家/中小品牌应该先把 SP/SB/SD 做透，有规模和品牌基础再考虑 DSP。

### DSP 的适用阶段

判断你的品牌该不该上 DSP：

```
适合上 DSP 的品牌特征：
  ✓ 有一定的品牌搜索量（说明有认知基础，DSP才能放大）
  ✓ 月销售额达到一定规模（能支撑DSP预算）
  ✓ 产品线丰富（DSP可跨产品线做矩阵）
  ✓ 有品牌建设预算（不只是短期ROI思维）
  ✓ 流量遇到瓶颈，SP触达不到更多人群

不该上 DSP 的阶段：
  ✗ 新品牌/新品类，连搜索流量都没站稳
  ✗ 月销量小，预算撑不起DSP门槛
  ✗ 单一SKU，DSP矩阵价值发挥不出来
  ✗ 只追求短期ROI，没品牌建设预算
  ✗ 团队对受众定向一窍不通

判断口诀：
  「搜索流量做到头了，要破圈做品牌认知」→ 考虑DSP
  「搜索流量还没做透，靠SP还能扩」→ 先把SP做透
```

### DSP 与 SP/SB 的配合逻辑

DSP 不是替代 SP，而是**配合 SP 做全漏斗**：

```
全漏斗配合（高级4.4会详细讲）：

漏斗上层（认知）：DSP 创造认知 + SB 品牌广告
  → 触达不认识你/没搜过你的人
  → 建立品牌印象

漏斗中层（考虑）：DSP 再营销 + SD
  → 召回看过没买的人
  → 推动考虑

漏斗下层（转化）：SP 搜索广告
  → 收割搜索需求
  → 直接出单

→ DSP 在上层和中层发力，SP 在下层收割
→ 单靠SP只能做下层，DSP补齐了上中层
```

> 关键认知：**DSP 的价值很多不在「直接转化」，而在「拉动上层认知，最终让 SP 的搜索流量增大」。** 跑 DSP 后，你的品牌搜索量、自然流量、SP 表现都可能提升——这是 DSP 的"飞轮效应"，但要时间才能看到。

### DSP 的关键指标（和 SP 不同）

DSP 看的指标和 SP 不一样，不能用 ACOS 衡量 DSP：

```
DSP 核心指标：

1. 上层（认知）指标：
   - Reach（触达人数）
   - Frequency（频次，每人看几次）
   - CPM（千次展示成本）
   - Viewability（可视率）

2. 中层（考虑）指标：
   - 中间指标：Detail page view（详情页浏览）
   - Add to cart rate（加购率）
   - Brand search lift（品牌搜索提升）

3. 下层（转化）指标：
   - ROAS（广告花费回报，类似ACOS的倒数）
   - 新客比例（New-to-brand，DSP的强项）
   - 总销量贡献

注意：DSP的上层广告不要用「直接转化率」苛求
  → 上层广告的价值在「认知建立」，转化可能滞后
  → 要看整体飞轮（品牌搜索/自然流量提升）
```

> 红线提醒：**别用 ACOS 衡量 DSP 上层广告。** 上层广告是做认知的，短期内"不出单"是正常的。苛求 DSP 上层广告像 SP 一样直接出单，会错误地砍掉真正有效的品牌建设。

## 实操要点

- **DSP 不是高级版 SP，是另一种物种**：SP 是搜索收割需求，DSP 是展示创造和召回需求，两者互补。
- **DSP 的核心价值是再营销**：召回看过没买的人 ROI 最高，新手投 DSP 优先把再营销做透。
- **DSP 有预算门槛，不是小卖家工具**：官方管理月预算数万美金起，规模和品牌基础不够先做透 SP。
- **DSP 的价值在飞轮不是单次转化**：上层广告不出单正常，要看品牌搜索/自然流量的整体提升。
- **别用 ACOS 衡量 DSP**：上层看 Reach/Frequency，下层看 ROAS/新客比例，分漏斗层看指标。
