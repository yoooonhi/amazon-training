# 高级1.3 跟卖打击 SOP

> **高级1：品牌保护与长期主义**
> **本课目标**：发现被跟卖后，能按标准流程分类处置——先判跟卖类型，再走对应对策，把"被动救火"变成"有SOP的体系化处置"。

## 这节课解决什么问题

前两课你学了 Transparency 和 Project Zero 两个"重型武器"。但实战中你打开 Listing 一看，Buy Box 被抢了，第一反应往往是慌——然后开始"乱投医"：找服务商赶、发警告信、自己申诉举报全上一遍。

问题是**跟卖有好几种，每种对策完全不同**。你用打假货的方法去打合法分销商，对方一申诉你账号反而受影响；你用 Transparency 去防 VC 自营，根本没用；你发警告信给职业跟卖团伙，对方完全不怕甚至反灌差评。

这节课给你一套**跟卖打击的标准SOP**：发现 → 分类 → 对策 → 执行 → 验证。核心不是某个技巧，而是"先分类再动手"——90% 的跟卖处置失败，都是因为没分清跟卖类型就乱打。

## 核心知识

### 第一步：发现跟卖（监控机制）

不能等销量掉了才发现。要有日常监控：

```
监控手段（按成本从低到高）：
  1. 每天扫一眼自己核心ASIN的Buy Box是否在自己手里（免费，但要纪律）
  2. 用Keepa/Helium10等工具设置「跟卖提醒」（工具费）
  3. Buy BoxShare报表（后台）→ 看自己占比有没有突然掉
  4. 第三方跟卖监控工具（自动化，适合多SKU）

触发跟卖SOP的信号：
  - Buy Box丢了 / Buy Box Share突然下降
  - 自己Listing的「其他卖家」栏出现陌生卖家
  - 价格被压到异常低（疑似甩货跟卖）
  - 突然来一批差评（疑似恶意跟卖捣乱）
```

> 关键纪律：**发现越早，处置越容易。** 跟卖挂了 48 小时再去处理，比挂了 2 小时就处理，难度翻几倍。所以"监控"本身就是防御。

### 第二步：分类——这是 SOP 的灵魂

发现跟卖后，**第一件事不是动手，是分类**。不同类型对策天差地别：

| 跟卖类型 | 特征 | 性质 | 处置方向 |
|---|---|---|---|
| **① 合法分销/授权商** | 是你授权的正规渠道，只是没管好价格 | 合法，但乱价 | 渠道管理（不是打） |
| **② 同款现货跟卖（无品牌壁垒）** | 卖跟你一模一样的通用货，你无独占权 | 合法竞争 | 靠品牌/差异化护城河 |
| **③ 假货跟卖（仿冒）** | 卖的是假货，盗用你商标/logo | 违法侵权 | Project Zero 自助下架 |
| **④ 未授权用你品牌跟卖** | 货可能是真的，但没授权用你商标 | 商标侵权 | 品牌备案/Project Zero |
| **⑤ 职业跟卖团伙（恶意）** | 专挑爆款挂，灌差评抢Buy Box | 恶意 + 常涉违规 | 组合拳 + 警惕反噬 |
| **⑥ VC/亚马逊自营跟卖** | 平台自有渠道跟卖你的ASIN | 平台规则内 | 极难处理，重在预防 |

> 红线警告：**最致命的错误是把①②当③④打。** 用 Project Zero 自助下架去打一个合法分销商或合法同款竞品，对方申诉成功，你的 Project Zero 资格受损，甚至触发对自己品牌备案的调查。

### 第三步：按类型走对应对策

#### 类型① 合法分销商乱价

```
处置：
  1. 不要用任何「打击」工具——他是合法的
  2. 走渠道管理：发正式邮件要求其遵守MAP（最低零售价）
  3. 在分销协议里写明「禁止在亚马逊低于X价销售」
  4. 屡教不改 → 切断供货（如果你能控制货源）
  5. 根本解决：建立渠道价格管控体系（Map Policy）
```

#### 类型② 同款现货合法跟卖

```
处置（核心：你没法「打」，只能「赢」）：
  1. 承认现实：通用产品你没独占权，人家合法跟卖
  2. 靠差异化拉开：升级Listing、A+、品牌故事、评论壁垒
  3. 靠品牌护城河：注册商标+品牌备案，让通用款无法蹭你品牌流量
  4. 长期：做差异化SKU/独家款，让跟卖者无货可跟
```

#### 类型③ 假货跟卖

```
处置（Project Zero 主场）：
  1. 立即截图存证（Listing页、卖家信息、侵权特写）
  2. 进 Project Zero → 自助下架（几小时内生效）
  3. 同时保留品牌备案 Report a Violation 作为备份手段
  4. 持续监控该卖家是否换马甲再来
  5. 长期：开 Transparency 从源头防假货进FBA
```

#### 类型④ 未授权用你品牌（货可能真）

```
处置（商标侵权）：
  1. 截图存证，重点是对方Listing标题/图片用了你的注册商标
  2. 品牌备案 → Report a Violation → 选「商标侵权」
  3. 描述要具体：你的商标注册号+对方哪处侵权
  4. 如果是批量/反复，升级到 Project Zero 自助下架
  5. 警惕：对方可能反申诉，证据链要扎实
```

#### 类型⑤ 职业跟卖团伙（最棘手）

```
处置（组合拳 + 防反噬）：
  1. 不要发软弱的「警告信」——职业团伙不怕，可能反灌差评
  2. 评估其违规点：是否假货？是否盗图？是否刷评？——找到可打击的实质违规
  3. 有实质违规（假货/侵权）→ Project Zero 自助下架
  4. 无明显违规但恶意 → Test Buy 买一件取证（证明是假货/不符）再投诉
  5. 持续骚扰 → 考虑 Transparency + 品牌备案组合长期防御
  6. 绝对不要：用黑科技/服务商暴力赶——合规风险极高
```

> 红线警告：**职业跟卖团伙最危险的不是他本身，而是你被激怒后用了违规手段，反而把自己的账号搭进去。** 永远在合规框架内反击，宁可慢一点。

#### 类型⑥ VC/亚马逊自营跟卖

```
处置（坦诚：极难）：
  1. 预防为主：如果你有VC账号/供货关系，谈判时明确独占条款
  2. 一旦平台自营挂上，普通工具基本无效
  3. 可尝试：通过品牌备案渠道申诉「未授权」
  4. 长期：增强品牌独占性，自有品牌运营为主，减少对VC依赖
```

### 完整跟卖打击 SOP 清单

```
发现跟卖 → 按6步走：

□ Step1 存证：截图Listing、卖家信息、侵权点、时间戳
□ Step2 分类：对照6类表格，判明是哪类跟卖
□ Step3 决策：该类对应的处置方向（打/管/赢/防）
□ Step4 执行：走对应工具（Project Zero/品牌备案/渠道管理）
□ Step5 验证：24-48h确认跟卖是否消失、是否反申诉
□ Step6 复盘：记录本次处置工具+效果，沉淀到自己的「跟卖案例库」
```

> 纪律：**每处置一次跟卖都记录。** 攒到一定量你会看清"哪类跟卖用什么最有效"——这是你专属的防御手册，比任何服务商的方案都准。

## 实操要点

- **先分类再动手**：发现跟卖别慌，花10分钟判明类型，90%的失败都是没分类就乱打。
- **合法跟卖不能「打」只能「赢」**：分销商乱价走渠道管理，同款合法竞品靠品牌差异化，别把合法对手当假货打，否则反伤自己。
- **职业团伙要在合规框架内反击**：找实质违规点（假货/盗图/刷评）再打，Test Buy 取证是有效手段，绝不用黑科技。
- **Project Zero 只打假货**：自助下架的射程是确凿假货/侵权，滥用会被踢出甚至反噬账号。
- **每案必存档复盘**：建你的跟卖案例库，长期看哪类用啥最有效，形成专属防御手册。
