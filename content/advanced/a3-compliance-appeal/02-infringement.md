# 高级3.2 侵权处理

> **高级3：合规深水区与申诉**
> **本课目标**：能分清"被投诉侵权"和"主动维权"两条线，掌握专利/商标/版权三类侵权的应对策略，不被投诉就慌，也不盲目反击伤到自己。

## 这节课解决什么问题

侵权是亚马逊深水区里最复杂的合规问题，因为它有**两个方向**：你被别人投诉侵权（要应对申诉），以及别人侵犯你（要主动维权）。很多卖家两个方向分不清，要么被投诉了慌乱下架认罚（其实你有抗辩理由），要么自己被侵权了乱投诉（结果证据不足反被认定恶意投诉）。

更复杂的是，侵权分三类——专利（发明/外观）、商标、版权，每类的判定标准、投诉渠道、申诉逻辑完全不同。你以为"我进货时供应商说没问题"就能抗辩，结果专利侵权供应商的承诺根本不算数；你以为"我没用对方logo"就不侵权，结果外观专利一比对输了。

这节课讲清侵权处理的两条线和三类侵权，给具体的应对策略和话术框架。它和高级3.1的申诉课是配套的——侵权冻结是账号被封的高频原因，处理逻辑相通但细节更专业。

## 核心知识

### 侵权处理的两条线

```
┌─────────────────────────────────┐
│ 线A：你被投诉侵权（防守）       │
│  → 别人投诉你侵权了             │
│  → 你的任务：评估是否成立       │
│    → 成立 → 下架整改 + 申诉恢复 │
│    → 不成立 → 提交抗辩反驳       │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ 线B：你被侵权（进攻/维权）      │
│  → 别人侵犯你的知识产权         │
│  → 你的任务：收集证据 → 投诉    │
│    → 用品牌备案/Project Zero    │
└─────────────────────────────────┘
```

> 关键认知：**两条线不能混。** 被投诉时要冷静评估抗辩空间（不是一被投诉就认），主动维权时证据要扎实（不是看谁不爽就投诉）。混淆两条线会导致：该抗辩的认罚了、不该投诉的乱投了。

### 三类侵权的本质区别

| 类型 | 保护什么 | 怎么判定 | 常见场景 |
|---|---|---|---|
| **专利-发明专利** | 技术方案/功能 | 权利要求比对 | 产品用了某项专利技术 |
| **专利-外观专利** | 产品外观设计 | 整体视觉效果比对 | 产品外观模仿了专利设计 |
| **商标** | 品牌/标识/商品名 | 是否混淆性相似 | Listing用了别人的注册商标词 |
| **版权** | 原创作品（图/文/视频） | 是否实质性相似 | 盗用别人的图片/文案 |

> 红线提醒：**很多卖家栽在「我以为没事」。** 供应商说"这个外观没专利"、自己觉得"没用对方logo"、抄了张图觉得"改改就行"——这些主观判断都不算数，侵权判定看的是法律比对。上架前要做知识产权审核，不能靠感觉。

### 线A：被投诉侵权的应对流程

#### 第一步：读投诉，确认侵权类型和权利基础

```
收到侵权投诉后，先确认：
  ☐ 投诉方是谁？（品牌方/权利人/律所）
  ☐ 投诉的是哪类侵权？（专利/商标/版权）
  ☐ 权利基础是什么？（专利号/商标注册号/版权登记号）
  ☐ 被投诉的具体是哪个产品/哪个元素？（标题词/图片/外观/技术）

  → 这些信息在投诉邮件或业绩通知里
  → 搞清楚才能判断「是否真的侵权」
```

#### 第二步：评估投诉是否成立

这是关键决策点——**不是所有投诉都成立**：

```
评估是否成立的方法：

商标侵权：
  - 投诉方的商标注册了哪个类别？覆盖你的产品吗？
  - 你用的是「描述性用法」还是「商标性用法」？
    （例：你的产品是「适用于iPhone的手机壳」，iPhone是描述兼容性，不是商标侵权；
          但你标题写「iPhone原装手机壳」误导成原厂，是侵权）
  - 商标是否有效？（有些商标已失效）

专利侵权：
  - 看专利权利要求/外观图片，你的产品是否真的落入保护范围？
  - 专利是否有效？（可查USPTO/EUIPO）
  - 是否有现有技术（prior art）可主张专利无效？

版权侵权：
  - 你的作品是原创还是抄袭？相似度多少？
  - 你是否有合法授权或合理使用？
```

#### 第三步：按成立与否采取对策

```
情况1：投诉成立（你确实侵权）
  → 立即下架侵权产品（不拖延）
  → 核查供应链：找出侵权源头（供应商？自己设计抄的？）
  → 撰写申诉POA（参考高级3.1四段结构）：
     根因：知识产权审核流程缺失
     临时措施：已下架 + 移除侵权元素
     长期方案：建立上架前知识产权审核SOP
     证据：改进后的审核流程文件
  → 恢复其他未侵权产品销售

情况2：投诉不成立（你有抗辩理由）
  → 不要盲目下架认罚
  → 撰写申诉反驳（Counter-Notice）：
     说明为什么不侵权（描述性用途/专利无效/合法授权等）
     提供证据（你的授权书/原创证明/专利无效比对）
     要求恢复Listing
  → 注意：反驳要专业，建议咨询知识产权律师
```

> 关键纪律：**被投诉不要盲目认罚。** 如果你有合理抗辩理由（描述性用途、合法授权、专利无效等），下架认罚等于承认侵权，可能被索赔甚至多次投诉导致封号。先评估，再决定是认是抗。

### 申诉反驳话术框架（英文参考）

```
Dear Amazon,

We received a complaint regarding [ASIN] alleging [侵权类型].
We have investigated and respectfully dispute this complaint 
for the following reasons:

1. The complainant's [商标/专利号] does not cover our product because
   [具体理由：类别不符/不构成混淆/描述性用途等].

2. Our use of [具体元素] is [合法理由：legitimate descriptive use /
   licensed use / independently developed].

3. Evidence supporting our position:
   - [授权文件/原创证明/专利比对分析/律师意见书]

We request reinstatement of our listing. 
[附件清单]

Sincerely,
[你的名字]
```

> 红线提醒：**反驳要基于事实和证据，不能赌气或狡辩。** 如果说不出具体法律理由、拿不出证据，反驳大概率失败，还可能被视为恶意申诉。拿不准时咨询知识产权专业人士。

### 线B：被侵权（主动维权）的策略

这条线在高级1.2（Project Zero）和高级1.3（跟卖SOP）讲过工具，这里补充策略：

```
主动维权的流程：

1. 固定证据（第一时间）
   - 截图侵权Listing（标题/图片/五点/卖家信息）
   - 记录ASIN、侵权方信息、时间戳
   - 比对：你的权利基础（商标号/专利号/版权登记）vs 对方侵权点

2. 选择投诉渠道（按侵权类型）
   - 商标侵权 → 品牌备案 Report a Violation / Project Zero
   - 版权侵权（盗图盗文）→ 品牌备案版权投诉
   - 专利侵权 → 亚马逊专利侵权投诉流程（APEX等，较复杂）

3. 撰写投诉（专业、具体）
   - 你的权利基础（注册号/登记号）
   - 对方具体哪处侵权（指向明确）
   - 比对说明（你的vs对方的）
   - 要求（下架侵权Listing）

4. 跟进
   - 跟踪投诉处理结果
   - 对方若反申诉，准备应对
   - 持续监控是否换马甲再来
```

> 关键纪律：**主动维权证据要扎实，描述要专业。** 模糊的投诉（"他抄我"）不会被受理。要具体到"对方ASIN XXX标题第X词使用了我们的注册商标XXX（注册号XXX）"。证据不足的投诉会被判定恶意，反伤自己品牌备案。

### 三类侵权的预防（比处理重要）

```
上架前知识产权审核SOP（每个新品必走）：

商标：
  ☐ 标题/五点/Search Terms不用别人的注册商标词
  ☐ 自己的品牌词已注册（防被反向投诉）
  ☐ 描述兼容性用「compatible with/for」而非原厂品牌词

专利：
  ☐ 选品阶段查专利（USPTO/EUIPO/Google Patents）
  ☐ 供应商提供的外观/技术，要求出具不侵权保证（但仅供参考）
  ☐ 高风险品类（电子/玩具/家居创新）必查

版权：
  ☐ 图片自己拍或购买授权（不用网图/竞品图）
  ☐ 文案原创（不抄竞品Listing）
  ☐ 核心视觉资产登记版权（高级1.4）

供应链：
  ☐ 与供应商签知识产权协议（责任划分）
  ☐ 要求供应商提供原创证明/授权文件
  ☐ 但要明白：供应商的「保证」不能替代你自己审核
```

> 红线警告：**供应商说「没专利」「不侵权」不能完全信。** 最终被投诉的是你（卖家），不是供应商。供应商拍胸脯的保证，在你被冻结时帮不了你。自己审核才是根本。

## 实操要点

- **分清两条线**：被投诉（防守，评估抗辩）vs 被侵权（进攻，维权），不要混淆。
- **被投诉先评估是否成立**：不是一投诉就认罚，有抗辩理由（描述性用途/合法授权/专利无效）要专业反驳。
- **三类侵权判定标准不同**：商标看类别和混淆、专利看权利要求和外观比对、版权看实质性相似。
- **主动维权证据要扎实**：具体到对方哪处侵权、你的权利基础是什么，模糊投诉会被判恶意。
- **上架前知识产权审核比事后处理重要**：查商标/专利/版权，别信供应商的口头保证。
