# 高级2.3 跨境物流方案

> **高级2：多站点全球扩张**
> **本课目标**：能在海运/空运/铁路（中欧班列）之间做成本时效决策，选对货代，并搞清清关的关键节点，让货物按时、合规、低成本地到达海外仓。

## 这节课解决什么问题

上节课你决定了"用FBA还是远程配送还是海外仓"。但无论哪种，货都得**从中国发到海外**——这就是跨境物流（头程）。很多卖家把头程外包给货代就不管了，结果踩坑不断：到货延误错过旺季、被货代乱收费、清关出问题整批货扣留、VAT/EORI没搞清被海关退运。

跨境物流是亚马逊运营里"水最深"的环节之一：海运有整箱/拼箱、空运有专线/商业、铁路只通欧洲、货代水平参差不齐、清关涉及税务和合规。一个决策失误，轻则多花几万运费，重则整批货卡海关错过销售季。

这节课讲清楚三种主流头程方式的成本时效、货代怎么选、清关的关键节点。它是全球扩张的"物理基础"——货到不了或到了被扣，一切免谈。

## 核心知识

### 三种主流头程方式对比

| 方式 | 时效 | 成本 | 适用 | 特点 |
|---|---|---|---|---|
| **海运** | 慢（美西25-35天，欧30-40天） | 低 | 大批量、非紧急、稳态补货 | 主流，按整箱/拼箱 |
| **空运** | 快（5-10天） | 高 | 小批量、紧急补货、新品试水 | 贵但快 |
| **铁路（中欧班列）** | 中（欧线18-25天） | 中 | 欧洲专供，介于海空之间 | 只通欧洲，性价比折中 |

```
头程选择的本质：在「成本 vs 时效」之间权衡

海运：成本最低，但慢，适合「有时间的大批量」
  → 稳态补货的主力，90%的常规货物走海运

空运：成本最高，但快，适合「没时间的小批量」
  → 紧急补货、新品首批试水、断货救急

铁路：成本时效居中，只通欧洲
  → 欧洲站点的折中选择，海运太慢空运太贵时用
```

### 海运的两种方式：整箱 vs 拼箱

海运不是只有一种，按货量分两种，成本结构完全不同：

| 方式 | 货量 | 计费 | 适合 |
|---|---|---|---|
| **FCL（整箱）** | 能装满一个集装箱（20尺/40尺） | 按箱计费（一个柜多少钱） | 货量大，单价低 |
| **LCL（拼箱）** | 装不满一箱，和别人拼 | 按体积/重量（CBM/吨）计费 | 货量小，灵活但单价高 |

```
整箱 vs 拼箱决策：
  货量 ≥ 8-10 CBM（立方米）→ 走整箱FCL更划算
  货量 < 8 CBM → 走拼箱LCL
  
  例：你的货共 12 CBM
  → 整箱一个40尺柜（约67 CBM）单价低，但你只装12方浪费空间
  → 整箱一个20尺柜（约33 CBM）可能正好
  → 拼箱按12 CBM计费，单价高但灵活

关键：让货代给你算两种方案的到门价，对比后再选
```

> 红线提醒：**拼箱的隐藏成本多**——除了基本运费，还有拼箱服务费、目的港收费、拆箱费等，问价时要让货代报"到门全包价"，别只比基本海运费。

### 空运的两种类型

| 类型 | 时效 | 成本 | 适合 |
|---|---|---|---|
| **商业快递（DHL/FedEx/UPS）** | 最快（3-7天） | 最高 | 极紧急、小件、样品 |
| **空运专线** | 快（5-10天） | 高但比快递低 | 紧急补货、新品首批 |

```
空运决策：
  样品/极小件/极急 → 商业快递（贵但省心）
  紧急补货几百件 → 空运专线（性价比好）
  常规补货 → 不该用空运（太贵），走海运

何时该用空运（值得花这个钱）：
  ✓ 即将断货，海运来不及（断货损失 > 空运溢价）
  ✓ 新品首批试水，等不及海运（市场窗口期）
  ✓ 旺季前最后补货窗口
  ✗ 常规稳态补货用空运 = 利润被运费吃光
```

### 中欧班列（铁路）：欧洲专供的折中

中欧班列是从中国经铁路到欧洲（德国杜伊斯堡等为主）的线路，时效和成本介于海运和空运之间，**只对欧洲站点有意义**：

```
中欧班列特点：
  时效：约18-25天（比海运快10-15天）
  成本：比海运高，比空运低很多
  线路：中国 → 哈萨克斯坦/俄罗斯 → 欧洲
  适合：欧洲站补货，海运太慢空运太贵的中间场景

局限：
  - 只通欧洲（美国日本没有对应铁路方案）
  - 受地缘政治影响（途经国局势影响线路稳定性）
  - 班次和运力有波动
```

### 货代选择：跨境物流最关键的人

货代（货运代理）是你和海运/空运/海关之间的中间人，**选错货代，后面的环节全是坑**。选货代的标准：

| 维度 | 好货代 | 差货代 |
|---|---|---|
| 报价透明 | 报到门全包价，列清明细 | 只报基本运费，目的港乱收费 |
| 时效稳定 | 说几天就几天，延误少 | 时效飘忽，旺季拖延 |
| 清关能力 | 熟悉亚马逊FBA清关、VAT/EORI | 清关出问题甩锅 |
| 沟通响应 | 主动同步货物状态 | 问一句答一句，甚至失联 |
| FBA经验 | 专门做亚马逊FBA头程 | 主做传统外贸，不懂FBA |

```
选货代的实操步骤：
  1. 至少找3家报价对比（别只看价格，看明细）
  2. 要求「到门全包价」含所有费用
  3. 问清：是否专做亚马逊FBA头程？做过你的品类吗？
  4. 要往期客户案例或参考
  5. 先小批量试一票，验证时效和沟通再放量
  6. 建立长期合作，但保留备选货代（不吊死一棵树）

红线：
  ✗ 报价异常低（低于市场价很多）—— 多半目的港乱收费
  ✗ 不肯提供明细 —— 后面加价没完
  ✗ 沟通不积极 —— 出事就失联
  ✗ 没做过FBA头程 —— 清关和标签会出问题
```

> 关键纪律：**货代是合作伙伴不是工具，要验资、要试单、要长期维护。** 一个靠谱货代能帮你省无数麻烦，一个烂货代能让你整批货报废。这部分值得你投入时间筛选。

### 清关：跨境物流的生死关

清关是货物进入目的国时过海关的环节，**清关出问题，整批货可能被扣、退运、罚款**。关键节点：

```
清关需要的核心文件/资质：
  - 提单（B/L，海运）或空运单（AWB）
  - 商业发票（Commercial Invoice）—— 货物价值/品名/数量
  - 装箱单（Packing List）
  - 原产地证（CO，部分情况需要）
  - 进口商EORI号（欧洲）/相关税号
  - VAT注册号（欧洲进口需要）
  - 产品合规文件（如CE认证、电子产品认证等，按品类）

清关常见问题（踩坑高发区）：
  - 申报价值过低 → 海关怀疑低报，查验/罚款
  - 品名/HS编码错 → 清关延误或税率算错
  - 缺合规认证（CE/FCC等）→ 整批货扣留
  - VAT/EORI没办或错 → 无法清关，退运
  - 侵权/敏感品 → 扣货甚至法律风险
```

> 红线警告：**清关问题往往致命。** 一个CE认证缺失，整批电子产品可能被海关扣押销毁。所以高合规品类（电子/儿童/食品/医疗相关）扩张前，**先搞定合规认证再发货**，不能赌。

### 完整头程决策流程

```
Step1 确定目的国和时效需求
  → 多久必须到货？（旺季前/断货救急/常规补货）

Step2 选运输方式
  → 常规大批量 → 海运（按货量选FCL/LCL）
  → 紧急小批量 → 空运（专线或快递）
  → 欧洲折中 → 中欧班列

Step3 选货代
  → 3家比价（到门全包）→ 验FBA经验 → 试单 → 长期合作

Step4 备齐清关文件
  → 发票/装箱单/认证/VAT/EORI/HS编码核对

Step5 发货+跟踪
  → 货代同步节点 → 到港 → 清关 → 提货 → 送FBA仓

Step6 到仓验收
  → FBA接收数量核对 → 入库 → 上架销售
```

### 头程成本占售价的警戒线

```
头程运费占售价比例的健康范围：
  轻小件：5-10%
  标准件：8-15%
  大件/重货：15-25%（更高，但仍要控）
  超大件：可能>25%（这类产品本身要重新评估选品）

超过警戒线的处理：
  → 重新谈货代/换运输方式降成本
  → 优化包装减体积（海运按体积算）
  → 提售价（但影响竞争力）
  → 实在不行 → 这个产品不适合跨境，砍掉
```

## 实操要点

- **按时效需求选方式**：常规大批量走海运（成本低），紧急小批量走空运（快但贵），欧洲折中用中欧班列，别一刀切。
- **整箱拼箱按货量选**：≥8-10 CBM走整箱更划算，问价要到门全包价别只比基本运费。
- **货代要验资试单**：找3家比价、看FBA经验、先小批量试一票再放量，异常低价的货代多半后面乱收费。
- **清关文件提前备齐**：发票/装箱单/认证/VAT/EORI/HS编码，高合规品类（电子/儿童/食品）先搞定认证再发货。
- **盯头程成本占售价比例**：轻小件5-10%、标准件8-15%、大件15-25%，超了就优化包装或砍产品。
