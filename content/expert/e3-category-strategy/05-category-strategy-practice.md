# 进阶3.5 品类战略规划实操：一份能用的总纲

> **进阶3：战略选品与品类布局**
> **本课目标**：把卡位、产品矩阵、护城河、长期规划整合成一份《品类战略规划》，输出给团队执行的统一纲领。

## 这节课解决什么问题

前四课你学了品类卡位、产品矩阵、护城河、长期规划。但实战中这四件事必须拧成一个整体——卡位决定了你在哪打、矩阵决定了用什么打、护城河决定了打完怎么守住、规划决定了打到什么时候。操盘手要做的，是把这四者整合成一份团队都能看懂、能执行、能追踪的《品类战略规划》。

很多团队的战略是"老板脑子里的想法"，没有落到文档、没有对齐团队，结果运营不知道整体方向、广告不知道为哪个战略服务、供应链不知道要为什么壁垒铺路。一份能用的战略规划，要让团队每个角色都能从中读出"我该做什么、为什么这么做"。这节实操课就是帮你产出这份总纲。

## 核心知识

### 品类战略规划的结构

一份完整的《品类战略规划》包含六部分：

```
1. 品类卡位：我们在哪个品类的哪个位置打？避谁、打谁？
2. 产品矩阵：用哪些SKU打？引流/利润/形象/防御各几个？
3. 护城河：打下来怎么守？四维壁垒各建到什么程度？
4. 财务目标：这个品类要贡献多少利润和现金流？（对接进阶2）
5. 路线图：1年/3年分阶段达成什么？
6. 资源匹配：要多少资金、团队、供应链支撑？
```

### 规划的内在一致性检查

战略规划最怕"各部分打架"。写完后要做一致性检查：

| 检查项 | 不一致的例子 | 后果 |
|---|---|---|
| 卡位 vs 矩阵 | 卡位是"小众利基"但矩阵铺10个SKU | 资源分散，利基做不深 |
| 矩阵 vs 护城河 | 矩阵全是引流品没利润品 | 没钱建护城河 |
| 护城河 vs 路线图 | 护城河要建品牌但路线图只规划1年 | 品牌来不及建 |
| 财务 vs 规划 | 规划要扩3倍但财务说资金不够 | 现金流断裂（进阶2.4） |
| 规划 vs 团队 | 规划要做品牌但团队没品牌能力 | 战略落地不了 |

> 操盘纪律：**战略规划的每个部分必须互相支撑，不能打架**。卡位决定矩阵，矩阵决定护城河投入，护城河决定路线图节奏，路线图决定资源需求，资源约束反过来校准卡位野心。这是一个闭环，任何一环脱节，战略就是空中楼阁。

### 落地案例：一个家居品类的战略规划示范

```
1. 卡位：家居-收纳细分，避开$10以下红海，切入$25-35中端价格带的「极简风」缝隙
2. 矩阵：引流品2个（收纳盒低价带流量）、利润品3个（极简收纳系列）、形象品1个（设计师款）、防御品2个
3. 护城河：供应链（2个独家SKU+成本优势12%）、数据（单品评论300+目标）、品牌（极简风调性+品牌词搜索）
4. 财务：第1年贡献净利润$8万、净利率18%、资金回报年化70%
5. 路线图：Y1站稳（3个明星品盈利）、Y2扩张（矩阵8SKU+品牌起量）、Y3壁垒（品牌+数据双层）
6. 资源：资金$15万、团队2运营+1广告+1库存、独家供应链2家
```

### 让战略不变成抽屉文件的三条纪律

**纪律一：战略要翻译成团队语言。** 别给运营讲"卡位利基"，要讲"今年我们只做这3个SKU，把这3个做到类目前20"。把战略翻译成每个角色的具体动作。

**纪律二：战略要季度复盘。** 每季度开战略复盘会，对照规划看进度——卡位还对吗？矩阵配比还合理吗？护城河建得怎么样？财务目标达成率多少？偏离了就调。

**纪律三：战略要全员对齐。** 把规划核心（不一定是全文）同步给团队每个人，让运营知道为什么做这3个SKU、让广告知道为哪个战略服务。战略不对齐，执行必然走样。

> 操盘纪律：**战略的价值不在写得多漂亮，在能不能指导每一天的决策**。检验一份战略好坏的标准：团队遇到一个新机会（比如一个新品、一个新站点），能不能用这份战略判断"做不做"？如果能，战略就是活的；如果还得问老板，战略就是死的。

## 实操要点

- **战略规划六部分要完整**：卡位/矩阵/护城河/财务/路线图/资源，缺一不可。
- **各部分必须互相支撑**：卡位决定矩阵决定护城河决定路线图决定资源，是闭环。
- **写完做一致性检查**：五个检查项，任何一处打架战略就是空中楼阁。
- **战略要翻译成团队语言**：把"卡位利基"翻译成"今年只做这3个SKU做到前20"。
- **战略好坏看能不能指导日常决策**：团队能用战略判断"新机会做不做"，战略才是活的。

## 动手练习

产出你品类的《品类战略规划 v1.0》：

1. 写品类卡位：在哪个品类的哪个位置、避谁打谁、切入什么缝隙。
2. 设计产品矩阵：引流/利润/形象/防御各几个SKU，配比是否健康。
3. 定护城河：四维壁垒各建到什么程度，优先哪个。
4. 定财务目标：1年净利润/净利率/资金回报（对接进阶2的算法）。
5. 画路线图：1年/3年分阶段目标。
6. 列资源需求：资金/团队/供应链。
7. 做一致性检查：六部分互相支撑吗？哪里打架？
8. 把核心战略翻译成团队语言，同步给全员，开第一次季度战略复盘会。
