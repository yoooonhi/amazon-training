# 进阶3.1 品类卡位分析：找位置、避红海、挖缝隙

> **进阶3：战略选品与品类布局**
> **本课目标**：能用一套品类卡位框架，在一个品类里找到自己的战略位置，判断哪些是红海要避开、哪些是蓝海缝隙可以切入。

## 这节课解决什么问题

单品运营时你关心"这个品能不能卖"，操盘手关心的是"我在这个品类里站在哪、打谁、避谁"。这是完全不同的视角。很多团队的产品布局是随机的——老板看到什么火就跟什么、运营推荐什么就上什么，结果一堆 SKU 散落在不同品类，互相没协同，每个都打得辛苦、每个都没壁垒，遇到价格战第一个死。

更深的问题是**只会看红海不会找缝隙**。打开一个品类，看到头部品牌垄断、价格卷到地板，就判断"这个品类不能做"。但操盘手的视角不同——红海里往往藏着缝隙：头部门看不上的长尾需求、大品牌没覆盖的细分人群、价格带的中间空档、功能组合的空白点。这些缝隙单看不大，但对中小卖家足够吃饱，而且竞争温和、利润可观。

这节课给你一套品类卡位的战略框架，让你学会在一个品类里找准位置，既能避开正面红海，又能发现可切入的蓝海缝隙。

## 核心知识

### 卡位的本质：需求×竞争×能力 三维定位

选一个品类位置，不是看"卖什么"，而是回答三个问题：**这个位置有需求吗？竞争我扛得住吗？我有能力做吗？** 三维交叉的地方就是你的卡位点。

| 维度 | 看什么 | 怎么判断 |
|---|---|---|
| 需求 | 市场够不够大、在涨还是在缩 | 搜索量趋势、品类总GMV、增长率 |
| 竞争 | 头部多强、价格多卷、壁垒多高 | 头部集中度、价格带、评价门槛 |
| 能力 | 我的供应链/资金/团队够不够 | 成本优势、资金盘、运营能力 |

> 操盘纪律：**好的卡位是"需求够大、竞争够弱、能力够得上"的交集**。三维缺一不可。需求大但竞争惨烈是红海，竞争弱但需求小是死市场，需求大竞争也弱但你没能力做是别人的菜。

### 红海识别：什么样的品类不要碰

学会识别红海比找蓝海更重要——避开必输的战场，本身就是战略。以下信号出现 3 个以上，基本是红海，中小卖家慎入：

| 红海信号 | 含义 |
|---|---|
| 头部3-5个品牌占品类GMV 60%以上 | 寡头垄断，新人难突围 |
| 头部评价数都过万、4.3星以上 | 评价壁垒极高，新人难追 |
| 价格带集中在低端、毛利被压到<15% | 价格战已到底，没利润空间 |
| 头部都在投重广告、CPC极高 | 流量成本高，新人烧不起 |
| 同质化严重、差异化空间小 | 只能拼价格，拼不过大卖家 |
| 品类增长率<5%甚至萎缩 | 存量博弈，新人分不到增量 |

### 蓝海缝隙的5种类型

红海品类里往往藏着缝隙。操盘手要学会识别这5类缝隙：

| 缝隙类型 | 例子 | 切入策略 |
|---|---|---|
| 人群细分 | 母婴品类的"早产儿专用"、"敏感肌专用" | 专注细分人群，做深做专 |
| 价格带空档 | 低端$10和高端$50之间$25-35没人占 | 占中间价格带，性价比定位 |
| 功能组合 | 产品A+功能B没人组合过（如保温杯+温度显示） | 微创新组合，做差异化 |
| 场景细分 | 户外品类的"超轻露营"、"城市通勤" | 切特定使用场景 |
| 颜值/审美 | 同功能但设计语言不同（极简风/复古风） | 靠审美和品牌调性差异化 |

```
找缝隙的方法：
1. 把品类按价格带、人群、功能、场景4个维度做矩阵
2. 看哪些"格子"里竞争者少、需求却存在
3. 验证这个缝隙的真实需求（搜索量、竞品销量、差评里的未满足需求）
4. 评估自己能不能做（供应链、成本、差异化能力）
```

### 卡位决策矩阵

把候选位置按"需求强度"和"竞争强度"两个维度放进去，决定打不打：

| | 竞争弱 | 竞争强 |
|---|---|---|
| **需求大** | **蓝海**：全力切入，先发优势 | **红海主流**：慎入，除非有碾压性优势 |
| **需求小** | **小众利基**：可做，但要控制规模预期 | **死市场**：避开 |

> 操盘纪律：**中小卖家的甜点位是"小众利基"和"蓝海缝隙"，不是红海主流**。红海主流是给有供应链碾压优势和充足资本的大卖的。中小卖家去红海正面硬刚，是用自己的短板打别人的长板。先把几个利基缝隙做扎实、积累资金和供应链，再考虑向上突围。

### 卡位的动态视角：缝隙会被填满

蓝海是会变红海的。一个缝隙被你做出来了，跟随者会进来。操盘手要有动态视角：

```
缝隙的生命周期：
发现期（你先做，竞争少）→ 验证期（跟随者进入）→ 拥挤期（变红海）→ 衰退/转型

策略：在「发现期」快速建立先发优势（评价、排名、品牌认知），
      在「验证期」开始布局下一个缝隙，在「拥挤期」要么做深壁垒要么退出。
```

这就是为什么操盘手要持续做品类扫描——手上永远有 2-3 个缝隙在孵化，等当前的变红海时，新的已经起来了。

## 实操要点

- **用"需求×竞争×能力"三维定位**：好的卡位是三者交集，缺一不可。
- **先学会识别红海**：6个信号出现3个以上就是红海，中小卖家慎入。
- **找5类蓝海缝隙**：人群细分、价格带空档、功能组合、场景细分、颜值审美。
- **甜点位是小众利基和蓝海缝隙**：别去红海正面硬刚，用自己的短板打别人长板。
- **动态视角**：蓝海会变红海，手上永远有2-3个新缝隙在孵化。

## 动手练习

选一个你正在看或想进的品类，做一次卡位分析：

1. 评估这个品类的红海信号（6项逐条打勾），出现几个？决定整体是红海还是有空间。
2. 按"价格带×人群×功能×场景"做矩阵，找出竞争者少的格子（缝隙）。
3. 对每个候选缝隙，验证真实需求（看搜索量、竞品销量、差评里的未满足需求）。
4. 把候选位置放进"需求×竞争"决策矩阵，判断是蓝海、利基、红海还是死市场。
5. 诚实评估自己的能力（供应链、资金、差异化）够不够切入这个缝隙。
