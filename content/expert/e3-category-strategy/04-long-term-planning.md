# 进阶3.4 长期品类规划：1年/3年路线图

> **进阶3：战略选品与品类布局**
> **本课目标**：能给一个品类画 1 年和 3 年的路线图，把战略拆成阶段性目标，让团队知道今年干什么、明年往哪走。

## 这节课解决什么问题

绝大多数团队的品类规划 horizon（视野）只有 3 个月——这个季度上什么品、备什么货。这种短视野会导致一个致命问题：永远在追眼前的销量，没有长期布局。等眼前的红利吃完，才发现没有下一个增长曲线、没有品牌积累、没有供应链深耕，整个盘子停滞甚至衰退。操盘手和运营主管最大的区别，就是操盘手要能"看到 3 年后"，今天做的每个决策都要为 3 年后的位置铺路。

但长期规划不是"画大饼"。很多战略规划之所以变成抽屉文件，是因为它太虚——"3 年做到品类第一"这种口号谁都会喊，但没有拆成可执行的年度目标、没有阶段性里程碑、没有资源匹配。好的长期规划是"战略意图清晰 + 阶段拆解具体 + 资源匹配到位"。这节课给你一套品类路线图的制定方法，让 3 年战略能落地、能追踪、能调整。

## 核心知识

### 品类路线图的三层结构

一个能用的品类规划要分三层：战略意图（3年）、阶段目标（年度）、行动拆解（季度）：

| 层级 | 时间 | 回答什么 | 颗粒度 |
|---|---|---|---|
| 战略意图 | 3年 | 这个品类3年后要变成什么样？ | 定性、方向性 |
| 阶段目标 | 1年 | 今年在3年路径上的哪个位置？要达成什么？ | 定量、可衡量 |
| 行动拆解 | 季度 | 这3个月具体做什么动作？ | 可执行、可追踪 |

### 战略意图：3年后品类要长成什么样

战略意图不是GMV数字，而是品类的"终局画像"。回答这4个问题：

```
1. 市场位置：3年后在品类里排第几？占多少份额？
2. 产品矩阵：引流/利润/形象/防御各几个SKU？护城河建到什么程度？
3. 品牌资产：品牌搜索占比多少？复购率多少？有没有品牌认知？
4. 供应链：有没有独家供应链？成本优势多大？抗风险能力如何？
```

举例（一个家居品类的3年战略意图）：

```
3年战略意图：
- 市场位置：进入品类Top 20，占细分市场份额8%
- 产品矩阵：15-20个SKU，2个明星利润品+3个引流品+2个形象品+若干防御品
- 品牌资产：品牌词搜索占比25%，复购率30%，类目有一定品牌认知
- 供应链：3个独家SKU，核心品成本比跟随者低15%
```

### 阶段目标：3年的分阶段路径

3年意图要拆成3个阶段，每个阶段有不同重点。典型的成长路径：

| 阶段 | 时间 | 核心任务 | 关键目标 |
|---|---|---|---|
| 第1阶段：站稳 | 第1年 | 跑通1-2个明星品，建立供应链基础 | 单品盈利、供应链稳定、数据积累 |
| 第2阶段：扩张 | 第2年 | 扩产品矩阵、建护城河、做品牌 | 矩阵成型、评论/复购积累、品牌词起量 |
| 第3阶段：壁垒 | 第3年 | 深耕品牌和数据护城河、巩固份额 | 品牌壁垒+数据壁垒双层，份额稳定 |

> 操盘纪律：**第1年别贪多，先把1-2个明星品做扎实**。很多团队第一年就想铺10个SKU，结果每个都没做透、供应链没深耕、数据没积累，第二年被卷死。先站稳再扩张，是品类成长的基本节奏。资源有限时，做深比做多重要。

### 年度目标：SMART化

阶段目标要落成年度的SMART目标（具体、可衡量、可达成、相关、有时限）：

| 维度 | 第1年目标示例 | 第2年目标示例 |
|---|---|---|
| GMV | $50万 | $150万 |
| SKU数 | 3-5个（1-2个明星） | 8-12个（矩阵成型） |
| 净利率 | 15% | 18% |
| 评论数 | 单品100+ | 单品300+ |
| 品牌词搜索 | <5% | 15% |
| 复购率 | 10% | 25% |

### 季度行动拆解：战略要落到动作

年度目标再拆成4个季度的具体行动。每个季度聚焦1-2个关键动作，不要贪多：

```
第1年Q1：选品+供应链落地（签独家、首批备货、Listing搭建）
第1年Q2：明星品起量（广告冷启动、评价积累、排名冲刺）
第1年Q3：明星品稳量+利润优化（控TACOS、提毛利、备旺季）
第1年Q4：旺季收割+孵化第二个SKU（旺季备货、第二个品冷启动）

每季度末复盘：目标达成率多少？下季度调整什么？
```

### 路线图的动态调整机制

3年路线图不是刻在石头上的。市场在变、竞争在变，路线图要定期校准：

| 节奏 | 做什么 |
|---|---|
| 每季度 | 复盘季度目标达成率，调整下季度行动 |
| 每半年 | 评估阶段目标是否还合理，微调年度目标 |
| 每年 | 重审3年战略意图，市场变了就调整方向 |

> 操盘纪律：**路线图是活的，但战略意图要稳**。阶段目标和行动可以随市场调整，但3年的战略方向（你要在品类里站什么位置、建什么壁垒）不要轻易变。频繁换方向的团队，3年后还在原地。

## 实操要点

- **规划分三层**：3年战略意图（定性）、年度阶段目标（定量）、季度行动（可执行）。
- **第1年别贪多**：先把1-2个明星品做透，站稳再扩张，资源有限时做深比做多重要。
- **年度目标要SMART化**：GMV/SKU数/净利率/评论数/品牌搜索/复购率，每个都可衡量。
- **每季度聚焦1-2个关键动作**：别贪多，季度末复盘达成率再调整。
- **战略意图要稳、行动可调**：方向不轻易变，但阶段目标和方法随市场动态调整。

## 动手练习

为你最看重的品类画一份路线图：

1. 写3年战略意图（市场位置/产品矩阵/品牌资产/供应链4个维度，定性描述）。
2. 把3年拆成"站稳/扩张/壁垒"3个阶段，定每阶段核心任务。
3. 把今年落成SMART年度目标（GMV/SKU数/净利率/评论数/品牌搜索/复购率6个指标）。
4. 把今年拆成4个季度，每季度定1-2个关键行动。
5. 定调整节奏：季度复盘达成率、半年微调年度目标、每年重审战略意图。
