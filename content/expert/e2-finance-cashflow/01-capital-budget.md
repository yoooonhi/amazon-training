# 进阶2.1 资金预算与占用：多 SKU 总资金盘规划

> **进阶2：财务与现金流管理**
> **本课目标**：能给一个多 SKU、多站点的盘子做总资金盘规划，算清每个 SKU 的资金占用周期，并判断哪些钱用自有、哪些该融资。

## 这节课解决什么问题

单品运营时你只关心一个 SKU 备多少货、赚多少钱。但操盘手管的是几十上百个 SKU 的盘子，问题完全不一样：**整个盘子需要多少钱？这些钱压在哪里、压多久？什么时候会缺钱？什么时候有闲钱可以再投？** 算不清这些，就会出现两种典型灾难——要么盘子扩张太快，资金链绷断，爆款做起来了却没钱备货；要么钱全压在滞销品和库存里，账面赚钱、口袋空空，连工资都快发不出。

更深的认知盲区是"资金占用周期"。一个毛利 30% 的产品，听起来很赚，但如果它从下单到回款要压 4 个月资金，一年只能转 3 圈，实际年化资金回报可能还不如把钱存定期。操盘手看产品不只看毛利率，更要看**资金周转效率**——同样一块钱，能转几圈、每圈赚多少。

这节课把多 SKU 资金盘的规划方法、资金占用周期的测算、融资与自有资金的取舍讲透，让你能对整个盘子的钱有清晰的全局观。

## 核心知识

### 总资金盘 = 库存占用 + 在途占用 + 广告预付 + 安全垫

一个盘子同时在飞的钱，远不止"仓库里的货"。完整资金盘包括四大块：

| 资金占用项 | 含义 | 典型占比 |
|---|---|---|
| 在库库存占用 | FBA 仓库里所有可售+预留的货的成本 | 30-40% |
| 在途库存占用 | 已下单未到货（生产中、海运中、清关中）的货的成本 | 25-35% |
| 广告与运营预付 | 广告费、促销费、测评等运营投入 | 10-15% |
| 现金安全垫 | 应对突发（退货激增、补货救急、账号冻结）的备用金 | 10-15% |

> 操盘纪律：**只算"在库库存"的资金盘都是错的**。很多老板算"我压了 100 万的货"，其实加上在途、广告预付、应收（亚马逊回款有 14 天延迟），真实资金占用可能是 180 万。少算的 80 万就是未来断链的窟窿。

### 资金占用周期：一块钱被压多久

这是操盘手最该会的算账。一个 SKU 的资金占用周期，是从你**付款给工厂**到**亚马逊把销售款打到你账上**的全过程。

```
资金占用周期 = 付款到出货 + 海运/空运 + 到仓上架 + 销售回款延迟

典型海运品：
- 付款到出货（生产+质检）：25 天
- 海运到港+清关：40 天
- FBA 上架：7 天
- 销售周转（备的货卖完）：45 天（看动销）
- 亚马逊回款延迟：14 天
→ 单次资金占用周期 ≈ 131 天（约 4.3 个月）
```

资金占用周期决定了一块钱一年能转几圈：

```
年周转次数 = 365 / 资金占用周期
= 365 / 131 ≈ 2.8 次/年

年化资金回报 = 单次毛利率 × 年周转次数
若单次毛利率 25%，则年化 = 25% × 2.8 = 70%
（看起来不错，但若动销慢、占用周期拉到 180 天，年化就跌到 50%）
```

> 操盘纪律：**评估一个 SKU 的"赚钱能力"，毛利率不是终点，年化资金回报才是**。一个毛利 40% 但一年只转 2 圈的品，可能不如一个毛利 20% 但一年转 6 圈的品赚得多。这就是为什么动销快的引流品有时比高毛利慢动销品更"值钱"。

### 多 SKU 资金盘规划步骤

操盘手每个季度要做一次全盘资金规划：

```
步骤1：列出所有在售和在备 SKU 的资金占用（在库+在途）
步骤2：按"资金占用周期×毛利率"给 SKU 排序，识别哪些是资金效率高的"现金牛"、哪些是低效占用
步骤3：算总资金需求 = 全部占用 + 广告预付 + 安全垫
步骤4：对比可用资金（自有+可融资），算资金缺口
步骤5：若缺口>0，决定砍哪些低效 SKU 的备货；若盈余，决定加码哪些高效 SKU
```

### SKU 资金效率矩阵

把 SKU 按毛利率和周转速度两个维度分四类，决定资金怎么分配：

| | 周转快（占用周期短） | 周转慢（占用周期长） |
|---|---|---|
| **毛利高** | **明星品**：重仓加码，资金优先供应 | **潜力品**：限量备货，逼着优化动销 |
| **毛利低** | **引流品**：维持小赚，控制备货量 | **鸡肋品**：清货退出，释放资金 |

> 操盘纪律：**资金是稀缺资源，要会"砍鸡肋养明星"**。很多团队的资金被鸡肋品（低毛利慢动销）占住，导致明星品没钱备货断货。每季度用这张矩阵过一遍，把鸡肋品的资金释放给明星品。

### 融资 vs 自有资金：什么时候该借钱

| 情况 | 该不该融资 | 理由 |
|---|---|---|
| 明星品多、年化回报>融资成本、自有不够 | **该融** | 资金回报高于利息，借钱是赚钱 |
| 全是鸡肋品、回报不确定 | **不该** | 借钱扩烂盘子，越借越亏 |
| 旺季前短期缺口（确定能卖出去） | **该融**（短期） | 旺季备货是确定性回报，短借划算 |
| 长期结构性缺口（一直缺钱） | **不该** | 是模式问题，不是钱的问题 |

融资成本红线：**只有当资金的预期年化回报 > 融资年化成本 + 5% 风险溢价时，才借钱**。比如融资年化 12%，那这个钱投出去的预期年化回报至少要 17% 才划算。

## 实操要点

- **资金盘不只算在库库存**：在途、广告预付、回款延迟、安全垫都要算，少算一项就是窟窿。
- **用"资金占用周期"评估 SKU**：毛利率不是终点，年化资金回报（毛利率×年周转次数）才是。
- **每季度用资金效率矩阵过一遍**：砍鸡肋、养明星，资金向高效 SKU 倾斜。
- **融资只投明星品**：回报高于成本+风险溢价才借钱，烂盘子借钱是加速死亡。
- **旺季短期缺口可以短借**：旺季备货是确定性回报，结构性长期缺口不能靠借钱掩盖。

## 动手练习

用你现在的盘子做一次资金盘规划：

1. 列出全部在售+在备 SKU，算每个的在库+在途资金占用。
2. 对每个 SKU 估算资金占用周期（付款到回款），算年周转次数。
3. 用"毛利率×年周转次数"算每个 SKU 的年化资金回报，按资金效率矩阵分类。
4. 算总资金需求（含广告预付+15%安全垫），对比可用资金，找缺口或盈余。
5. 如果有缺口，决定砍哪些鸡肋品；如果有盈余，决定加码哪些明星品。把这次规划的结果作为下季度备货预算的依据。
