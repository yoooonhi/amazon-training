# 第3节 · 自动广告精细化

> **手册：[广告打法手册](./)**
> **本节收录**：✅ 自动广告四组拆分、✅ 自动广告阶梯竞价、✅ 标签打法、⚠️ 关键词调研广告
> **本节目标**：掌握自动广告从"开起来就放着"到"精细化运营"的 4 种方法，知道每种方法解决什么问题。

## 这节解决什么问题

很多卖家开自动广告就是"四组匹配全开 + 一锅炖"，跑一段时间发现数据全集中在紧密匹配和宽泛匹配，另外两种几乎没数据——**因为没数据，根本不知道这两种匹配类型表现如何**。

另外常见的问题：
- 自动广告跑出来的搜索词跟你产品八竿子打不着 → **标签偏差**
- 不知道某个产品在什么竞价水平能出单 → **没有阶梯测试**
- 想挖长尾词但不知道从哪下手 → **关键词调研广告**

这节讲 4 种精细化方法，分别解决上述问题。

> 关键认知：**自动广告不是"开起来就不用管"**。精细化运营的核心是"分组建、分别看数据、分别调策略"。

## 核心知识

### ✅ 打法八：自动广告四组拆分

**底层问题**：自动广告默认四种匹配类型全开，但跑出来的数据会严重倾斜——紧密和宽泛吃掉大部分预算，同类商品和关联商品几乎没曝光。

```
自动广告四种匹配类型：

1. 紧密匹配（Close Match）
   → 买家搜与你的商品紧密相关的词，展示你的广告
   → 通常吃掉大部分预算

2. 宽泛匹配（Loose Match）
   → 买家搜与你的商品不那么密切相关的词，展示你的广告
   → 数据次多

3. 同类商品（Substitutes）
   → 买家查看类似商品详情页时，展示你的广告
   → 通常数据很少（被前两种挤掉）

4. 关联商品（Complements）
   → 买家查看互补商品详情页时，展示你的广告
   → 通常数据极少
```

**精细化做法**：把四种匹配类型**单独拆成四个广告活动**测试，一个产品对应四个自动广告活动。

```
广告结构：
  自动广告 1 —— 紧密匹配（Close Match）
  自动广告 2 —— 宽泛匹配（Loose Match）
  自动广告 3 —— 同类商品（Substitutes）
  自动广告 4 —— 关联商品（Complements）
```

**为什么拆分**：

```
拆分前（一锅炖）：
  - 紧密 + 宽泛吃掉 90% 预算
  - 同类 + 关联饿死，没数据
  - 你根本不知道后两种到底表不表现

拆分后（四组独立）：
  - 每组都有独立预算
  - 能看到每种匹配类型的真实表现
  - 发现哪组值得加预算、哪组该关掉
```

> 实操要点：四组拆分**主要解决"看不到后两种匹配表现"的盲区**。拆分后跑 2-4 周，根据数据决定：表现好的组加预算、表现差的组关掉或保留小预算监控。

### ✅ 打法九：自动广告阶梯竞价

**底层问题**：你不知道自己的产品在不同竞价水平上能不能出单。开一个自动广告用建议竞价，等于**赌一个数字**——可能出价太高白烧钱，也可能太低跑不起来。

**阶梯竞价做法**：开多个（5-10 个）自动广告活动，**每个活动用不同竞价**，形成阶梯。

```
阶梯竞价示例（假设建议竞价 $0.9）：

五组方案：
  活动 1：$0.2
  活动 2：$0.4
  活动 3：$0.6
  活动 4：$0.8
  活动 5：$1.0

十组方案（更细）：
  活动 1：$0.1
  活动 2：$0.2
  活动 3：$0.3
  ... 
  活动 10：$1.0
```

**预算和判断规则**：

```
每组预算：够点击 10 次的金额
  → 嫌贵的话，$0.5 以上的活动设 $5 预算
  → 跑一周看数据

判断规则：
  → 数据对比明显时，2-3 天就能关掉太差的活动
  → 转化率（出单量 / 点击数）≥ 10% 的活动，可以一点点加预算
    （如从 $5 加到 $8、$12）
  → 转化率不达标（如点 15 下出 1 单）就烧完不加

如果 10 个活动没一个行：
  → 要么是类目特殊（玩具类目、偏个性化工艺品）
  → 要么是 Listing 的图文/价格/评论有问题，需要先优化 Listing
```

> 实操要点：阶梯竞价是**用最小成本摸清自己产品的竞价曲线**。摸清后保留 2-3 个有效竞价段的活动长期跑，其余关掉。

### ✅ 打法十：标签打法（诊断自动广告乱跑）

**底层问题**：自动广告跑出来的搜索词经常跟你产品八竿子打不着——明明卖男士止鼾带，却跑出 "health beauty" 这种泛词；明明卖蓝色男士棉质 T 恤，却匹配到女士/儿童 T 恤。

**根本原因**：亚马逊系统给你的产品贴的**「标签」有偏差**。系统对产品的"认知"不对，自动广告就会乱匹配。

**三种标签**：

```
标签的本质：
  亚马逊系统根据对你的关键词、所在类目、商品分类的「认知」，
  为你的商品贴的「标签」。标签越准确，代表亚马逊越「懂」你的产品。

三种标签：

(1) 关键词标签
    → 后台选择关键词定位时，系统推荐的关键词列表
    → 决定你的关键词流量
    
(2) 类目标签
    → 后台选择商品投放时，出现的分类列表
    → 决定你的类目流量
    
(3) ASIN 标签
    → 后台选择商品投放时，出现的各个商品列表
    → 决定你 ASIN 投放的流量是否精准
    → 比关键词和类目标签更具体
```

**三种标签偏差及纠正策略**：

| 偏差类型 | 表现 | 纠正策略 |
|---|---|---|
| 标签相关度低 | 止鼾带却跑出 health beauty 大词 | **先开手动广告确定并纠正标签，再投自动广告** |
| 标签太宽泛 | T 恤跑出 "T-shirt / 蓝色棉质" 等泛词 | **先确定主推词，检查标题最左侧是否被大词影响** |
| 标签已精准 | 高腰比基尼推荐"高腰比基尼/黄色比基尼/高腰 swimsuit" | **直接开自动广告，效率最高** |

**比喻（理解三种标签的关系）**：

```
把产品比作一个人：
  关键词标签 = 他的特征（帅/聪明/幽默）
  类目标签 = 他的群体归属（男生/学生）
  ASIN 标签 = 他的名字（林XX）

标签越具体，亚马逊对你的认知越精准
```

> 红线提醒：**标签偏差时不要直接开自动广告**。先用**手动广告**告诉系统"我的产品到底是什么"，纠正标签后再开自动广告。否则自动广告会按错误标签乱匹配，烧预算。

### ⚠️ 打法十一：关键词调研广告（长尾词挖掘）

**底层逻辑**：为你的产品寻找表现好的关键词并重点投放推广，重点关注 **long tail keyword（长尾词）**。

```
关键词调研广告的标准做法：

① 用工具做关键词库
   工具：卖家精灵、鸥鹭、Sif、Helium10、JS
   → 挖出与产品相关的长尾词

② 用调研好的关键词库，测试所有匹配类型
   → 广泛匹配 / 词组匹配 / 精准匹配 各开一组

③ 给予足够的预算和测试时间
   → 让广告积累足够数据

④ 根据数据筛选有效长尾词
   → 表现好的词单独建精准匹配活动重点投放
```

**经典案例（理解为什么打长尾词）**：

```
Schmidt's（一个天然除臭剂品牌）：
  - 在长尾词「natural deodorant for men」的所有搜索中占据主导地位
  - 如果它去打大词「men's deodorant」，建议出价可能高于 $12
  - 大词对资金雄厚的卖家可能可持续
  - 但中小卖家根本烧不起这个竞价

结论：
  - 大词竞价高、转化未必好
  - 长尾词竞价低、转化更精准
  - 中小卖家优先打长尾词
```

> ⚠️ **为什么标 ⚠️**：关键词调研广告**本身完全合规**，标 ⚠️ 是因为它常被包装成"秘籍"，其实就是建词库+测匹配类型的常识操作。**不是秘籍，是基础**。学完别觉得发现了什么捷径——这是每个运营都该做的基本功。

## 实操要点

- **自动广告四组拆分主要解决"看不到后两种匹配表现"的盲区**：拆分后跑 2-4 周再根据数据决定哪组加预算、哪组关掉。
- **阶梯竞价用最小成本摸清竞价曲线**：5-10 组活动分别给不同竞价，跑一周看转化率，转化率 ≥ 10% 的加预算，不达标烧完不加。
- **标签偏差时先开手动纠偏再开自动**：自动广告按系统认知匹配，认知错了就乱跑，手动广告能告诉系统你产品到底是什么。
- **关键词调研广告是基础不是秘籍**：建词库+测匹配类型，每个运营都该做，别当成捷径。
- **标签偏差的根源常在标题**：检查标题最左侧是否出现了影响系统判断的大词。

## 动手练习

选一个自动广告跑得不好的产品，做一次诊断：
1. 打开 Search Term Report，看前 20 个搜索词
2. 判断标签是否偏差：≥40% 的词跟你产品无关 → 标签偏差
3. 如果标签偏差，先暂停自动广告，开一个手动精准广告投你真正的主推词（2-3 个），跑 7-10 天"教"系统认识你的产品
4. 再重新开自动广告，对比跑出来的搜索词是否更精准

重点练的是**用 Search Term Report 诊断标签偏差**，而不是单纯调整竞价。
