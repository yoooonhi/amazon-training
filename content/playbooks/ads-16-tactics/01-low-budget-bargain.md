# 第1节 · 小预算捡漏组合

> **手册：[广告打法手册](./)**
> **本节收录**：✅ 乞丐捡漏法、✅ 错词法、✅ 西班牙语选词法、✅ 品牌法
> **本节目标**：掌握 4 种低预算拓流量打法，知道什么产品、什么阶段、用什么设置最划算。

## 这节解决什么问题

广告费越来越贵，大词的 CPC 动不动 $1-3，中小卖家根本烧不起。但亚马逊上还有大量被竞品忽略的低价流量：

- **错词/简写/他国语言**——没人专门竞价
- **大品牌词**——大品牌自己不太投广告，竞价低
- **高 ACOS 的大词热词降到极低竞价**——靠"只降低"策略捡偶发订单

这 4 种打法的共同特征是 **低竞价 + 高位置溢价 + 选词避开红海**。但**核心不在竞价低，而在选词**——选错了词，竞价再低也是浪费。

> 关键认知：**捡漏不是"出价低就能赢"，而是"找到没人抢的流量洼地"**。选词这一步错了，后面竞价策略再精妙也白搭。

## 核心知识

### ✅ 打法一：乞丐捡漏法（低竞价 + 高溢价）

**适用前提**（三条都要满足）：
- ✅ 自然出单已经较多，有 Review 积累
- ✅ 客单价 ≤ $35（高客单价产品捡漏效果差）
- ✅ 已经跑过主推词，捡漏只是补充流量

```
乞丐捡漏法的底层逻辑：
  - 用极低的基座竞价（0.02-0.30 美金）
  - 配合高位置溢价（搜索页顶部 / 商品页 200%-900%）
  - 靠"只降低"或"固定"策略只接便宜的展示机会
  - 大部分时间不展示，偶尔抢到便宜的坑位就出单

为什么有效：
  - 大词的竞价被头部卖家顶得很高
  - 但同样的词在非高峰时段、非首页位置，竞价会回落
  - 你用低竞价 + 高溢价，专挑这些"低洼时段"出手
```

**4 种常见设置**（按多变体 / 单属性区分）：

| 场景 | 设置 | 竞价策略 | 竞价 | 位置溢价 |
|---|---|---|---|---|
| 多变体 · 预算 $10-20/天 | 自动广告组 | 固定竞价 | $0.1-0.3 | 顶部/商品页 200-900% |
| 多变体 · 极低预算 | 自动广告 | 固定/只降低 | $0.02-0.1 | 顶部/商品页 200-900% |
| 多变体 · 动态测试 | 自动广告 | 动态竞价 | $0.02-0.1 | 顶部/商品页 200-900% |
| 多变体 · 品牌主页 | 品牌广告（SB）+ 商品定位 | — | $0.1-0.3 | 顶部/商品页 200-900% |
| 单属性 · 预算 $5-10/天 | 自动广告 | 固定竞价 | $0.02-0.05 | 顶部/商品页 200-900% |
| 单属性 · 极低预算 | 自动广告 | 动态竞价 | $0.02-0.03 | 顶部/商品页 200-900% |
| 单属性 · 类目捡漏 | 展示广告（SD）+ 类目 | — | $0.1-0.3 | 顶部/商品页 200-900% |

> ⚠️ **现实提醒**：文档原文里 $0.02-0.1 的竞价在 2024 年后的多数类目已经跑不动了。**先用建议竞价工具看你的类目实际水平**，再按建议竞价的 1/3-1/5 设基座竞价。

**选词原则（乞丐法的灵魂）**：

```
✅ 该选的词：
  - 拼写错误（cocktail shaker → cocktail shakr）
  - 简写（USB-C → usbc）
  - 他国语言（见打法三：西班牙语）
  - 偏门长尾（蓝色加大号瑜伽垫 男士 加厚）
  - Review 里买家自己用的非标准词

❌ 不该选的词：
  - 大词热词的主形式（已经被竞品打烂）
  - 跟你产品无关的词（点了也不转化）
```

> 实操要点：选词的来源有两个——(1) 用第三方工具（卖家精灵/鸥鹭/Helium10/JS/Sif）挖错词、长尾词；(2) 自己收集买家 Review 里出现的非标准表达。

---

### ✅ 打法二：错词法（拼写错误词竞价）

**底层逻辑**：买家搜索时常打错关键词，这些错词：
- 搜索量可能不低（很多人都会拼错）
- 几乎没有卖家专门竞价 → 流量成本极便宜
- 亚马逊虽然能识别错词并"算到正确词范围"，但**广告排名和权重**不如直接投错词

```
错词法的两步操作：

① 用第三方工具找出错词
   工具：卖家精灵、鸥鹭、Sif、Helium10、JS（都有关键词调研功能）
   找法：输入核心关键词 → 看工具返回的搜索词报告 → 筛出含拼写偏差的词

② 专门创建一个错词广告活动
   - 放入这些错词
   - 用广泛匹配或精准匹配
   - 竞价可以明显低于主词的竞价
```

**常见错词类型**：

| 错词类型 | 例子 |
|---|---|
| 字母顺序颠倒 | ypga mat（应为 yoga） |
| 漏字母 | water bottle（应为 bottle） |
| 多字母 | backpakc（应为 backpack） |
| 同音替代 | their / there 混用 |
| 简写 | USB-C → usbc, iPhone → iphon |

> 实操要点：错词法**单独建活动**，不要混进主词广告活动——否则会被主词的高竞价拉高。

---

### ✅ 打法三：西班牙语选词法（美国站特有红利）

**底层逻辑**：美国市场英语之外，**西班牙语是第二大语言**（约 13% 家庭主要讲西语）。很多产品大词的西语版本有可观搜索量，但**几乎没卖家专门竞价**。

```
西语选词法的四步操作：

① 确定英文核心关键词
   例：cocktail shaker

② 翻译成西班牙语
   例：coctelera（调酒器）

③ 用关键词工具验证搜索量
   工具：卖家精灵、鸥鹭、Sif、Helium10、JS
   验证：是否有真实搜索量？搜索量是否值得投？

④ 反查竞品 ASIN 验证
   用工具反查头部竞品 ASIN 的关键词
   → 一般能发现他们吃到的西语词

⑤ 单独创建精准匹配广告活动
   只投验证过有流量的西语词
```

**适用判断**：
- ✅ 美国站
- ✅ 产品类目有西语用户群体（家居、厨房、母婴、美容等日常品类）
- ❌ 欧洲/日本站（西语用户少）
- ❌ B2B/工业品类（西语用户极少）

> 红线提醒：**别凭感觉翻一个西语词就投**。一定要用工具验证：(1) 真实搜索量；(2) 是否是西语用户实际使用的表达（有时直译并不地道）。验证不过的词投了也是浪费。

---

### ✅ 打法四：品牌法（蹭大牌品牌词）

**底层逻辑**：大品牌在亚马逊上有自然流量，但**大品牌自己投广告的不多**（他们觉得不需要），所以这些品牌词的竞价反而很低。蹭这些大牌词，等于用低价吃下品牌自带的搜索流量。

```
品牌法的三步操作：

① 亚马逊前台搜索框搜索产品核心关键词

② 在搜索结果页左侧找到「品牌」栏目
   → 这里有该类目下所有品牌名

③ 收集这些品牌名后：
   - 专门设置一个品牌广告活动
   - 用「广泛匹配」投放这些品牌词
   - 在其他广告活动里对这些品牌词做「词组否定」（避免内部竞价冲突）
```

**适用判断与风险提醒**：

| 维度 | 判断 |
|---|---|
| ✅ 适用 | 大品牌词流量大、竞价低、且你的产品有相对优势（价格/评分/差异化） |
| ⚠️ 谨慎 | 大品牌已开 Transparency / Project Zero 的，转化率可能极低，ACOS 难看 |
| ❌ 不适用 | 你产品比大牌差很多，蹭了词也转化不了 |

> ⚠️ **现实提醒**：2024 年后越来越多大品牌开了品牌保护，加上买家搜品牌词时心智已被品牌锁定，**蹭大牌的转化率通常远低于中性词**。ACOS 经常很难看。建议先小预算测试 2 周，看转化数据再决定要不要继续。

## 实操要点

- **捡漏的核心是选词不是竞价**：错词、西语、偏门长尾这些"被忽略的流量洼地"才是捡漏的灵魂，低竞价只是手段。
- **乞丐法有前提**：客单价 ≤ $35 + 已有 Review 积累 + 自然单已经跑起来，三条全满足才适合捡漏。新品/光杆司令产品不适用。
- **错词法单独建活动**：不要混进主词广告活动，否则会被主词的高竞价拉高整体 CPC。
- **西语词必须工具验证**：别凭翻译软件直译，要用第三方工具验证搜索量、反查竞品 ASIN。
- **品牌法谨慎用**：大品牌的转化率往往低于中性词，先小预算测 2 周，ACOS 难看就停。

## 动手练习

选一个你已有自然单 + Review 积累的产品（客单价 ≤ $35 优先），做一次捡漏测试：
1. 用第三方工具挖 20 个错词 + 10 个偏门长尾词
2. 单独建一个错词/长尾广告活动，竞价设为建议竞价的 1/3，固定竞价策略
3. 跑 7-14 天，看 CPC / CTR / 转化率 / ACOS
4. 数据达标的词保留，不达标的否掉，把预算集中到有效词上

重点练的是**用数据判断"哪些低价词值得继续投"**，而不是开了就放着。
