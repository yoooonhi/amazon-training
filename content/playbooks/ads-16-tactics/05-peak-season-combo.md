# 第5节 · 旺季与综合打法

> **手册：[广告打法手册](./)**
> **本节收录**：✅ 旺季流量七分法、⚠️ 海王打法（类目广撒网）、✅ 竞争对手投放打法
> **本节目标**：掌握旺季广告的结构化布局方法，知道哪些流量转化高该重点用、哪些流量要谨慎，并能用四类广告同时卡位抢竞品。

## 这节解决什么问题

旺季（Prime Day、黑五、网一、圣诞）流量大但竞争也激烈，预算容易乱花。很多卖家旺季广告就是"加大预算 + 加大竞价"，结果预算烧光转化却没提升——因为没搞清楚**自己的流量从哪来、哪些流量值得重点投**。

另外两种综合打法也常见：
- **海王打法**（低竞价铺多类目）——号称能用极低成本广撒网，但**流量极泛、转化极差**，文档自己也承认了
- **竞品投放打法**——同时跑多种广告类型卡位竞品，激进但合规

这节把这三种打法讲透，重点在**旺季如何用结构化思维分配预算**。

> 关键认知：**旺季广告不是"加大预算"，而是"明确哪些流量转化高，把预算集中到那些流量上"**。流量七分法就是帮你做这个判断的工具。

## 核心知识

### ✅ 打法十六：旺季流量定位法（流量七分法）

**底层逻辑**：旺季来临，必须先搞清楚**自己的流量从哪来、哪些关键词流量高、哪些流量转化更好**。明确了流量来源，才能有针对性地布局广告，而不是无脑加大预算。

**流量来源工具**：可以用 Sif 等第三方工具查自己链接和竞品链接的流量来源（Sif 在查关键词来源这块比较强）。

**七种流量来源分类**：

```
以「running shoes」（跑步鞋）为例，流量来源可分为七种：

① 中性关键词流量
   → 颜色、场景等关键词
   → 如「blue running shoes」「marathon shoes」
   → 容易产生点击，需重点关注和使用

② 品牌关键词流量
   → 品牌、产品线名称
   → 如「Nike running shoes」
   → 开销较大且竞争激烈
   → ⚠️ 建议除旺季外平时谨慎使用

③ 同类商品流量（ASIN 流量）
   → 来自相似产品的详情页
   → 如买家在看 Asics 时看到你的鞋
   → ✅ 比较建议使用此流量渠道

④ 相关商品流量
   → 来自互补品的详情页
   → 如买家在看跑步袜时看到你的鞋
   → 一般建议少量考虑，根据商品关联特性决定

⑤ 类目流量
   → 来自类目浏览页
   → ✅ 很重要的流量结构，建议使用

⑥ 社区流量
   → 亚马逊 Influencer Program（网红带货）
   → ✅ 可以尝试使用

⑦ 站外流量
   → 来自亚马逊站外的引流
   → 根据公司在站外的流量布局和资源使用
```

**七种流量的旺季使用建议**：

| 流量类型 | 旺季建议 | 平时建议 |
|---|---|---|
| ① 中性关键词 | ✅ 重点投 | ✅ 重点投 |
| ② 品牌关键词 | ✅ 旺季可投（流量大） | ⚠️ 谨慎（开销大、竞争激烈） |
| ③ 同类商品（ASIN） | ✅ 重点投 | ✅ 重点投 |
| ④ 相关商品（互补） | ⚠️ 少量投 | ⚠️ 少量投 |
| ⑤ 类目流量 | ✅ 重点投 | ✅ 重点投 |
| ⑥ 社区流量 | ✅ 尝试 | ✅ 尝试 |
| ⑦ 站外流量 | 看资源 | 看资源 |

**重点：同类商品流量（ASIN 流量）**

> 关键认知：**ASIN 流量是非常建议使用的流量渠道**。它能同时触达商品推广广告（SP）、品牌推广广告（SB）、展示型推广广告（SD）三种广告类型——是性价比最高、覆盖最广的流量来源。

> 实操要点：旺季前**先用工具查清楚自己链接和竞品链接的流量结构**，然后按"重点投/谨慎用/少量投/看资源"四档分配预算，而不是均匀撒网。

### ⚠️ 打法十七：海王打法（类目广撒网）

**底层逻辑**：用足够多的 SKU 去触达足够多的流量，且消耗的流量成本较低。其中**「类目打法」是海王打法最常见的**。

**适用场景**（很窄）：
- ✅ 服饰鞋包家纺（变体多）
- ✅ 特殊场景商品（室内运动场景）
- ✅ 配件类商品（钓鱼配件等）
- ✅ 需求类商品（美容类）
- ❌ 主推品、单一 SKU、垂直精品

**具体实战操作**：

```
海王打法五步：

① 投放的 SKU 或关键词要足够多
   → 比如卖连衣裙，先选 100 个连衣裙 SKU 或多个连衣裙关键词

② 设置「合适的低价」竞价
   → 参考建议竞价
   → 如建议 $1，可尝试 $0.2-0.3
   → ⚠️ 避免远低于建议竞价（如 $0.05），会导致广告无效

③ 预算可以「高一些」
   → 流量大时建议预算设高些（如 $1000）
   → 因采用超低竞价，一天通常只花几美金到几十美金
   → 不用担心预算浪费

④ 类目选择要「慎重」
   → 选与 ASIN 有共同二级类目的高关联度类目
   → 例：卖「dog bed」可选「狗毯」等关联性高的宠物狗家具

⑤ 适用范围判断
   → 见上面的适用场景
```

**常见问题及解答**：

```
问题 1：海王打法竞价高但曝光低，怎么办？
答：可能是类目关联问题
   → 你的产品关联性过低
   → 亚马逊认为类目与产品无关，不给曝光
   → 提高 SKU 和计划投放类目的关联度

问题 2：海王打法会影响产品相关性吗？
答：不会
   → 海王打法每个关键词获得的点击数量有限（1-2 次）
   → 主要是为了引流
   → 配合精细化运营（某个关键词的点击和转化提升）反而提升相关性
```

> ⚠️ **为什么标 ⚠️**：
> 1. 海王打法**流量极泛、转化极差**是公认问题——文档自己也承认。
> 2. 主推品用海王打法会拉低整体转化率，影响 Listing 权重。
> 3. **只适合清货或配件类**——清货可以（成本低、不指望转化），主推别用。
> 4. 预算虽然花得少，但管理 50+ 个类目的人力成本不低，ROI 要算清。

### ✅ 打法十八：竞争对手投放打法

**底层逻辑**：用多种广告类型同时卡位竞品，从搜索结果页到详情页到展示位全方位抢占竞品流量。这是合规但比较激进的打法，**适合"获取流量为首要目标"的策略**。

**三类操作**：

```
竞品投放的三种操作：

① SP + SB 投放竞品品牌关键词
   → Sponsored Product + Sponsored Brand campaigns
   → 投放带有竞争对手品牌的关键词

② SP 商品投放定位竞品 ASIN
   → 用 JS 等工具拉主要关键词搜索结果页排名靠前的 ASIN
   → 用 SP 广告的 product targeting（商品投放）定位这些 ASIN

③ SP 类目定位 + Refine 细化
   → 用 SP 广告的 category targeting（类目定位）
   → 结合 Refine 功能按售价、星级筛选竞品
```

**激进版：四类广告同时跑**（获取流量为首要目标）：

```
四类广告同时卡位竞品：

1. Sponsored Product - competitor ASIN targeting
   → SP 商品投放定位竞品 ASIN

2. Sponsored Brand - competitor product targeting
   → SB 商品投放定位竞品

3. Sponsored Brand - competitor keyword targeting
   → SB 投放竞品品牌关键词

4. Sponsored Display - competitor product targeting
   → SD 商品投放定位竞品

→ 四管齐下，覆盖搜索结果页、详情页、展示位
→ 买家搜竞品、看竞品、看过竞品后都能看到你
```

**适用判断**：

| 维度 | 判断 |
|---|---|
| ✅ 适用 | 你的产品相对竞品有明显差异化优势（价格/评分/功能），且预算充足 |
| ⚠️ 谨慎 | 竞品已开品牌保护（Transparency/Project Zero），转化率可能低 |
| ❌ 不适用 | 你的产品比竞品差，蹭了词也转化不了，纯烧钱 |

> 实操要点：竞品投放前**先客观评估自己相对竞品的优势**。没优势就别硬蹭——转化率会很难看。另外建议从 1-2 类广告开始测试，数据达标再扩展到四类全开，避免一开始就烧太多预算。

## 实操要点

- **旺季广告不是加大预算，是明确哪些流量转化高**：用流量七分法先查清自己链接和竞品链接的流量结构，按"重点投/谨慎用/少量投/看资源"四档分配预算。
- **ASIN 流量是最值得重点投的**：能同时触达 SP/SB/SD 三种广告类型，性价比和覆盖面都最高。
- **海王打法只适合清货或配件类**：流量极泛、转化极差，主推品别用，会拉低整体转化率影响 Listing 权重。
- **竞品投放要先评估自己相对竞品的优势**：没优势蹭了也不转化；从 1-2 类广告开始测试，达标再扩到四类全开。
- **旺季前一周就要把广告结构搭好**：旺季当天才调整广告已经晚了，系统学习新广告活动需要时间。

## 动手练习

**旺季广告布局练习**：
1. 选一个你的主力产品，用第三方工具（如 Sif）查它的流量来源结构
2. 按"七种流量来源"给自己的流量分类，判断每种流量占比
3. 写出旺季广告预算分配方案：
   - 重点投（占 60% 预算）：哪些流量？
   - 谨慎用（占 20% 预算）：哪些流量？
   - 少量投（占 10% 预算）：哪些流量？
   - 看资源（占 10% 预算）：哪些流量？
4. 旺季前 7-10 天把广告结构搭好，旺季当天只做竞价微调，不做结构调整

重点练的是**用结构化思维规划旺季广告**，而不是临时拍脑袋加大预算。
