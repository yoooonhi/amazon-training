# 初级1.6 品牌广告与展示广告（SB/SD）

> **初级1：广告优化实战**
> **本课目标**：知道 SB 和 SD 各自干什么、什么时候该开、怎么和 SP 配合，而不会被"还有两种广告没开"的焦虑驱动着盲目铺开烧钱。

## 这节课解决什么问题

初级1 的前五课，全程讲的都是 **Sponsored Products（SP，商品推广）**——搜索词、否定、Placement、ACOS、周度清单。SP 是亚马逊广告的基本盘，90% 的运营只需要把 SP 跑顺就够吃。

但你在后台会看到另外两种广告：**Sponsored Brands（SB，品牌推广）** 和 **Sponsored Display（SD，展示推广）**。新手最常见的两种错误反应：

- **错误 A：完全无视**——"我 SP 还没玩明白，这俩更复杂，不碰"。结果错失了品牌词防御和再营销的增量。
- **错误 B：盲目铺开**——"三种广告一起开，流量最大化"。结果 SP 还没跑顺，预算被 SB/SD 分散，ACOS 全线失控。

这节课给你一个清晰的判断框架：**SB 和 SD 是 SP 的放大器，不是替代品。SP 没跑顺之前，别碰它们。** 以及，当你 SP 跑顺之后，什么时候该开、怎么配预算、怎么选 audiences。

## 核心知识

### 先认清三种广告的定位

| 广告类型 | 展示位置 | 触发逻辑 | 核心作用 | 适合阶段 |
|---|---|---|---|---|
| **SP（商品推广）** | 搜索结果 + 商品详情页 | 买家搜某个关键词 | 抢搜索流量、精准转化 | **所有阶段**，基本盘 |
| **SB（品牌推广）** | 搜索结果**顶部**头条 + 品牌旗舰店 | 买家搜**品牌词或品类词** | 品牌曝光、防御品牌词、引流旗舰店 | SP 跑顺后，有品牌词搜索量时 |
| **SD（展示推广）** | 商品详情页侧栏 + 站外（第三方网站/App） | **再营销**（看过/买过你的人）+ 上下文定向 | 追回流失客户、防御自己 ASIN、抢竞品流量 | 有稳定流量和复购逻辑后 |

> 一句话区分：**SP 是"别人搜你时你出现"，SB 是"你霸占搜索最顶部告诉别人你是谁"，SD 是"看过你的人没买，你去追他"。**

### SB：什么时候开，开在哪里

SB 的核心价值有两个，按优先级：

```
SB 的两个用法（优先级从高到低）：

① 防御品牌词（最高优先级）
   - 当有人搜你的品牌名时，SB 头条位置必须是你
   - 否则竞品会用他的 SP/SB 抢你的品牌词搜索流量
   - 这是最该开的 SB，品牌词转化率通常极高（买家本来就是来找你的）

② 品类词曝光（进阶）
   - 买家搜"yoga mat"这种品类词时，用 SB 头条占位
   - 适合产品在品类里有一定竞争力、想从头部截流
```

**开 SB 的前提条件**：
- 品牌已备案（SB 是品牌专属功能）
- 品牌词已有自然搜索量（去 Brand Analytics 或搜索结果看，搜你品牌名的人多不多）
- SP 已经跑顺（ACOS 达标、转化稳定）

> 铁律：**如果还没人搜你的品牌名，开 SB 品牌词广告就是空转。** 品牌词搜索量是 SB 的地基，先通过 SP 把产品做起来、攒出品牌搜索量，再开 SB 防御。

### SD：再营销的 audiences 怎么选

SD 最有价值的功能是**再营销（Retargeting）**——把看过你商品但没买的人再追回来。选 audiences 是 SD 的核心：

| Audience 类型 | 含义 | 什么时候用 |
|---|---|---|
| **Viewed（浏览过）** | 过去 7/14/30 天看过你商品详情页的人 | 最常用，追回"看了没买"的流量 |
| **Purchased（购买过）** | 买过你产品的人 | 推新品/复购型产品/配件 |
| **Competitor（竞品）** | 浏览过同类竞品的人 | 抢竞品流量（但成本高，需谨慎） |

```
SD 再营销的选法：
- 新手优先选「Viewed 14天」——追回近期看过没买的，最容易转化
- 「Competitor」要小心：抢竞品流量 CPC 通常很高，转化率不一定撑得住
- 「Purchased」适合复购产品（耗材、补给品），低频复购产品别用
```

> 认知：**SD 的本质是"挽回"和"防御"，不是"拓新"。** 它追的是已经和你产品有过接触的人。如果你产品本身流量就少（每天访客 < 50），SD 的 audiences 池子太小，跑了也是空转。

### SP / SB / SD 的预算分配比例

当你三种都开时，预算怎么分？对绝大多数处于"初级"阶段的运营：

```
推荐预算分配（SP 已跑顺的前提下）：
  SP  : 70%   ← 基本盘，永远占大头
  SB  : 20%   ← 防御品牌词为主
  SD  : 10%   ← 再营销试水

绝对不要：三种平均分（33/33/33）
→ SP 是出单主力，被分走预算会导致整体 ACOS 恶化
```

| 阶段 | SP | SB | SD | 说明 |
|---|---|---|---|---|
| SP 还没跑顺 | 100% | 0% | 0% | 别分心，先把 SP 跑到 ACOS 达标 |
| SP 跑顺，有品牌搜索量 | 70% | 20% | 10% | 开始防御品牌词 + 再营销试水 |
| 成熟品牌，多 SKU | 60% | 25% | 15% | SB 做品类占位，SD 做深度再营销 |

> 最常见的翻车：**SP 还没跑顺（ACOS 还很高、转化还不稳），就急着开 SB/SD。** 三种广告一起烧，每一种都预算不足、数据稀疏，根本优化不了。广告类型是"一层一层加上去的"，不是"一次全开"。

### 判断"SP 跑顺"的标准

到底什么算"SP 跑顺"了，可以开 SB/SD 了？用这几个硬指标自检：

| 指标 | 达标标准 | 没达标说明 |
|---|---|---|
| SP 整体 ACOS | 在你的目标值以内（如 < 25%） | ACOS 还高，优化空间还在 SP 内部 |
| 核心词自然排名 | 稳定在前 3 页（最好首页） | 排名没稳，广告一停就没流量，铺 SB/SD 无意义 |
| 日均广告出单 | 稳定（连续 2 周波动不大） | 出单不稳定，数据都不准，开 SB/SD 只会更乱 |
| 品牌词搜索量 | Brand Analytics 里能搜到自己品牌 | 没人搜你品牌，SB 防御谁？ |

> 如果上面四条有任何一条没达标——**继续优化 SP，别开 SB/SD。** 这不是"开得越多流量越大"，是"基础没打好，上层建筑全是空中楼阁"。

## 实操要点

- **SB/SD 是 SP 的放大器，不是替代品**：SP 是基本盘，占预算 70%+，SP 没跑顺别碰 SB/SD。
- **SB 优先开品牌词防御**：有人搜你品牌名时，头条必须是你自己，否则被竞品截流。品牌词转化率极高。
- **SD 新手选「Viewed 14天」再营销**：追回看过没买的流量最容易转化，竞品 audiences 谨慎用（CPC 高）。
- **预算分配 7:2:1**：SP 70% / SB 20% / SD 10%，永远不要三种平均分。
- **开 SB/SD 前自检四条**：SP ACOS 达标、核心词排名稳、出单稳、品牌词有搜索量。任一不达标就继续磨 SP。

## 动手练习

先做一个判断：你的 SP 跑顺了吗？打开你的广告后台，对照本课的"SP 跑顺"四条标准逐项打勾（SP ACOS / 核心词排名 / 出单稳定性 / 品牌词搜索量）。

- 如果有任何一条没达标——写下你现在最该优化的 SP 动作是什么（参考初级1.4 的 ACOS 诊断树），SB/SD 暂缓。
- 如果四条都达标——去 Brand Analytics 搜一下你的品牌名，看有没有搜索量。如果有，规划一个 SB 品牌词防御广告（预算先给广告总预算的 15%-20%），SD 先开一个「Viewed 14天」的小预算再营销测试（10%）。

把这个判断结论和动作记下来。SB/SD 不是"现在必须开"，是"SP 跑顺后水到渠成"。
