# 初级4.4 月度促销表（实操）

> **初级4：定价与促销策略**
> **本课目标**：把前三课（竞品跟踪、促销节奏、价格弹性）串成一张可执行的"月度定价+促销作战表"，学完能直接输出下个月的价格和促销计划。

## 这节课解决什么问题

前三课你分别学了：怎么跟踪竞品价格、怎么设计促销节奏、怎么测价格弹性。但很多运营学完单独的知识点，回到日常还是"今天想起来挂个 Coupon、明天听说 Deal 能冲量就报一个"——知识点是散的，没有拧成一股绳。

结果是：每月做了什么促销、为什么做、花了多少钱、换回了什么，一笔糊涂账。月底复盘时，老板问"这个月定价和促销的策略是什么"，答不上来。

问题出在：**缺一张把所有动作串起来的作战表。** 这节课就给你这张表——它不是新知识，而是前三课的"集成应用"。把竞品跟踪、促销节奏、弹性测试、利润测算、广告配合全部装进一张月度规划表里，每周照着执行、每月复盘迭代。学完这一课，你就能独立输出一个 SKU 一个月的完整定价与促销计划。

## 核心知识

### 月度作战表的五个模块

一张完整的月度定价+促销作战表，必须包含这五块，缺一不可：

| 模块 | 内容 | 对应的前置课 |
|---|---|---|
| **① 基础信息与利润护栏** | 产品当前售价、最优价、每单净利、利润护栏（净利率红线） | 入门6 + 初级4.3 |
| **② 竞品价格跟踪** | 核心 3 个竞品的售价/Coupon/Deal/到手价/排名 | 初级4.1 |
| **③ 每周促销安排** | 本月每周做什么促销（或正价期）、目的、工具 | 初级4.2 |
| **④ 利润测算** | 每个促销的真实成本（折扣+费+广告加投+退货）和盈亏平衡量 | 初级4.2 + 入门6 |
| **⑤ 广告配合节奏** | 促销期广告加投、正价期广告回收 | 初级1 |

> 核心理念：**定价和促销不是孤立动作，是一张需要每周更新、每月复盘的活表。** 不更新的表等于没有。

### 模块一：基础信息与利润护栏（每月初填一次）

先把产品的"地基信息"定下来，这是后面所有决策的基准：

```
产品：XXX（ASIN：B0XXXXX）
当前售价：€14.99    弹性测试最优价：€14.99（初级4.3测出）
当前每单净利：€3.00  当前净利率：20%
利润护栏：净利率不低于 15%（即每单净利不低于 €2.25）
当前月销：约 1200 单   月总利润：约 €3600
库存可用天数：45 天
评论数：86    核心词排名：稳定首页 #420 已 3 周
```

**利润护栏是红线**：任何促销或调价方案，先重算利润链（入门课），确认净利率还在护栏之上才能执行。跌破护栏的一律否决。

### 模块二：竞品价格跟踪表（每周更新）

每月初填一次基线，之后每周一更新（初级4.1 的模板）：

```
| 竞品ASIN | 售价 | Coupon | Deal | 到手价 | BuyBox | 评论数 | 排名 | 本周变化 | 我的判断 |
|----------|------|--------|------|--------|--------|--------|------|----------|----------|
| B0XX01   |13.99 | €1券   | 无   | 12.99  | 对方   | 312    | #380 | 无变化   | 价差€2，我偏高，盯转化 |
| B0XX02   |15.99 | 无     | 无   | 15.99  | 对方   | 98     | #610 | 涨了€1   | 新品冲完涨价，我不必跟 |
| B0XX03   |12.99 | 无     | 7DD中| 10.99  | 对方   | 450    | #290 | 促销中   | 短期冲击，促销结束会回落 |
```

**"我的判断"列最重要**：每周看完竞品，必须写一句话结论——"跟"还是"不跟"、为什么。这是把跟踪变成决策的关键动作。

### 模块三：每周促销安排（月度规划）

把一个月的四周排出来，每个促销都要有"目的"，正价期也要明确标出来（初级4.2）：

```
| 周次 | 促销动作 | 目的 | 工具 | 折扣力度 | 预计销量 |
|------|----------|------|------|----------|----------|
| W1   | 正价期 + 挂€1 Coupon | 提点击率、稳转化 | Coupon | 到手€13.99 | 300单 |
| W2   | 报 7-Day Deal | 冲核心词销量/排名 | 7DD | 活动价€11.99 | 500单 |
| W3   | 正价恢复期 | 促销后回调、沉淀排名 | 无促销，价格分步回€14.99 | - | 280单 |
| W4   | 正价期 + 小额Coupon | 维持、观察弹性测试效果 | Coupon | 到手€13.99 | 300单 |
```

**注意三点纪律**：
1. 每个促销必须写"目的"，说不清就别做（初级4.2）。
2. 两次促销之间留正价期（W3），不能连轴转。
3. 促销后必有价格恢复期，分步回调（€11.99→€13.49→€14.99）。

### 模块四：每个促销的利润测算

以 W2 的 7-Day Deal 为例，按初级4.2 的"真实成本四项"算清楚：

```
W2  7-Day Deal 利润测算：
  平时售价 €14.99，平时每单净利 €3，平时7天卖 80 单
  活动价 €11.99（每单折扣 €3），活动费 €35
  预计7天卖 200 单
  广告加投：促销期日预算 €20→€45，7天多花 €175
  退货率从 8% 升到 11%

  折扣成本   = €3 × 200 = €600
  活动费     = €35
  广告加投   = €175
  退货增量   = 200 × 3% × €11.99 ≈ €72
  ────────────────
  真实总成本 ≈ €882
  促销后每单净利 = €3 − €3(折扣) = €0
  盈亏平衡增量销量：靠销量提升换排名长期价值

  判断：促销后单笔利润为0，必须靠冲上来的排名在W3-W4转化成
        自然订单才回本。如果W3排名立刻回落，这次促销不划算。
        → 设置验收点：W3结束时核心词排名是否仍在首页
```

> 算账纪律：**每个促销都要有这么一张测算表。** 算不出"换什么、值不值"的促销，一律不上。

### 模块五：广告配合节奏

促销和广告必须配合，不能各搞各的。把整个月的广告节奏和促销节奏对齐：

```
| 周次 | 促销状态 | 广告动作 | 原因 |
|------|----------|----------|------|
| W1   | 正价+小Coupon | 维持日常预算€20/天 | 稳定期，不烧 |
| W2   | 7DD促销中 | 预算加到€45/天，核心词提bid抢TOS | 促销期流量贵，要接住Deal流量冲排名 |
| W3   | 正价恢复 | 预算分2天从€45降到€20，别断崖 | 价格回调期，广告也平滑回收 |
| W4   | 正价+小Coupon | 维持€20/天 | 观察弹性测试、稳态运行 |
```

> 配合纪律：**促销期广告加码接流量，促销后广告分步回收。** 别促销结束了广告还按促销强度烧，那是白烧钱。

### 月度复盘：四个必答问题

月底必须复盘，用这四个问题检验这个月的定价和促销策略有没有效：

```
1. 竞品跟踪：这个月竞品有什么大动作？我应对对了吗？
   （查模块二的每周判断 vs 实际结果）

2. 促销效果：每个促销的真实成本是多少？换回了什么（排名/销量/沉淀）？
   （查模块四的测算 vs 实际数据，对账）

3. 价格弹性：我有没有做调价测试？最优价变了吗？
   （查初级4.3的记录表，更新最优价）

4. 总账：这个月总利润是多少？比上月多还是少？原因是什么？
   （定价和促销最终都要为总利润服务）
```

> 复盘纪律：**不复盘的月度计划，做三个月还是原地踏步。** 复盘是让计划越做越准的唯一方法。

## 实操要点

- **五模块缺一不可**：基础信息/护栏、竞品跟踪、促销安排、利润测算、广告配合——少一块，计划就不完整。
- **利润护栏是红线**：任何方案先重算利润链，净利率跌破 15% 一律否决，不管销量多诱人。
- **每个促销写目的、每个竞品写判断**：把"跟踪"和"促销"从动作变成决策，这是从执行到策略的分水岭。
- **促销和广告必须对齐**：促销期广告加码接流量，促销后广告分步回收，别各搞各的。
- **每月必复盘**：四个必答问题（竞品/促销/弹性/总账），不复盘就重复犯错。

## 动手练习

这是整个初级4模块的综合实操。拿你一个在售产品（评论 50+、排名较稳的），完整做一份下个月的定价+促销作战表。

要求产出：
1. **模块一**：填产品基础信息和利润护栏（用入门课的利润链算每单净利，确认护栏）。
2. **模块二**：找 3 个核心竞品，填竞品价格跟踪表，写出每个竞品的"我的判断"。
3. **模块三**：排下个月四周的促销安排，每个促销写清目的和工具，必须留正价期。
4. **模块四**：对计划里的每个促销，做真实成本四项测算，写明"换什么、值不值"。
5. **模块五**：把广告节奏和促销节奏对齐，写出每周的广告动作。

做完后，把这张表存档。一个月后回来，用"四个必答问题"复盘：竞品判断对了没、促销换回什么了、最优价变没变、总利润增减多少。坚持做 3 个月，你的定价和促销会从"凭感觉"彻底变成"有策略"。
