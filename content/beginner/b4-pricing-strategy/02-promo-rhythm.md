# 初级4.2 促销节奏设计

> **初级4：定价与促销策略**
> **本课目标**：能区分 4 种促销工具的特点和适用场景，设计有节奏的促销计划，并在上促销前算清这次促销的真实利润（含折扣+广告加投+退货）。

## 这节课解决什么问题

入门课教过你"促销有成本、要算盈亏平衡增量销量"。但很多运营上促销还是凭感觉——Listing 没流量了，挂个 Coupon；这周想冲一下排名，报个 Deal；同事说 Prime 折扣管用，又开一个。结果一个月下来，产品几乎天天在打折，买家学会"等折扣才买"，正价反而卖不动了。

更危险的是促销叠加：Coupon 忘了关就报了 Deal，两个折扣摞在一起，到手价直接跌破成本，卖一单亏一单，亏了 $500 才发现。

问题出在两个地方：**第一，不懂每种促销工具的脾气**（Coupon、LD、7DD、PED 各有适用场景，乱用会互相打架）；**第二，没有"促销是有限弹药"的意识**（促销不是想促就促，它是"有目的的让利"，每个促销都要能回答"我让这笔利，换什么回来"）。这节课把这两件事讲透。

## 核心知识

### 四种促销工具的特点对比

| 工具 | 折扣形式 | 展示位置 | 持续时间 | 成本 | 最适合的场景 |
|---|---|---|---|---|---|
| **Coupon** | 金额或百分比 | 搜索结果页绿色标记 + Listing 页 | 1-90 天可设 | 折扣 + 每次兑换 $0.60 手续费 | 提升点击率（绿色标记）、新品冷启动稳转化 |
| **Prime Exclusive Discount (PED)** | 百分比 off | Prime 专属页面/大促期间搜索结果红标 | 自设周期，大促期间强制 20% off 起 | 仅折扣，无固定费 | Prime Day / 黑五等大促、日常吸引 Prime 会员 |
| **Lightning Deal (LD)** | 固定折扣价 | Deal 专页 + 推送 | 数小时（4-12 小时） | 折扣 + 固定活动费（美国站平日 $150-$500，旺季 PD/黑五可翻倍至 $500-$1000） | 短时间冲销量/冲排名、清少量尾货 |
| **7-Day Deal (7DD)** | 固定折扣价 | Deal 专页 | 7 天 | 折扣 + 固定活动费（比 LD 低） | 持续一周稳排名、给新品拉曝光 |

> 认知：**这四个工具不是"哪个折扣大选哪个"，而是"我要达成什么目的，选对工具"。** 想提点击率用 Coupon，想冲瞬间排名用 LD，想稳一周排名用 7DD，想蹭大促流量用 PED。

### 促销叠加的禁忌：哪些能叠，哪些会要命

亚马逊的折扣机制里，**多种促销可能同时生效，折扣是累加的**。新手最常踩的坑就是叠加。

```
叠加规则（必须背下来）：
✗ Coupon + Sale Price（促销价）     → 可叠加，最容易亏死
✗ Coupon + Lightning Deal/7DD       → Deal 会与 Coupon 叠加，灾难级
✗ PED + 其他折扣                     → 大促期间要特别小心
✓ 同类型一般取其一，不同类型多会累加
```

最经典的事故场景：

```
原价 $14.99
挂着一个 $2 的 Coupon（忘了关）
又报了一个 7-Day Deal，活动价 $10.99
买家实付 = $10.99 − $2 = $8.99
实际到手价比成本还低 $1.5
→ 卖了 300 单，亏 $450，外加活动费
```

> 铁律：**报任何 Deal 之前，先去 Listing 关掉所有 Coupon 和 Sale Price。** 上促销前的"清场"动作和促销本身一样重要。

### 促销的真实成本：不止折扣

入门课教过盈亏平衡增量销量，这里要加三个常被漏算的成本：

```
一次促销的真实总成本 =
  折扣成本（折扣金额 × 预计销量）
+ 活动固定费（LD/7DD 才有）
+ 广告加投（促销期间通常要加预算配合）
+ 预期退货增量（促销冲量常带来更高退货率）
```

很多人只算前两项，结果促销"看起来赚钱、实际亏"。

举个例子，一个 7 天 Deal：

```
平时售价 $14.99，平时每单净利 $3，平时 7 天卖 80 单
7DD 活动价 $11.99（每单折扣 $3），活动费 $35
预计 7 天卖 200 单
广告加投：促销期间每日预算从 $20 加到 $50，7 天多花 $210
促销期间退货率从 8% 升到 12%（冲量带来的）

算账：
  折扣成本 = $3 × 200 = $600
  活动费   = $35
  广告加投 = $210
  退货增量 = 200 单 × (12% − 8%) × $11.99(损失售价+回收成本摊) ≈ $96
  ────────────────
  真实总成本 ≈ $941
  促销后每单净利 = $3 − $3(折扣) = $0
```

注意这里**促销后每单净利是 $0**——意味着光靠折扣后卖出的单，每单都不赚钱，必须靠"销量提升带来的自然排名长期价值"才回得了本。如果促销一停排名立刻回落，这次促销就是纯亏。

> 算账纪律：**促销前必须把折扣 + 活动费 + 广告加投 + 退货增量四项全算进去。** 少算一项，你看到的"利润"就是假的。

### 促销前/中/后的准备清单

促销不是"点一下开始"就完事。一个设计良好的促销，前中后都有动作：

| 阶段 | 必做动作 | 为什么 |
|---|---|---|
| **促销前 3-5 天** | ① 关掉所有 Coupon/Sale Price 防叠加 ② 检查库存是否够撑过促销期（断货比不促销更伤） ③ Listing 图文/评论/A+ 处于最佳状态（促销带来流量，转化靠 Listing） ④ 广告预算上调计划定好 ⑤ 写好促销后的价格恢复方案 | 清场 + 备货 + 备战 |
| **促销中** | ① 每天盯销量和广告 ACOS ② 监控是否断货（断货立刻暂停促销） ③ 监控 Buy Box 是否在自己手里 ④ 竞品有没有跟风做活动 | 防意外，保转化 |
| **促销后** | ① 价格按计划恢复（分步回调：$11.99→$13.49→$14.99，比直接回原价平滑） ② 广告预算分 2-3 天降回正常，别断崖 ③ 看自然排名有没有稳住（促销沉淀的核心指标） ④ 复盘：这次促销的真实成本 vs 带来的增量销量和排名提升 | 收尾 + 复盘 |

### 促销是"有目的的让利"

最重要的一条认知：**每个促销，开始前必须能回答"我让这笔利，换什么回来"。**

| 让利目的 | 合理的促销 | 不合理 |
|---|---|---|
| 新品冷启动，要第一批转化 | 小额 Coupon + 少量广告 | 大额 Deal 砸在新品上（评论少，转化撑不住，钱白花） |
| 冲某个关键词的销量/排名 | 配合精准广告的 7DD | 无广告配合的纯折扣（排名冲不上去） |
| 大促蹭平台流量 | PED（Prime Day/黑五） | 大促期间不参加，白白让竞品抢流量 |
| 清库存退场 | 深度折扣的 LD/Outlet | 用 Coupon 慢慢清（太慢，库存耗不起） |

> 如果一个促销，你说不清"换什么"，就别做。**促销不是清库存的万能药，更不是"没想法了就打折"的遮羞布。**

## 实操要点

- **每个促销先定目的**：冲排名？冷启动？蹭大促？清库存？目的不同工具不同，说不清目的就别做。
- **报 Deal 前必清场**：关掉所有 Coupon 和 Sale Price，叠加是头号杀手，亏死了才发现最蠢。
- **真实成本算四项**：折扣 + 活动费 + 广告加投 + 退货增量，少算一项就是自欺欺人。
- **促销前就写好收尾方案**：价格分步回调、广告分天降回、判断自然排名有没有沉淀。不写收尾方案的促销，结束就是乱局的开始。
- **促销别连轴转**：产品天天在打折，买家会学会"等折扣才买"，正价卖不动。两次促销之间至少留 2-3 周正价期。

## 动手练习

你有一个产品，平时售价 $14.99，平时每单净利 $3，平时一周卖 80 单。你想下周报一个 7-Day Deal，活动价 $11.99，活动费 $35，预计促销 7 天卖 200 单。促销期间你计划广告每日预算从 $20 加到 $50。

请完成：
1. 算这次促销的真实总成本（折扣 + 活动费 + 广告加投，退货增量按你的类目估一个，比如多 4 个百分点）。
2. 促销后每单净利是多少？盈亏平衡增量销量是多少？
3. 这次促销"换什么回来"？（冲排名？清库存？）如果促销一停排名就回落，这次促销值不值？
4. 写一份促销前/中/后的准备清单，重点写出促销后价格怎么分步回调、广告怎么降回。

把这份方案存档，促销结束后回来对比实际数据——你会发现自己第一次算的和实际跑出来的差距，这就是成长。
