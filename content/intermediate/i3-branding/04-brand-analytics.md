# 中级3.4 Brand Analytics 实战

> **中级3：品牌化基础**
> **本课目标**：学完能用 Brand Analytics 的四张核心报表（搜索词表现/重复购买/商品比较/人群画像）做运营决策，把"免费的数据金矿"真正用起来。

## 这节课解决什么问题

前几节你学了品牌备案、A+、旗舰店。但 Brand Analytics 是品牌备案解锁的"隐藏金矿"——它提供的数据是其他渠道买不到的，很多运营却从来没用过。

很多运营的真实状态是：备案了，Brand Analytics 解锁了，但从来没点开过；或者点开了，看到一堆报表不知道看什么、看完不知道做什么。这节课解决三个问题：四张报表各看什么、怎么解读、看完做什么动作。

学完你能用 Brand Analytics 回答：客户在搜什么词、我的客户还买什么、客户拿我跟谁比、我的客户长什么样——这四个问题直接指导选品、定价、广告、Listing 优化。

## 核心知识

### Brand Analytics 的四张核心报表

| 报表 | 回答什么问题 | 核心价值 |
|---|---|---|
| **① 亚马逊搜索词表现** | 客户在搜什么？哪些词点击多转化高？ | 选词、广告优化 |
| **② 重复购买行为** | 客户复购率多少？多久复购？ | 复购产品识别、库存预测 |
| **③ 商品比较和替代行为** | 客户拿我跟谁比？最后买了谁？ | 竞品识别、差异化 |
| **④ 人口统计信息（人群画像）** | 我的客户是谁？年龄/收入/地区 | 受众定位、 Listing 文案 |

> 金句：**Brand Analytics 是亚马逊"免费送"的调研公司。** 这些数据在外面花几十万都买不到，备了案却不用是最大的浪费。

### 报表①：亚马逊搜索词表现

这张报表告诉你"客户在搜什么词、这些词表现如何"。

| 数据维度 | 含义 | 怎么用 |
|---|---|---|
| **搜索词排名** | 该词在所有搜索词中的热度排名 | 找高热度词 |
| **搜索量指数** | 相对搜索量（非绝对值） | 比较词与词的热度 |
| **点击率** | 搜这个词的客户点产品的比例 | 判断词的购买意图 |
| **转化率** | 点击后购买的比例 | 判断词的转化质量 |
| **点击最多的产品** | 这个词点击都给了谁 | 识别竞品 |

```
搜索词表现报表的三大用法：

用法1：找广告词
  → 筛选「高搜索量 + 高转化率」的词
  → 这些是优质广告词（量大且能转化）
  → 加入手动精准广告

用法2：找选品方向
  → 看「高搜索量 + 低点击率」的词
  → 说明市场需求大但现有产品不满足（点击率低）
  → 这可能是选品机会（有需求没好产品）

用法3：识别竞品
  → 看某个核心词「点击最多的产品」
  → 这些就是你的直接竞品
  → 可以投ASIN定位广告截他们的流量

⚠️ 注意：搜索量指数是相对值，不是绝对搜索量
   指数100不代表100次搜索，只是相对其他词的热度
```

### 报表②：重复购买行为

这张报表告诉你"客户复购情况"，对判断产品是不是"复购型"很关键。

| 数据维度 | 含义 | 怎么用 |
|---|---|---|
| **复购率** | 买过的客户再次购买的比例 | 判断产品复购属性 |
| **平均复购周期** | 多久复购一次 | 预测补货需求 |
| **复购客户占比** | 老客户vs新客户比例 | 评估品牌忠诚度 |

```
复购行为报表的用法：

用法1：识别复购型产品
  → 复购率 > 15% 的产品是「复购型」
  → 这类产品值得重点投入（客户终身价值高）
  → 可以做订阅（Subscribe & Save）、套装

用法2：预测补货需求
  → 看平均复购周期
  → 例：平均30天复购 → 一个月后这批客户又要买
  → 提前备货，避免复购客户来了断货

用法3：评估品牌健康度
  → 复购客户占比 > 30% → 品牌忠诚度高
  → 复购客户占比 < 10% → 纯靠拉新，品牌没沉淀
  → 提升复购：产品质量、售后服务、套装优惠

判断标准：
高复购产品（消耗品/日用品）：复购率 > 20%
中复购产品（工具/配件）：复购率 10-20%
低复购产品（耐用品/一次性）：复购率 < 10%
```

### 报表③：商品比较和替代行为

这张报表是"竞品情报"——告诉你客户拿你的产品跟谁比，最后买了谁（包括没买你的）。

| 数据维度 | 含义 | 怎么用 |
|---|---|---|
| **被比较的竞品 ASIN** | 客户看你的产品时还看了谁 | 识别直接竞品 |
| **比较后客户买了谁** | 看完之后最终下单的是谁 | 判断竞争胜负 |
| **你的胜率** | 比较后买你的比例 | 评估竞争力 |

```
商品比较报表的三大用法：

用法1：识别真实竞品
  → 客户拿你跟谁比，谁就是你的真实竞品
  → 这比用工具找竞品更准（这是客户的真实行为）
  → 把这些竞品加入ASIN定位广告

用法2：分析竞争胜负
  → 你跟竞品A比，胜率只有20% → 竞品A是你的克星
  → 研究竞品A：价格？功能？评论？图片？
  → 找到差距，针对性优化

用法3：找差异化机会
  → 你总是输给某个竞品
  → 看客户为什么选他不选你
  → 在Listing/A+里强化你的差异化卖点

胜率标准：
> 50%：竞争力强，可以扩大广告
30-50%：中等，需要优化
< 30%：竞争力弱，要么大幅优化要么换赛道
```

### 报表④：人口统计信息（人群画像）

这张报表告诉你"你的客户是谁"。

| 数据维度 | 含义 | 怎么用 |
|---|---|---|
| **年龄分布** | 客户的年龄段 | 调整Listing文案语气 |
| **收入分布** | 客户的收入水平 | 定价策略参考 |
| **教育水平** | 客户的教育背景 | 文案复杂度 |
| **地理位置** | 客户集中的地区 | 仓储/物流策略 |
| **性别** | 客户性别比例 | 产品设计/营销 |

```
人群画像的用法：

用法1：优化Listing文案
  → 客户主要是25-34岁 → 用年轻化语言
  → 客户主要是高收入 → 强调品质而非价格
  → 客户教育水平高 → 可以用专业术语

用法2：广告人群定位
  → 知道客户画像后
  → 在SD展示广告里做人群定向
  → 把广告精准投给「像你客户」的人群

用法3：选品/产品迭代
  → 客户集中在某个地区
  → 针对该地区需求开发产品
  → 例：客户多在阳光充足的州 → 开发防晒/遮阳产品

⚠️ 注意：人群画像数据有滞后性（基于历史购买）
   且部分品类数据样本少，仅供参考，不要过度依赖
```

### 四张报表的配合使用

四张报表不是孤立看，要配合起来形成洞察：

```
四报表联动的实战场景：

场景1：选品验证
  ① 搜索词报表 → 找到高搜索量低点击的词（需求大产品少）
  ② 商品比较报表 → 看现有产品的胜率（有空间就进）
  ③ 人群画像 → 确认目标人群规模
  → 综合判断值不值得做这个产品

场景2：广告优化
  ① 搜索词报表 → 找高转化词加入手动精准
  ② 商品比较报表 → 找竞品ASIN投定位广告
  ③ 人群画像 → 在SD广告做人群定向
  → 三报表配合，广告精准度大幅提升

场景3：竞品分析
  ① 商品比较报表 → 识别客户拿你跟谁比
  ② 搜索词报表 → 看这些竞品在什么词上抢你流量
  ③ 人群画像 → 看竞品的客户跟你客户是否重叠
  → 制定针对性防守/进攻策略
```

### Brand Analytics 的使用频率

不同报表的更新频率和查看频率不同：

| 报表 | 更新频率 | 建议查看频率 |
|---|---|---|
| 搜索词表现 | 每周 | 每周1次（配广告优化） |
| 重复购买 | 每月 | 每月1次（配库存预测） |
| 商品比较 | 每周 | 每周1次（配竞品监控） |
| 人群画像 | 每季度 | 每季度1次（配战略调整） |

## 实操要点

- **四张报表各有分工**：搜索词选词、复购看产品属性、商品比较看竞品、人群画像看客户。
- **搜索词报表找"高搜索+高转化"词**：这些是优质广告词，加手动精准。
- **商品比较报表识别真实竞品**：客户拿你跟谁比，谁就是真竞品，投 ASIN 广告。
- **复购率 > 15% 是复购型产品**：值得重点投入，做套装/订阅，客户终身价值高。
- **四报表要联动**：选品、广告、竞品分析都要综合多张报表，不能只看一张。
