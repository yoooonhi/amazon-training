# 中级2.3 SKU 精简与聚焦（砍掉哪些、为什么、何时砍）

> **中级2：多SKU与产品线管理**
> **本课目标**：学完能用明确的标准判断哪些 SKU 该砍、什么时候砍、怎么砍，把精力聚焦到能赚钱的产品上，而不是被一堆尾巴产品拖垮。

## 这节课解决什么问题

上一节你学会了资源分配。但还有个反向问题：当 SKU 太多、资源不够分的时候，怎么办？很多运营的真实状态是——舍不得砍。每个 SKU 都投入过心血，总觉得"再给它一点时间就能起来"，结果 20 个 SKU 里有 12 个是半死不活的尾巴，占用库存、占用广告预算、占用精力，拖垮整个盘子。

**SKU 不是越多越好。** 中级运营必须学会"做减法"——砍掉该砍的，聚焦能赚钱的。这节课给你一套砍 SKU 的标准、时机、流程，让你下得了手。

## 核心知识

### 为什么必须砍 SKU

尾巴 SKU 的隐藏成本远超你的想象：

| 隐藏成本 | 具体损失 | 量化 |
|---|---|---|
| **资金占用** | 库存压款 + 仓储费 | 一个滞销 SKU 压 2-5 万，20 个就是 40-100 万 |
| **精力分散** | 每天看一堆不出单的产品 | 每个尾巴占 5-10% 精力，12 个吃掉 60%+ |
| **广告浪费** | 给不出单的 SKU 投广告 | 每月白烧 10-30% 广告预算 |
| **店铺指标拖累** | 低转化 SKU 拉低整体转化率 | 影响其他产品的权重 |
| **心理负担** | 每天看到一堆红色不出单 | 决策疲劳，影响判断力 |

> 金句：**砍掉一个不赚钱的 SKU，比新上一个赚钱的 SKU 价值更高。** 因为它释放的资金、精力、广告预算可以投到能赚钱的地方。

### 砍 SKU 的判定标准

不是"不出单就砍"，要按多维标准综合判定：

| 维度 | 砍掉标准 | 保留观察 | 重点投入 |
|---|---|---|---|
| **月销量** | < 10 单（持续3个月） | 10-30 单 | > 30 单 |
| **毛利率** | < 10% | 10-25% | > 25% |
| **ACOS** | > 100%（优化后仍高） | 50-100% | < 50% |
| **库存周转** | > 180 天 | 90-180 天 | < 90 天 |
| **评论星级** | < 3.5 星（且无法修复） | 3.5-4.0 星 | > 4.0 星 |
| **生命周期** | 衰退期且无救 | 成熟期 | 成长/导入期 |

```
砍 SKU 的「三振出局」规则：

一个SKU符合以下任一条件，立即进入「砍单流程」：
✗ 持续3个月月销 < 10单，且优化无效
✗ 毛利率 < 10%（怎么算都亏钱）
✗ 库存周转 > 180天（资金压死）
✗ 评论星级 < 3.5 且产品缺陷无法修复
✗ ACOS长期 > 100% 且搜索词优化无效

符合以下条件，进入「观察期」（给3个月）：
⚠ 月销10-30单但利润薄
⚠ 库存周转90-180天
⚠ 评论3.5-4.0星但有改进空间
→ 观察3个月无改善 → 砍
```

### 不同 SKU 角色的砍单标准

三款（引流/利润/形象）的砍单标准不同：

| 角色 | 砍掉标准 | 特别说明 |
|---|---|---|
| **引流款** | 销量崩塌（月销掉 50%+）且带不动利润款流量 | 引流款不考核利润，但考核"是否还能引流" |
| **利润款** | 毛利率持续 < 15%、ACOS > 80% 且优化无效 | 利润款门槛最高，能救尽量救 |
| **形象款** | 不考核（它的使命是撑品牌） | 除非完全不出单且占资源，否则保留 |

### 砍 SKU 的时机选择

砍 SKU 不是想砍就砍，时机很重要：

| 时机 | 是否适合砍 | 原因 |
|---|---|---|
| **旺季前** | ❌ 不适合 | 旺季还能冲一波，等旺季后再砍 |
| **旺季中** | ❌ 不适合 | 砍了会打乱库存和广告节奏 |
| **旺季结束后（淡季初）** | ✅ 最适合 | 旺季后数据最真实，淡季有时间处理库存 |
| **季度复盘时** | ✅ 适合 | 跟资源分配节奏对齐 |
| **资金紧张时** | ✅ 必须砍 | 不砍现金流会断 |

> 最佳时机：**旺季结束后 1-2 个月**。这时有完整的旺季数据可以判断产品真实表现，淡季也有时间清库存、调资源。

### 砍 SKU 的执行流程

砍 SKU 不是"下架了之"，要按流程执行，把损失降到最低：

```
砍SKU四步流程：

第1步：判定（数据支撑）
  → 用判定标准打分，确认该砍
  → 列出砍的具体原因（数据），不是「感觉不行」

第2步：清库存（止损）
  → 库存 > 50件：开Liquidation（清算）/Outlet Deal/大额Coupon清仓
  → 库存 < 50件：低价甩卖或直接弃置（弃置费 < 继续仓储费时）
  → 目标：30-60天内清完

第3步：降广告（止血）
  → 立即停掉所有SP广告
  → 保留少量SD广告（如果有品牌曝光价值）
  → 广告预算立即转移到其他产品

第4步：下架/归档
  → 库存清完后下架Listing
  → 归档数据（销量/利润/失败原因）→ 下次选品避坑
  → 释放的广告预算、资金、精力 → 投到重点产品

⚠️ 不要：删Listing（数据有价值，留着复盘）
⚠️ 不要：突然下架（会影响店铺指标，要逐步降权）
```

### 砍 SKU 后的资源重新分配

砍掉 SKU 不是目的，释放资源重新分配才是：

```
砍掉1个SKU后，释放的资源怎么分：

资金（清库存回收的钱）：
  → 70% 投到现有利润款补货
  → 30% 作为新品开发基金

广告预算（停掉的广告费）：
  → 60% 加到表现最好的利润款（扩量）
  → 40% 加到正在冷启动的新品（加速）

精力（省下的时间）：
  → 50% 给核心利润款（深化优化）
  → 50% 给新品（加快冷启动）

原则：砍掉「不赚钱」的，把资源投到「能赚更多钱」的。
     这是聚焦，不是偷懒。
```

### SKU 精简的目标状态

健康的产品线不是 SKU 越多越好，而是"少而精"：

| 状态 | SKU 数量 | 特征 |
|---|---|---|
| ❌ 散沙型 | > 20 个 | 一堆半死不活的尾巴，没重点 |
| ⚠️ 膨胀型 | 15-20 个 | SKU 偏多，需要精简 |
| ✅ 健康型 | 8-15 个 | 三款矩阵清晰，每个都有定位 |
| ✅ 聚焦型 | 5-8 个 | 少而精，每个都是主力 |

> 目标：**8-15 个有清晰定位的 SKU，胜过 30 个平庸的 SKU。** 精力聚焦，每个产品都做到类目前列。

## 实操要点

- **用数据砍，不用感觉砍**：月销、毛利、ACOS、周转、星级，多维度打分，符合标准才砍。
- **三振出局规则**：符合任一硬标准（月销<10、毛利<10%、周转>180天等）立即进砍单流程。
- **砍 SKU 选对时机**：旺季结束后 1-2 个月最适合，有数据有时间处理。
- **砍的流程要完整**：判定→清库存→降广告→下架，不能突然下架影响店铺指标。
- **砍完资源要重分配**：释放的资金/预算/精力投到能赚钱的产品，这才是砍的目的。

## 动手练习

盘点你所有的 SKU，做一张「SKU 砍单评估表」：列出每个 SKU 的月销量、毛利率、ACOS、库存周转天数、评论星级，对照砍单标准给每个打分。找出符合"三振出局"规则的 SKU，按砍单流程制定清库存和资源重分配计划。砍完之后，统计释放的资金、广告预算、精力，重新分配到你的核心利润款和新品上。
