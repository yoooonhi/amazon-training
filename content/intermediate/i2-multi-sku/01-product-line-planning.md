# 中级2.1 产品线规划（引流/利润/形象款矩阵）

> **中级2：多SKU与产品线管理**
> **本课目标**：学完能用「引流款/利润款/形象款」矩阵思维规划产品线，而不是东一榔头西一棒子地上架，让每个 SKU 都有明确的战略定位。

## 这节课解决什么问题

初级课教的是"把一个产品做好"。但到了中级，你要从"运营一个产品"升级到"管理一条产品线"。很多运营扩品时的典型状态是：看什么火就上什么，一年上了 20 个 SKU，结果利润加起来还不如之前单打一个产品。

问题在于缺少"矩阵思维"。**不是 SKU 越多越好，而是每个 SKU 要有清晰的定位**——有的负责引流、有的负责赚钱、有的负责撑品牌。如果没有定位，20 个 SKU 互相抢资源、抢流量、抢精力，最后一堆半死不活的尾巴产品拖垮整个盘子。

这节课教你怎么用矩阵思维设计产品线，让引流款带流量、利润款赚利润、形象款撑品牌，三者配合形成健康的"产品线结构"。

## 核心知识

### 三种 SKU 的战略定位

一个健康的产品线，SKU 按战略定位分成三种角色：

| 类型 | 核心使命 | 价格带 | 毛利率 | 销量特征 | 广告角色 |
|---|---|---|---|---|---|
| **引流款** | 拉新、带流量 | 低 | 低（10-20%） | 高销量、低单价 | 主力投放，跑量 |
| **利润款** | 赚钱 | 中-高 | 高（30-50%） | 中等销量、高单价 | 精准投放，保利润 |
| **形象款** | 撑品牌、拉信任 | 高 | 中（20-30%） | 低销量、高溢价 | 少投或不投 |

> 金句：**引流款是「钩子」，利润款是「钱包」，形象款是「招牌」。** 三者缺一不可，但比例必须对。

### 健康产品线的配比结构

不同发展阶段，三种 SKU 的数量和销量配比不同：

| 产品线规模 | 引流款 | 利润款 | 形象款 | 说明 |
|---|---|---|---|---|
| **起步期（1-3个SKU）** | 1 | 1-2 | 0-1 | 先把利润款做起来 |
| **成长期（4-8个SKU）** | 1-2 | 3-5 | 1 | 引流款带流量 |
| **成熟期（9-15个SKU）** | 2-3 | 5-8 | 2-3 | 完整矩阵 |
| **超过15个SKU** | 警惕！ | — | — | 精力分散，该砍了 |

```
健康产品线的销量/利润贡献比：

销量贡献：引流款 40% / 利润款 50% / 形象款 10%
利润贡献：引流款 20% / 利润款 70% / 形象款 10%

反例（不健康的结构）：
✗ 销量全靠引流款（60%+），利润薄
✗ 利润款只有1个，它一断货全盘崩
✗ 形象款占了一半SKU，占用资金不产出
✗ 15个SKU全是「不上不下」的中庸款，没有清晰定位
```

### 引流款的设计原则

引流款不是"随便找个便宜货"，它要满足特定条件：

| 设计原则 | 标准 | 为什么 |
|---|---|---|
| 刚需、高频、低决策门槛 | 搜素量大、复购率高 | 容易转化，快速起量 |
| 价格低于类目均价 20-30% | 有明显价格优势 | 抢价格敏感型客户 |
| 毛利率虽低但不能亏 | 毛利 > 10% | 引流也要覆盖成本 |
| 跟利润款强关联 | 同品类/互补/可搭配 | 流量能溢出到利润款 |

```
引流款的终极价值：流量溢出

引流款客户买了A（引流款）→ 发现你的B（利润款）也不错 → 买B
引流款客户买了A → 进你的品牌旗舰店 → 浏览C（形象款）→ 记住品牌

关键：引流款要能「导流」到利润款和形象款
方法：A+关联模块、品牌旗舰店、组合套装（引流款+利润款打包）
```

### 利润款的设计原则

利润款是产品线的"钱包"，必须重点保护：

| 设计原则 | 标准 | 为什么 |
|---|---|---|
| 差异化明显 | 功能/设计/品质有独到之处 | 避免价格战 |
| 毛利率 > 30% | 售价 - 全成本 > 30% | 有空间扛广告和促销 |
| 复购率或客单价高 | 要么复购要么高单价 | 利润可持续 |
| 竞争壁垒 | 专利/供应链/品牌 | 不容易被抄 |

> 核心纪律：**利润款不能只有一个**。如果整条产品线 70% 利润来自 1 个 SKU，它一旦断货/被跟卖/上差评，全盘崩。至少要有 2-3 个利润款分散风险。

### 形象款的设计原则

形象款不指望赚钱，它的价值是"撑品牌、拉信任"：

| 设计原则 | 标准 | 为什么 |
|---|---|---|
| 高品质、高溢价 | 价格是利润款的 2-3 倍 | 建立品牌高端认知 |
| 销量低但精 | 月销几十单即可 | 不追求规模 |
| 展示技术/设计实力 | 有亮点功能或工艺 | 让客户觉得品牌有实力 |
| 不占用主力资源 | 广告预算 < 10% | 不跟利润款抢预算 |

```
形象款的隐藏价值：

1. 提升品牌整体溢价感
   → 客户看到你有一个$199的高端款，会觉得你$49的利润款「性价比高」
   
2. 媒体/KOL合作素材
   → 形象款适合做开箱测评，引流回旗舰店
   
3. 拦截高端客户
   → 预算充足的客户买形象款，贡献高客单价
   
4. 占位搜索结果
   → 形象款即使不出单，也在搜索页占一个位，挤压竞品
```

### 产品线规划的实操步骤

```
第1步：盘点现有SKU
  → 列出所有SKU，标注当前销量、利润、定位（如果有的话）

第2步：给每个SKU重新定位
  → 对照三款标准，重新判定每个SKU是引流/利润/形象
  → 定位不清晰的SKU → 候选砍掉（中级2.3会讲）

第3步：找缺口
  → 看三款比例是否健康
  → 缺引流款 → 规划1-2个低价刚需款
  → 缺利润款 → 规划2-3个差异化高毛利款
  → 缺形象款 → 规划1个旗舰展示款

第4步：规划新品
  → 按缺口优先级排新品开发顺序
  → 原则：先补利润款（赚钱优先），再补引流款（带量），最后补形象款

第5步：配置资源
  → 引流款：广告预算占40%，跑量
  → 利润款：广告预算占45%，精准
  → 形象款：广告预算占15%，少量
```

## 实操要点

- **每个SKU必须有定位**：没有定位的SKU就是在浪费资源，要么给它定位要么砍掉。
- **利润款不能只有一个**：至少 2-3 个分散风险，单个断货不至于全盘崩。
- **引流款要能导流**：靠 A+ 关联、旗舰店、组合套装把流量溢出到利润款。
- **形象款不占主力资源**：广告预算 < 10%，它的价值是撑品牌不是冲销量。
- **SKU总数超过15要警惕**：精力分散，优先砍掉定位不清的尾巴产品。

## 动手练习

盘点你现在所有的 SKU，做一张「产品线矩阵表」：列出每个 SKU 的当前销量、利润、毛利率，然后对照三款标准给每个 SKU 重新定位（引流/利润/形象/待砍）。统计三款的比例，找出最大的缺口（比如利润款只有1个），规划未来 3 个月的新品开发优先级。这个练习能让你立刻看清自己的产品线是"健康矩阵"还是"一堆散沙"。
