# 中级2.5 产品线健康度盘点（实操）

> **中级2：多SKU与产品线管理**
> **本课目标**：把前 4 节课串成一份"可执行、可复盘"的产品线健康度盘点清单——让你每季度用固定时间给整条产品线做体检，而不是等问题爆发了才救火。

## 这节课解决什么问题

前 4 节课你学了产品线规划、资源分配、SKU 精简、生命周期管理。但学了不等于做到。大多数运营的真实状态是：每天埋头盯广告、调出价、处理差评，从来不跳出来看"整条产品线健康不健康"。等到发现利润下滑、库存积压、精力不够用，问题已经积累了好几个月。

**产品线管理不是日复一日的救火，是定期的体检和调整。** 你需要一个固定的、每季度做一次的"产品线健康度盘点"流程：用一张表看清整条产品线的状态，找出最该优化的地方，制定下季度的行动计划。这节课就是把前 4 课固化成一份可执行的盘点清单。

## 核心知识

### 盘点的三个维度

产品线健康度从三个维度全面体检：

| 维度 | 检查什么 | 对应课节 |
|---|---|---|
| **① 结构健康度** | 三款矩阵是否合理、SKU 数量是否过多 | 2.1 |
| **② 资源健康度** | 资金/广告/精力分配是否平衡 | 2.2 |
| **③ 生命周期健康度** | 各阶段产品分布是否滚动更替 | 2.4 |

> 关键纪律：**每季度做一次完整盘点（淡季最适合）**，不要等问题爆发才做。盘点一次 3-4 小时，能避免后续几个月的救火。

### 维度一：结构健康度盘点

检查产品线的"三款矩阵"是否健康：

| 检查项 | 健康标准 | 你的产品线 | 差距 |
|---|---|---|---|
| 总 SKU 数 | 8-15 个 | ___ | |
| 引流款数量 | 1-3 个 | ___ | |
| 利润款数量 | 2-5 个（≥2） | ___ | |
| 形象款数量 | 1-3 个 | ___ | |
| 引流款销量贡献 | ~40% | ___% | |
| 利润款利润贡献 | ~70% | ___% | |
| 定位不清的 SKU | 0 个 | ___ | |

```
结构健康度判定：

✅ 健康：SKU 8-15个，利润款≥2个，无定位不清的SKU
⚠️ 亚健康：SKU 15-20个，或利润款只有1个
❌ 不健康：SKU >20个，或大量定位不清，或利润款为0

发现问题后的动作：
- SKU > 20 → 启动精简（中级2.3）
- 利润款 < 2 → 规划新品补利润款
- 定位不清的SKU → 重新定位或砍掉
- 引流款销量贡献 > 60% → 跑量不赚钱，加利润款
```

### 维度二：资源健康度盘点

检查资金、广告、精力的分配是否平衡：

| 检查项 | 健康标准 | 你的情况 | 差距 |
|---|---|---|---|
| 资金集中度 | 头部3个产品占资金 < 70% | ___% | |
| 应急储备 | 总资金 10% | ___% | |
| 广告预算分布 | 新品40%/成熟品30%/老品30% | ___ | |
| ACOS 达标率 | > 70% 产品 ACOS 达标 | ___% | |
| 精力分配 | 新品40%/成熟品30%/老品20% | ___ | |
| 广告浪费率 | < 10%（给不出单SKU的广告） | ___% | |

```
资源健康度判定：

✅ 健康：资金分散（无单一产品占>70%），广告预算分布合理，浪费率<10%
⚠️ 亚健康：资金略集中，或广告浪费率10-20%
❌ 不健康：资金过度集中，或广告浪费率>20%，或无应急储备

发现问题后的动作：
- 资金过度集中 → 分散到其他利润款
- 广告浪费率高 → 砍掉不出单SKU的广告
- 无应急储备 → 立即留出10%
- ACOS达标率低 → 回到初级1.4优化广告
```

### 维度三：生命周期健康度盘点

检查各阶段产品分布是否"滚动更替"：

| 检查项 | 健康标准 | 你的情况 | 差距 |
|---|---|---|---|
| 导入期产品数 | 1-2 个 | ___ | |
| 成长期产品数 | 2-3 个 | ___ | |
| 成熟期产品数 | 3-5 个 | ___ | |
| 衰退期产品数 | 1-2 个（≤2） | ___ | |
| 现金牛（成熟期利润款） | ≥ 2 个 | ___ | |
| 6个月内有产品将进入衰退 | 有接班产品 | ___ | |

```
生命周期健康度判定：

✅ 健康：各阶段都有产品，成熟期利润款≥2个，有成长期产品接班
⚠️ 亚健康：某个阶段断档，或现金牛只有1个
❌ 不健康：全是成熟期（6个月后断档），或全是新品（没现金牛），或衰退期产品>3个

发现问题后的动作：
- 成熟期产品多但无新品 → 立即规划新品开发
- 衰退期产品>3个 → 启动批量清退
- 现金牛只有1个 → 加速成长期产品冲成熟
- 全是新品 → 砍掉部分，保住少数重点培育
```

### 产品线健康度总评分表

把三个维度综合，给产品线打一个总分：

| 维度 | 权重 | 评分（1-5分） | 加权分 |
|---|---|---|---|
| 结构健康度 | 35% | ___ | ___ |
| 资源健康度 | 35% | ___ | ___ |
| 生命周期健康度 | 30% | ___ | ___ |
| **总分** | 100% | — | ___ |

```
总分判定标准：

4.0-5.0 分：健康，保持现状，小幅优化
3.0-3.9 分：亚健康，有1-2个维度需要改进
2.0-2.9 分：不健康，需要重大调整
< 2.0 分：危险，立即停止扩品，先修复现有产品线
```

### 季度盘点的标准流程

```
产品线健康度盘点流程（每季度1次，3-4小时）：

第1步：数据收集（30分钟）
  → 导出所有SKU的近90天数据：销量、利润、ACOS、库存、排名
  → 整理成一张总表

第2步：结构盘点（45分钟）
  → 给每个SKU重新定位（引流/利润/形象/待砍）
  → 检查三款比例和SKU总数
  → 标记定位不清或该砍的SKU

第3步：资源盘点（45分钟）
  → 检查资金、广告、精力的当前分配
  → 找出过度集中或浪费的地方
  → 重新计算下季度的资源分配方案

第4步：生命周期盘点（45分钟）
  → 判断每个SKU的当前阶段
  → 检查各阶段分布是否滚动更替
  → 识别6个月内将进入衰退的产品及接班人

第5步：制定行动计划（30分钟）
  → 总评分 + 找出最大3个问题
  → 制定下季度的3-5个重点行动
  → 排优先级和时间表

第6步：归档（15分钟）
  → 盘点结果存档（飞书文档）
  → 跟上季度对比，看改善情况
  → 设定下季度盘点日
```

## 实操要点

- **每季度固定盘点一次**：淡季最适合，3-4 小时做一次全面体检，避免救火式运营。
- **三个维度都要查**：结构、资源、生命周期，任何一个维度不健康都会拖累全局。
- **用总分量化健康度**：4 分以上健康，3 分以下要重大调整，别自我感觉良好。
- **盘点完必须出行动计划**：找出最大 3 个问题，制定具体行动和时间表。
- **跟上季度对比看趋势**：是改善还是恶化，趋势比单次分数更重要。

## 动手练习

拿出你现在的整条产品线，做一次完整的「季度健康度盘点」：①收集所有 SKU 近 90 天数据，整理成总表；②按三个维度（结构/资源/生命周期）逐项检查并评分；③算出加权总分，判定健康等级；④找出最大的 3 个问题，制定下季度的行动计划和时间表；⑤把盘点结果存档，标注下季度盘点日期。这个练习能让你第一次用"全局视角"看自己的产品线，而不是每天埋头盯单个产品。
