# 中级1.3 冷启动四阶段细化（种子/成长/冲刺/稳定）

> **中级1：新品冷启动全流程**
> **本课目标**：学完能把新品拆成「种子期 → 成长期 → 冲刺期 → 稳定期」四个阶段，每个阶段有明确的进阶标准、切换信号和广告+运营配合方案。

## 这节课解决什么问题

上一节课给了你 0-60 天的大节奏。但真实打新品时，节奏不是"按天数走"，而是"按数据走"——可能你的产品 D20 就跑通了验证期，也可能 D40 还卡在那里。

这节课解决的是：**每个阶段什么时候算"完成"、什么时候该切换、切换后做什么**。很多运营的问题不是不努力，而是"不知道自己现在在哪个阶段"——明明还在种子期却用冲刺期的打法（烧钱冲榜），或者已经进稳定期了还在用冲刺期的高预算（白白烧利润）。

把这四个阶段的标准和切换信号刻进脑子，你打新品就不会"凭感觉"。

## 核心知识

### 四阶段总览

| 阶段 | 核心任务 | 广告角色 | 运营角色 | 典型时长 |
|---|---|---|---|---|
| **种子期** | 让亚马逊识别你 | 自动广告采集数据 | Listing 完善 + Vine 启动 | 7-14 天 |
| **成长期** | 找到出单组合 | 手动精准 + 广泛 | Coupon 提转化 + 评论积累 | 14-30 天 |
| **冲刺期** | 推核心词上首页 | TOS 抢位 + 集中预算 | 大额 Coupon + 站内外联动 | 15-30 天 |
| **稳定期** | 保排名、收利润 | 降预算 + 扩长尾词 | 收 Coupon + 优化利润 | 长期 |

> 金句：**四个阶段的切换不看天数，看数据。** 达标了才能切，没达标硬切只会浪费资源。

### 种子期：让亚马逊识别你

这是最容易被忽视、但最决定成败的阶段。亚马逊的 A9 算法在前 14 天会给你贴标签：你是哪个类目、卖给谁、什么价位段。标签贴错了，后面所有努力都事倍功半。

| 配置项 | 种子期标准 |
|---|---|
| 自动广告 | 紧密匹配，预算日均 ¥80-150，**绝不撞线** |
| 手动广告 | 不开（还没数据，开了也是瞎打） |
| Coupon | 0%（不靠折扣，靠产品力采集真实数据） |
| TOS 加价 | +25%（少量测试，不抢位） |
| Vine | D3-D7 内必须启动 |

```
种子期进阶标准（全部达标才能进成长期）：
✅ 自动广告连续 7 天有稳定曝光（日均 > 5000）
✅ 自动广告点击率 CTR > 0.3%
✅ 至少跑出 5-10 个有点击的搜索词
✅ Listing 已上架且所有属性填满

未达标怎么办：
- 曝光 < 5000 → Listing 关键词没埋对，回去查 title/backend keywords
- CTR < 0.3% → 主图/价格/标题有问题，先优化再继续
- 没有出单词 → 产品跟市场不匹配，回中级1.1重新验证
```

### 成长期：找到出单组合

这个阶段核心是"把种子期采集的数据变成能出单的武器"。

| 配置项 | 成长期标准 |
|---|---|
| 自动广告 | 降到 50%，只保留紧密匹配 |
| 手动精准 | 开启，投放种子期 Top 5-10 出单词 |
| 手动广泛 | 开启，扩词用 |
| Coupon | 5-15%（提转化，不冲量） |
| TOS 加价 | +50%（开始抢位） |
| Vine | 持续跟进，预计本阶段末到第一批评论 |

| 成长期关键动作 | 频率 |
|---|---|
| 搜索词五分类（初级1.1）| 每周 1 次 |
| 否定词入池（初级1.2）| 每周 1 次 |
| 手动精准扩词 | 每 3 天 1 次 |
| Coupon 效果复盘 | 每周 1 次 |

```
成长期进阶标准（全部达标才能进冲刺期）：
✅ 转化率稳定达标（大众品 > 8%，小众品 > 12%）
✅ 日销量稳定在 3-5 单
✅ 手动精准 ACOS < 100%
✅ 已有 3-5 条评论（Vine 或自然）

未达标怎么办：
- 转化率不达标 → 停在成长期，先优化主图/价格/Listing，别硬冲
- 评论 < 3 条 → 继续等 Vine，或主动做售后回访
- ACOS 太高 → 广告词不准，回到搜索词报表优化
```

### 冲刺期：推核心词上首页

这个阶段是"集中所有资源打关键战役"，把成长期验证过的词推上首页。

| 配置项 | 冲刺期标准 |
|---|---|
| 自动广告 | 降到 20-30%（让位给手动） |
| 手动精准 | **升到 60%**，集中打 3-5 个核心词 |
| Coupon | **20-30%**（短期冲转化） |
| TOS 加价 | **+75% 到 +100%**（首页抢位） |
| 评论 | 主动跟进，目标累计 10-15 条 |

```
冲刺期的「三集中」原则：
集中词：3-5 个核心词，不撒胡椒面
集中预算：这 3-5 个词吃 70% 总预算
集中时间：选 7-14 天窗口期，一鼓作气

切换到稳定期的信号（达到任一即可）：
✅ 核心词自然排名进首页前 3 行（前 10 位）
✅ BSR 进入类目前 30 名
✅ 自然销量占比 > 50%（不再纯靠广告）

⚠️ 冲刺失败信号（立即暂停，回成长期复盘）：
- 连续 7 天排名没动且预算烧光
- ACOS 连续 3 天 > 目标 2 倍
- 转化率突然腰斩（可能被跟卖/上差评/竞品降价）
```

### 稳定期：保排名、收利润

很多人以为"冲上首页就赢了"，其实冲上去只是开始，**守排名比冲排名更难**。

| 配置项 | 稳定期标准 |
|---|---|
| 广告总预算 | 较冲刺期降 30-50% |
| 手动精准 | 保留核心词低出价维持，长尾词扩量 |
| Coupon | 收到 5% 或取消 |
| TOS 加价 | 降到 +25-50% |
| 重点 | 利润优化、库存管理、防跟卖 |

```
稳定期的核心是「自然流量替代广告流量」：

健康的新品成熟结构：
- 自然流量占比 > 60%
- 广告流量占比 < 40%
- 广告 ACOS < 25%（保利润红线）

如果稳定期自然流量占比 < 40%：
→ 这是「广告依赖症」，一旦停广告销量立刻崩
→ 要么继续优化 Listing 和关键词，要么承认这个产品本身需求弱
```

## 实操要点

- **种子期别急**：前 14 天给亚马逊喂对数据比出几单重要，乱投广告会被贴错标签。
- **切换看数据不看天数**：每个阶段都有进阶标准，达标才切，没达标硬切是浪费钱。
- **冲刺期三集中**：集中词、集中预算、集中时间，撒胡椒面打不上去。
- **稳定期盯自然流量占比**：自然流量 > 60% 才算健康，纯靠广告撑的排名是沙堡。
- **每个阶段都有失败信号**：遇到失败信号立即暂停，别用"再坚持一下"自我安慰。

## 动手练习

拿你现在正在打的一个新品（或最近打过的），对照四阶段标准判断它现在在哪个阶段：列出当前阶段的进阶标准，逐条打勾看哪些达标、哪些没达标。没达标的项，写出具体优化动作和复查时间。这个练习能让你立刻知道"下一步该干什么"，而不是继续凭感觉调广告。
