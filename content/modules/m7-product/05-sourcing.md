# 7.5 找供应商与采购

> **模块2：选品与采购**
> **本课目标**：学会在 1688/Alibaba 上找靠谱供应商、下样品单、谈 MOQ、做品控——把一个"纸面合格"的产品变成"真货到仓"。

## 这节课解决什么问题

7.1-7.4 走完，你手里应该有一个"数据过关"的产品。但纸面合格 ≠ 真能做——**实际供货、质量、成本、合规可能完全是另一回事**。

很多新手这步就开始翻车：

- 在 1688 上找了个最便宜的供应商，结果货到了才发现质量差、退货率爆表
- 没下样品单直接下大单，500 件全是次品，砸手里
- 没查专利就上架，被权利人投诉侵权，listing 移除、库存冻结
- 没算清生产周期 + 海运时间，旺季断货、淡季压仓

这节课教你**从找供应商到货到仓**的全流程：找 → 评 → 询 → 样 → 谈 → 验 → 发。每一步都做好，才能把"数据合格"变成"实物合格"。

## 核心知识

### 第 1 步：在哪找供应商

| 平台 | 定位 | 适合 |
|---|---|---|
| **1688.com** | 阿里国内批发站，工厂/贸易商扎堆 | **首选**，价格最低、品类最全，中文沟通 |
| **Alibaba.com** | 阿里国际站，对外贸出口商 | 备选，部分工厂只在国际站开店 |
| **工厂直联** | 通过展会（广交会）/ 行业群直接找厂 | 进阶，跳过中间商，价格最优 |
| **义乌购 / 义乌小商品市场** | 小商品、低价日用 | 适合 $5 以下小件 |

**新手首选 1688**——价格透明、起订量友好、沟通无障碍。

### 第 2 步：怎么评估一个供应商靠不靠谱

1688 上同款产品几十家在卖，价格从 ¥5 到 ¥30 都有。**不要只看价格**，要看下面这些信号：

| 评估维度 | 看什么 | 靠谱信号 |
|---|---|---|
| 经营年限 | "已开业 X 年" | ≥3 年（≥5 年更稳） |
| 经营模式 | 工厂 / 经销商 | 标注「生产加工」= 工厂 |
| 交易额 | "近 30 天成交额" | 有持续成交，不是僵尸店 |
| 回头率 | "回头率" | ≥30%（老客户复购 = 质量稳定） |
| 响应速度 | "响应时长" | <8 小时 |
| 出口经验 | 是否有外贸能力 / 国际站关联 | 有出口经验，懂包装/认证 |
| 评价 | 买家评价和星级 | 4.5 星以上，有真实文字评价 |

**红旗：** 经营 <1 年、零成交、零评价、价格异常低（比同行便宜 30%+）、客服不回消息——直接跳过。

### 第 3 步：工厂 vs 贸易商——选哪个？

1688 上的卖家分两类：

| 类型 | 优点 | 缺点 |
|---|---|---|
| **工厂**（自己生产） | 价格最低、能定制、能改包装 | MOQ 高、起订量死板、客服慢 |
| **贸易商**（中间商） | MOQ 灵活、品类全、服务好 | 价格贵 10-30%、不能深度定制 |

**新人建议：** 首单从贸易商拿货（MOQ 低、试错成本低），跑通了再找工厂直供降本。

### 第 4 步：必须下样品单（Sample Order）

**这是新手最容易省略、却最不能省的一步。**

下大单前**一定要先买样品**——通常 2-3 件，对比 2-3 家供应商。

**样品要测什么：**

| 测试项 | 怎么测 |
|---|---|
| 实物质量 | 和图片一致吗？材质、做工、颜色有没有色差？ |
| 功能性 | 实际使用，跑 5-10 次完整流程，看耐不耐用 |
| 包装 | 包装够不够保护？海运会不会压坏？标签合规吗？ |
| 尺寸重量 | 实测包装后的长宽高和重量——这是查 FBA 费的依据 |
| 拍照存档 | 每个细节都拍照，作为大货对比的基准 |

**省下样品费的代价：** 500 件大货到亚马逊仓，开卖后发现做工差、退货率 15%，所有利润被退货吃光 + 差评毁掉 listing——损失是样品费的几百倍。

### 第 5 步：MOQ 谈判

MOQ = Minimum Order Quantity（最低起订量）。1688 上标的 MOQ 通常有砍价空间。

**新人首单原则：**

- 首单尽量**小批量试水**（200-500 件）
- 跑通卖动再加量，不要一上来就 1000+ 件
- 库存压仓比断货更可怕——压仓 = 资金占用 + 仓储费 + 清仓亏损

**砍 MOQ 的话术：**

| 供应商说 | 你怎么回 |
|---|---|
| "MOQ 1000 件" | "首单想下 300 件试款，跑通了稳定返单，能否通融？" |
| "小单不接" | "可以加 10% 加工费，先试 300 件" |
| "价格 ¥10" | "300 件按 ¥11 走，1000 件按 ¥10 走，先试小单" |

**关键：用「加价换小单」+「承诺返单」换供应商让步。** 大多数工厂对真有诚意的小单会让步。

### 第 6 步：品控（QC）——出货前必做

**大货生产完、发货前，一定要验货。** 不验货发的货，相当于盲盒。

| 验货方式 | 怎么做 | 成本 |
|---|---|---|
| 自己去 | 飞到工厂随机抽检 | 时间和差旅成本高 |
| **第三方 QC** | 请验货公司（如测标、SGS、当地验货服务）抽检 | 约 ¥300-800/次 |
| 视频验货 | 让工厂拍视频随机抽样 | 免费但不可全信 |

**抽检标准（AQL 2.5）：** 一般抽大货的 10%（小批量至少抽 30 件），检查外观、功能、包装。

**重点查：**

- 外观瑕疵（划痕、色差、毛刺）
- 功能性（通电、运转、闭合）
- 包装抗压（模拟海运压挤）
- 标签合规（条码、原产地标、警示语）
- 数量对不对

**发现问题立刻让工厂返工**——发出去就晚了。

### 第 7 步：定制——OEM vs ODM

如果你想打自己的品牌（做亚马逊必须品牌备案），有两种定制方式：

| 类型 | 说明 | 适合 |
|---|---|---|
| **OEM**（贴牌） | 工厂现有产品，贴你的 logo / 换包装 | 新手首选，成本低、风险小 |
| **ODM**（开模） | 你出设计，工厂重新开模生产 | 进阶，开模费 ¥5000-50000，周期长 |

**新人从 OEM 起步**——把工厂现有好产品贴你的品牌、改包装设计、加说明书，做出差异化即可。等销量稳定再考虑 ODM。

### 第 8 步：知识产权检查（上市前必做）

**侵权 = listing 移除 + 库存冻结 + 账号风险**。采购前一定要查：

| 查什么 | 在哪查 |
|---|---|
| 美国商标 | USPTO（uspto.gov，TESS 系统） |
| 欧盟商标 | EUIPO（euipo.europa.eu） |
| 专利（外观/发明） | USPTO / EUIPO 专利数据库，或 Google Patents |
| 版权 | 商品图案、卡通形象（迪士尼、漫威等不能碰） |

**红旗：** 产品上有别人的 logo（Nike 勾、Apple 苹果）、卡通形象、电影 IP——一律不碰。

**经验：** 选品时就避开容易侵权的类目——电子（专利密集）、玩具（IP 密集）、运动品牌周边。

### 第 9 步：算清生产 + 物流周期

新手最容易踩的"时间坑"：以为下单到上架 2 周，结果 2 个月。

| 阶段 | 时长 |
|---|---|
| 下单到生产完成 | 15-30 天（看复杂度和 MOQ） |
| 国内验货 + 贴标 + 发货准备 | 3-7 天 |
| 海运到美/欧亚马逊仓 | 30-45 天 |
| 空运（应急） | 7-14 天 |
| FBA 入库上架 | 3-10 天 |

**总计：海运 50-90 天，空运 25-40 天。** 这意味着你今天下单，2-3 个月后才能上架卖——选品要提前 1 个季度做，旺季备货更要倒推时间。

## 实操要点

- **供应商评估不只看价格**：经营年限、回头率、响应速度、出口经验同样重要，便宜 20% 但质量差 = 全盘亏。
- **样品单必须下**：2-3 家对比，每家 2-3 件，是省不得的"保险费"。
- **首单小批量**（200-500 件）：跑通再加量，库存压仓比断货更可怕。
- **MOQ 可谈**：用"加价换小单"+"承诺返单"换让步。
- **出货前必验货**：第三方 QC ¥300-800，挡掉次品大货。
- **OEM 起步，ODM 进阶**：新手别一上来开模。
- **侵权必查**：USPTO / EUIPO / Google Patents，IP 类目（玩具、电子、运动）避开。
- **算清 50-90 天周期**：选品和备货要提前 1 个季度。

## 动手练习

接着 7.4 通过的产品，做完整的采购准备：

1. **找供应商**：在 1688 上搜这个产品的关键词，列出 **3 家**候选供应商，按本课的评估表打分（年限、回头率、响应、价格）。
2. **询价 + 索样**：给这 3 家发询盘，要求报 MOQ、价格、定制费，并下样品单（每家 1-2 件）。
3. **样品评估**：收到样品后，按"质量/功能/包装/尺寸重量/拍照"5 项打分对比，选出最优一家。
4. **MOQ 谈判**：和选定供应商谈首单 300 件的价格和 MOQ，把对话记录下来。
5. **IP 检查**：在 USPTO / EUIPO / Google Patents 搜产品关键词，确认有没有冲突的商标或专利。
6. **周期规划**：算出"下单到上架"的总时间，倒推出你最早能开卖的时间。

把这 6 步的结果整理成一页采购计划——这是 7.6 模块毕业练习的一部分。
