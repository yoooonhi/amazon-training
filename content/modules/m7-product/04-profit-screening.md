# 2.4 利润预筛

> **入门2：选品与采购**
> **本课目标**：在投入时间和金钱之前，用简单公式和官方工具快速判断一个产品"扣完成本能赚钱吗"——这是选品 4 条标准的第 3 条。

## 这节课解决什么问题

7.2、7.3 看完需求和竞争，你可能已经挑出一个"有人买、竞争也能打"的产品。但还有一个最关键的判断：**它能不能赚钱？**

很多新手跳过这一步直接下单采购，结果货到亚马逊卖了三个月，账户报表一拉——**每卖一单亏 $0.5**。卖得越多亏得越多。原因都是选品时没做利润预筛。

利润预筛不需要你把整个利润链算到小数点（那是入门 5 的事），但需要你**在下采购单之前，用 5 分钟粗算一下**：扣完成本、FBA、佣金、运费、广告，还能剩多少？剩得过 25% 吗？

这节课给你一套**选品阶段的快速利润筛法**——简单、粗暴、但能挡掉 80% 的坑货。

## 核心知识

### 利润的 6 个扣项（粗略版）

选品阶段，你不需要像入门 5 那样精算。但要心里有数，一个产品的售价要扣掉这 6 项才是真正的利润：

```
售价（买家付的钱）
  − 亚马逊销售佣金（通常 15%）
  − FBA 配送费（按重量/尺寸分段）
  − 产品成本（进货价）
  − 头程运费（从工厂到亚马逊仓）
  − 退货成本（按退货率分摊）
  − 广告花费（按销量分摊）
  ════════════════════════════
  = 净利润
```

每项都要估一个数。任何一项漏算，都会让你高估利润，然后实际亏钱。

### 快速判断 1：售价 ≥ 成本的 3 倍

这是最简单粗暴的"能不能做"的判断公式：

> **目标售价 ≥ 产品成本 × 3**

为什么是 3 倍？因为：

- 佣金 15% + FBA 配送费 + 头程运费 ≈ 售价的 35-45%
- 还要预留广告 15%（按 ACOS 15% 估）
- 减去这些，剩下的就是毛利
- 3 倍是经验下限，再低就很难有净利

**举例：**

| 产品成本 | 售价下限（3 倍） | 是否可行 |
|---|---|---|
| $3 | $9 | 售价太低，FBA 吃光，**不可行** |
| $5 | $15 | 刚到下限，紧巴巴 |
| $8 | $24 | 合理 |
| $15 | $45 | 健康 |
| $25 | $75 | 高客单，利润空间大 |

**如果市场价（参考 7.2 价格带分析）低于你成本 × 3 的售价下限——直接放弃**，不管这个产品多好卖。

### 快速判断 2：最低售价 $15-20

售价太低，FBA 费和佣金会吃光所有利润。原因：

- FBA 有**最低配送费**（无论多小的件，都要收基础费）
- 佣金 15% 是**比例费**，低价品佣金绝对值小，但 FBA 配送费的绝对值不变
- 比如一个 $8 的产品，FBA 配送费 $3.5 + 佣金 $1.2 = $4.7，光这两项就吃掉 58%——剩下 $3.3 要覆盖成本、运费、广告，根本不够

**经验值：新手选品，售价尽量 ≥ $15（美国站）或 ≥ $15（欧洲站）。** $8-12 的产品留给有供应链优势的大卖家。

### 快速判断 3：目标净利率 ≥ 25-30%

把所有成本扣完后，**净利润率要 ≥ 25%**（按售价算）。

为什么是 25-30%？

- 广告 ACOS 通常 15%，意味着每单要烧掉售价 15% 的广告费
- 还要预留促销、退货、汇率波动的空间
- 低于 25%，一个小波动（退货率上升、广告费上涨、被迫降价）就转亏

**粗算公式：**

```
净利率 ≈ (售价 − 佣金15% − FBA − 成本 − 头程 − 退货5%) ÷ 售价 − 广告15%
```

如果算下来 <25%，**这个产品不能做**。

### 怎么查 FBA 配送费：用官方计算器

FBA 配送费是利润链里**最不准估**的一项——它严格按产品包装后的**尺寸和重量**分段计费。你不能凭感觉估，要用亚马逊官方工具。

**工具：Amazon FBA Revenue Calculator（FBA 收入计算器）**

- 美国站：sellercentral.amazon.com/fba/revenuecalculator
- 欧洲站各站点也有对应入口（在卖家后台搜 "Revenue Calculator"）

**怎么用：**

1. 输入产品的**长 × 宽 × 高**（单位 inch 或 cm）和**重量**（含包装！）
2. 输入预估售价
3. 工具会自动算出 FBA 配送费、佣金、仓储费
4. 直接显示净利润和利润率

**关键：尺寸和重量必须用「包装后」的数**——产品本身 $3，加上包装盒后体积重量翻倍，FBA 费就跳档了。这是新手最容易忽略的坑。

### 头程运费估算

头程运费 = 从中国工厂运到目标国家亚马逊仓库的运费。粗略经验值（按海运）：

| 路线 | 单位成本（人民币） | 说明 |
|---|---|---|
| 中国 → 美西/美东 | ¥15-25 / 公斤 | 海运，30-45 天 |
| 中国 → 欧洲（DE/FR/UK） | ¥18-30 / 公斤 | 海运，35-50 天 |
| 空运（任何路线） | ¥60-100+ / 公斤 | 7-14 天，贵但快 |

**估算方法：** 用产品包装后的重量 × 单位成本 ÷ 每箱数量 = 每件头程分摊。

**例：** 一个 0.5 公斤的产品，海运到美国 = 0.5 × ¥20 = ¥10 ≈ $1.4 / 件。

### 利润预筛 5 步法（实操流程）

把上面所有点串起来，每个候选产品走一遍这 5 步：

| 步骤 | 动作 | 工具 |
|---|---|---|
| 1. 查市场价 | 看 7.2 的价格带，定一个合理售价 | Amazon 第 1 页 |
| 2. 估成本 | 在 1688/Alibaba 查进货价（7.5 详讲） | 1688.com |
| 3. 验证 3 倍规则 | 售价 ≥ 成本 × 3？不过就放弃 | 心算 |
| 4. 查 FBA 费 | 用 Revenue Calculator 输入尺寸重量算 | 官方计算器 |
| 5. 算净利率 | 套公式，≥25% 才过 | 表格 |

### 红旗产品：直接淘汰的特征

有些产品**结构上就不可能赚钱**，看到这些红旗直接放弃，不要浪费时间精算：

| 红旗 | 原因 |
|---|---|
| **重 / 体积大**（>2kg 或体积重很大） | FBA 配送费跳档，吃光利润 |
| **易碎品**（玻璃、陶瓷） | 退货率高，破损成本高 |
| **便宜但巨大**（如充气游泳池） | 海运头程费爆炸 |
| **电子带电池** | 需要 UN38.3 等认证，FBA 入库审核严 |
| **食品/护肤品** | 认证、保质期、退货复杂 |
| **客单价 < $10** | FBA + 佣金吃光，无利可图 |

**这些产品不是"不能做"，是"新手不要做"**——有供应链和合规经验的老手能做，新手碰了一定亏。

## 实操要点

- **下采购单前必须算利润**：跳过这步直接下单，3 个月后才知道亏了多少。
- **3 倍规则先卡**：售价 < 成本 × 3 直接淘汰，不要纠结细节。
- **FBA 费用必须用官方计算器查**：不能估，尺寸和重量"包装后"是关键。
- **净利率目标 25-30%**：低于 25% 不做，留出广告和波动空间。
- **避开红旗产品**：重、大、易碎、便宜巨大、带电池、食品——这些新手别碰。
- **粗算即可，不必精算**：选品阶段是"过/不过"的筛选，入门 5 的精算是上架后再做。
