# 7.4 跟卖防御与 Buy Box

> **模块7：日常运营与判断力**
> **本课目标**：搞懂跟卖是怎么回事、怎么发现、怎么防御，以及为什么 Buy Box 决定你的销量。

## 这节课解决什么问题

某天你打开后台，发现销量突然腰斩——Listing 没改、广告没动、库存还有，但订单就是掉了。跑去前台一看，你的产品页面右侧那个"Add to Cart"按钮，推荐的是**别人的报价**，不是你的。这就是**跟卖**抢走了你的 **Buy Box**。

跟卖是亚马逊平台特有的机制，新手最容易在两件事上吃亏：一是**不知道自己被跟卖了**（被人分走流量还以为销量自然下滑）；二是**用错方法去打跟卖**（自己下单给差评、私下威胁，结果违反平台政策，把自己账号搞坏）。

这节课讲清楚：跟卖是什么、有哪几种、怎么发现、用亚马逊的**官方工具**正确防御，以及 Buy Box 到底怎么决定谁能卖出去。

## 核心知识

### 什么是跟卖（Hijacking）

亚马逊允许**多个卖家在同一个 ASIN 上挂自己的报价**。也就是说，别人可以在**你创建的 Listing 上**卖东西，前提是他们声称卖的是"同一个产品"。这就是跟卖。

跟卖的本质：别人**蹭你 Listing 的流量和评论**。你花钱打广告、花时间优化 Listing、积累好评，跟卖者什么都不做，直接把报价挂上来，价格比你低一点，就可能抢走订单。

> 为什么亚马逊允许？因为亚马逊的核心理念是"给买家最低价和最多选择"。同一个产品页面有多个卖家竞争，买家能拿到更便宜的价格——这是平台设计，不是 bug。但对卖家来说，这就是防线的起点。

### 跟卖者的三种类型

不是所有跟卖都一样。判断对方是哪一类，决定你怎么应对：

| 类型 | 是什么 | 怎么应对 |
|---|---|---|
| **合法竞品卖家** | 卖的是和你**真正一样**的正品（比如同一个品牌的分销商） | 合法但烦人。只能靠价格、FBA、卖家绩效竞争，或做好产品差异化 |
| **假冒卖家（Counterfeit）** | 卖的是**假货**，挂在你 Listing 上 | **违法，必须打**。用 Brand Registry + Report a Violation 投诉侵权 |
| **未授权分销商（灰市）** | 没经你授权卖你的品牌产品（可能从其他渠道拿货） | 如果你有品牌注册和渠道管控，可以按侵权或未授权销售投诉 |

**关键区别**：第一类你只能靠竞争；第二、三类是**侵权**，可以用亚马逊的工具清理。

### 怎么发现被跟卖

跟卖不会主动通知你，必须自己查。每天检查：

- **前台看 Listing**：打开你的产品页，看右边是否有"Other Sellers on Amazon"（其他卖家）框。如果有别人的报价，就是被跟卖了。
- **看 Buy Box 占比**：如果你原来独占 Buy Box，突然占比掉到 80%、50% 甚至 0%，很可能来了跟卖者抢走了 Buy Box。
- **看销量突变**：没有改价、改广告、改 Listing 的情况下销量突然下滑，第一反应应该是"是不是被跟卖了"，去前台确认。
- **用工具设警报**：Brand Analytics、第三方工具（很多 ERP 或跟卖监控工具）可以在有新卖家挂上来时主动提醒你，不用每天手动盯。

### 怎么防御跟卖

**核心原则：用亚马逊官方工具，不要用野路子。**

| 防御手段 | 是什么 | 适用场景 |
|---|---|---|
| **Brand Registry（品牌注册）** | 注册你的商标到亚马逊品牌库，获得品牌保护工具的入口 | 所有自有品牌卖家**必做**。这是后续所有维权工具的前提 |
| **Transparency 透明计划** | 给每个产品贴一个唯一的防伪溯源码，亚马逊入库时扫码验证真假 | 有品牌注册、且产品容易造假（美妆、保健品、3C 配件等） |
| **Report a Violation（举报违规）** | 在品牌后台提交商标/版权侵权投诉，要求下架假冒或未授权 listing | 对方卖假货、盗用你的图片/商标文字 |
| **Project Zero** | 已加入的品牌可以**自助一键下架**疑似假冒 listing，不必等亚马逊审核 | 已通过品牌注册并加入 Project Zero 的品牌 |
| **产品本身难复制** | 定制包装、独特设计、注册专利/版权 | 从根本上降低被跟卖的价值——别人抄不动 |

**防御的顺序**：先做 Brand Registry（地基）→ 对假冒卖家用 Report a Violation / Project Zero（清理）→ 对核心产品上 Transparency（防伪）→ 产品设计上做差异化（治本）。

### 什么不要做（会害了自己）

新手一急容易犯两个错，**都会触发亚马逊对你账号的处罚**：

- **不要自己买对方产品然后留差评**：这违反亚马逊的评论政策（禁止对竞争对手恶意评论），查到轻则评论被删，重则账号受限。
- **不要私下威胁跟卖者**（发消息警告、要求对方下架）：亚马逊禁止卖家之间私下施压。所有纠纷必须走官方投诉渠道。

记住：**跟卖是平台允许的机制，你只能用平台提供的工具去应对。用野路子，输的是你自己。**

### Buy Box（黄金购物车）是什么

Buy Box（也叫 Featured Offer）是产品页面**右侧那个白色框**，里面有价格、"Add to Cart"（加入购物车）和"Buy Now"（立即购买）按钮。

绝大多数买家不会去对比下方"其他卖家"的报价，直接点这个按钮就下单了。**所以谁占据 Buy Box，谁就拿走绝大部分销量。** 失去 Buy Box，即使你的 Listing 再好、广告再猛，转化率也会断崖式下跌——因为买家根本看不到你的报价。

### 谁能赢得 Buy Box

亚马逊用一个算法决定哪个卖家的报价占据 Buy Box。影响因素：

| 因素 | 说明 |
|---|---|
| **价格** | 通常是**总到岸价最低**的报价（=商品价+运费）占优。但不是绝对——价格最低不一定必胜 |
| **配送方式** | FBA 通常比 FBM 更有优势（亚马逊物流更快更稳，买家体验好） |
| **卖家绩效** | 订单缺陷率（ODR）、迟发率、取消率、响应买家消息的速度。绩效差的卖家即使价格低也拿不到 Buy Box |
| **配送速度** | 到货越快越有利（这也是 FBA 占优的原因之一） |

**对你来说的含义**：

- 如果你有品牌注册、产品是独家的（别人没同款），那**跟卖者是你 Buy Box 的主要威胁**。
- 如果产品有多家卖同款，你要靠**价格 + FBA + 好绩效**守住 Buy Box。
- **每天监控 Buy Box 占比**：占比掉了，立刻查是不是来了跟卖、或你的绩效/价格出了问题。

## 实操要点

- **每天扫一眼核心 Listing 有没有新卖家挂上来**——跟卖不会通知你，必须主动查。
- **销量无理由突降，第一反应查 Buy Box 占比和跟卖**，不要先怀疑广告或季节。
- **品牌注册（Brand Registry）是所有防御的起点**，没注册就没有官方维权工具。
- **打跟卖只用亚马逊官方工具**（Report a Violation、Project Zero、Transparency），不要自己买货给差评、不要私下威胁。
- **Buy Box 占比每天监控**，掉了立刻定位原因（价格？绩效？跟卖？）。
- **从根上降低被跟卖价值**：差异化产品、定制包装、注册 IP——别人抄不动你就不会来。

## 动手练习

挑一个你自己（或公司）在卖的产品，做一次"跟卖与 Buy Box 自检"：

1. 打开前台产品页，看右侧是否只有你的报价，还是有"Other Sellers"。如果有别人，记录对方卖家名、价格、是否 FBA。
2. 查这个产品有没有做 Brand Registry。如果没做，列出注册需要的材料（商标注册号等）。
3. 假设明天突然来了一个卖假货的跟卖者，写出你的处理步骤（用哪几个官方工具、顺序是什么）。
4. 查一下你的 Buy Box 占比（后台或工具），如果低于 90%，想想可能的原因是什么。

做完这份自检，你就有一份针对自己产品的"跟卖防御清单"了。
