# 7.6 决策记录与复盘

> **入门7：日常运营与判断力**
> **本课目标**：学会把每个决策记下来，并在 D+7 / D+14 / D+30 回头检验预期对不对——这是运营能不能持续进步的关键。

## 这节课解决什么问题

新手做了一堆决策（调价、改广告、补货、清仓），但从来不记。三个月后老板问"当时为什么这么调"，答不上来；遇到类似情况，又凭感觉重来一遍，犯过的错反复犯。

**没有记录，就没有复盘；没有复盘，就没有进步。** 运营能力不是靠"做得多"涨上去的，是靠"做 → 记 → 复盘 → 改"这个循环涨上去的。

这节课教你：每个决策要记什么、什么时候复盘、复盘的五种结果怎么归类、以及怎么区分"决策对但执行弱"还是"决策错但执行干净"——这个区分决定了你下一轮该改什么。

## 核心知识

### 每个决策都要记这 5 个字段

不是写流水账，是结构化记录。每个决策记：

| 字段 | 内容 | 举例 |
|---|---|---|
| 日期 | 什么时候做的 | 2026-07-12 |
| 决策 | 做了什么 | A 产品主广告出价从 €0.8 降到 €0.6 |
| 理由 | 为什么这么做 | 当前 ACOS 25% > BE ACOS 18%，要压成本 |
| 预期结果 | 具体到数字 | ACOS 降到 18-20%，日均单量不掉超过 20% |
| 实际结果 | （复盘时填） | D+7：ACOS 22%，单量掉 35% |
| 复盘结论 | （复盘时填） | 出价降太狠，流量掉太多，回滚一部分 |

**预期结果必须是具体数字**，比如"ACOS ≤20%"、"日均 ≥15 单"、"利润率 ≥12%"。不能写"销量会好一点"、"广告会改善"这种模糊的话——模糊的预期没法复盘，因为你永远不知道"算不算达到了"。

### 三个复盘窗口

不同决策需要不同时间长度才能下结论：

| 窗口 | 看什么 | 举例 |
|---|---|---|
| **D+7** | 短期方向对不对，要不要回滚 | 调完出价 7 天，ACOS 是降了还是没降？单量崩没崩？方向错就回滚。 |
| **D+14** | 中期趋势确认 | 排名/销量趋势有没有稳住？还是只是短期波动？ |
| **D+30** | 长期结构性结论 | 这个价/这个广告结构是不是真的能持续赚钱？月度利润改善了没有？ |

**为什么不能只看 7 天就下永久结论？** 因为 7 天数据有太多噪声（周末效应、对手动作、偶然订单）。结构性结论要 30 天以上才靠得住（6.5 会专门讲这个坑）。

### 复盘的五种结果

每个决策复盘后，归到五类之一：

1. **有效（valid）**：达到预期。ACOS 确实降到了目标，单量也没掉。→ 继续保持。
2. **无效（invalid）**：完全没达到。ACOS 没降，单量还掉了。 → 找原因，可能要回滚。
3. **混合（mixed）**：部分指标达标，部分没达标。ACOS 降了但单量掉太多。 → 权衡，调参再试。
4. **无数据（no-data）**：没法判断。数据没拉、被其他变动干扰了。 → 重新设计观察。
5. **待跟进（needs follow-up）**：时间还不够，要再等。7 天不够，等 14 天再看。 → 标注下次复盘时间。

### 最关键的区分：决策对不对 vs 执行好不好

复盘时一定要分清这两件事：

- **决策对、执行弱**：方向没错（降价确实能提量），但执行有问题（降得太猛、节奏不对、配套没跟上）→ 改执行，保留决策方向。
- **决策错、执行干净**：方向本身就错（不该清仓的产品清了），但执行很到位（清得很快、记录很全）→ 改决策逻辑，执行方式可以复用。

这两类原因的改法完全不同。**混为一谈，你下一轮该改什么就抓不准**——明明是执行弱，却去推翻决策方向，等于把对的也扔了。

### 待跟进项不能靠"感觉"关闭

"待跟进（needs follow-up）"这一类有个严格纪律：**它不能靠主观感觉关闭，必须靠数据关闭。**

```
错的关闭方式：「感觉差不多好了吧」「应该没问题了」「老板没再问，就当解决了」
  → 主观判断，下次同样的坑还会再踩

对的关闭方式：拿出 D+14 / D+30 的数据，对照当初写下的预期数字
  → 达标 → 标记"有效"，正式关闭
  → 没达标 → 继续跟进或改方案，不能关
```

一个待跟进项要么变成"有效/无效/混合/无数据"四类之一（要数据支撑），要么继续挂着等下一次复盘。**没有"我觉得它好了，所以关掉"这个选项。** 靠感觉关闭待跟进项，等于把没验证完的决策当成了已验证，下一个坑已经在路上了。

### 预期必须是数字，不能是模糊的话

再强调一遍：预期结果写"销量会改善"= 没法复盘。因为：

- "改善"是改善多少？+5% 算不算？+50% 算不算？
- 三个月后销量涨了 10%，到底是这个决策的功劳，还是季节性、对手退出、平台流量？

只有预期是**具体数字 + 时间**（"7 天内 ACOS 从 25% 降到 20% 以下，单量不低于日均 15 单"），复盘时才能客观判断"达成 vs 未达成"，不被主观感觉带偏。

## 实操要点

- **每个决策都记 5 字段**：日期、决策、理由、预期（具体数字）、实际（复盘填）。不记 = 没法复盘。
- **预期必须是数字**：ACOS、单量、利润率，带明确阈值和时间。拒绝"会好一点"这种话。
- **D+7 看方向，D+14 确认趋势，D+30 下结构结论**。不要用 7 天数据下永久结论。
- **复盘归五类**：有效、无效、混合、无数据、待跟进。归类决定下一步动作。
- **分清"决策对错"和"执行好坏"**：执行弱就改执行，决策错就改逻辑，别混为一谈。

## 动手练习

翻出你过去一周做的某个决策（调过价、改过广告、补过货都行）。如果当时没记录，凭回忆补一份决策日志，写满 5 个字段：

1. 日期：？
2. 决策：做了什么？
3. 理由：为什么做？
4. 预期结果：**用具体数字写**（如果当时只想着"会变好"，现在补成数字）
5. 实际结果：现在过去几天了，数据是多少？

然后判断：这个决策复盘归哪一类（有效/无效/混合/无数据/待跟进）？如果没达到预期，是"决策方向错"还是"执行有问题"？下一轮你会改什么？

如果发现"我连当时为什么这么做都记不清了"——这就是为什么必须当时就记。
