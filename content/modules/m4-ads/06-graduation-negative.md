# 5.6 关键词毕业与否定

> **模块5：广告体系**
> **本课目标**：学会"毕业"和"否定"两个核心动作，让广告越跑越精准、越跑越省。

## 这节课解决什么问题

广告跑一段时间后，Search Term Report 里会出现三类搜索词：能出单的好词、不出单的烂词、还在观望的中间词。怎么把好词"提升"到精确层？怎么把烂词"砍掉"省钱？这节课讲的就是这两个关键动作。

## 核心知识

### 动作一：毕业（Graduation）—— 把好词提升到 Exact

**毕业的定义**：当一个搜索词在 Auto / Broad 广告里证明自己（≥2 单 + ACOS 可接受），把它单独拿出来放到 Exact 广告里精准投放，这个过程叫"毕业"。

```
毕业前：搜索词在 Auto / Broad 里（流量分散、出价不精确）
         │
         │ 满足毕业条件：≥2 单 + ACOS 可接受
         ▼
毕业后：放到 Exact 广告里（精准锁定、可独立调出价和 TOS）
```

为什么要毕业：

- Exact 流量更精准，转化率更高
- 可以单独给这个词调出价、加 TOS 加价
- 不再受 Auto / Broad 整体预算拖累

### 毕业时最容易被忽略的一步：在源广告里加否定

**这是新手最常踩的坑**。毕业一个词时，必须**同时做两件事**：

```
1. 把这个词加到 Exact 广告 ✅
2. 把这个词在源 Auto / Broad 广告里加 Negative Exact ❌ ← 必须做！
```

为什么第 2 步必须做？

> 如果你不否定，这个词在 Exact 和 Auto / Broad 里**会同时跑** → 你在跟自己竞价同一批流量 → **双重烧钱**，CPC 还被自己抬高了。

举例：`thick yoga mat` 从 Auto 毕业到 Exact：

| 状态 | 后果 |
|---|---|
| 只加 Exact，没否定 Auto | 这个词在两个广告里都跑 → 同一个买家搜这个词，你的两条广告互相竞价 → CPC 被自己抬高，预算双倍花 |
| 加 Exact + 在 Auto 里 Negative Exact | 这个词只在 Exact 跑，流量干净，出价可控 ✅ |

### 动作二：否定（Negation）—— 把烂词砍掉省钱

否定的两种类型（5.2 课讲过，这里讲怎么用）：

| 否定类型 | 用法 | 举例 |
|---|---|---|
| **Negative Exact（否定精确）** | 只屏蔽一个具体搜索词 | 屏蔽 `cheap yoga mat`，其他带 cheap 的不影响 |
| **Negative Phrase（否定词组）** | 屏蔽所有包含这个词组的搜索词 | 屏蔽 `cheap`，所有 cheap xxx 都被屏蔽 |

### 什么时候该否定一个词

判断标准（**两个条件都满足**）：

1. **≥10 次点击 0 出单**（点了 10 次都不买，基本不是你的目标客户）
2. **词义明显不相关**（不是偶然，是这个词本身的方向就错了）

举例该否定的：

| 搜索词 | 点击 | 订单 | 词义相关性 | 判断 |
|---|---|---|---|---|
| `office chair` | 12 | 0 | 你卖户外椅，完全不对路 | ✅ 否定 |
| `kids chair` | 15 | 0 | 你卖成人椅 | ✅ 否定（Negative Phrase 加 kids） |
| `camping chair` | 8 | 0 | 完全相关 | ❌ 别否定，数据还不够（继续观察） |
| `camping chair red` | 11 | 0 | 相关，但你没红色款 | ⚠️ 可考虑加否定，或先补红色款 |

### 不要否定太激进

新手容易犯的错：看到某个词"今天点了 5 次没出单"就立刻否定。

问题：**5 次点击的样本太小**，可能只是偶然。

建议节奏：

```
连续观察至少 3 天 →
  累计 ≥10 次点击且 0 单 + 词义不相关 → 才否定
```

> 给词一个"证明自己"的机会。新品尤其要谨慎，太早否定会杀掉学习流量。

### 估算否定一个词能省多少钱

简单算账：

```
假设词 X：过去 7 天 20 次点击，0 单，CPC 0.6 美金
花费 = 20 × 0.6 = 12 美金（这 12 块完全白烧）

如果未来 30 天还是这个比例：
  预计浪费 = 12 × 4 ≈ 48 美金

否定这个词 → 直接省下这 48 美金（理论值）
```

把这种"白烧词"挨个否定，ACOS 会肉眼可见地降下来。

## 实操要点

- **毕业 = 加 Exact + 在源广告加 Negative Exact**，两步必须同时做。
- **否定要 ≥10 点击 0 单 + 词义不相关**，两条都满足再动手。
- **新品不要太激进否定**：连续观察至少 3 天，给学习流量一个机会。
- **优先用 Negative Phrase** 屏蔽一类烂词（如所有带 cheap / free / wholesale 的），效率高。
- **毕业和否定要定期做**：建议每周看一次 Search Term Report，处理一次。
