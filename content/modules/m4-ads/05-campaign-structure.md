# 5.5 广告结构设计

> **模块5：广告体系**
> **本课目标**：学会用"漏斗式"广告结构组织你的广告活动，知道每种类型该放什么、什么时候升级到 Exact。

## 这节课解决什么问题

很多新手开广告就是把所有关键词一股脑塞进一个广告活动，结果预算花在哪都搞不清，更没法分别优化。这节课讲清楚怎么用**漏斗式结构**把广告分层，让每一层都各司其职、数据清晰。

## 核心知识

### 漏斗式广告结构：Auto → Broad/Phrase → Exact

广告本质上是一个"找词 → 验词 → 收割"的过程：

```
                ┌─────────────────┐
                │   Auto（自动）   │  ← 发现层：让亚马逊告诉你买家搜什么
                │   "撒网找词"     │
                └────────┬────────┘
                         │
                         ▼ 跑出来的好词
                ┌─────────────────┐
                │ Broad / Phrase  │  ← 验证层：测试具体场景词
                │   "小范围试"     │
                └────────┬────────┘
                         │
                         ▼ 验证有效的词
                ┌─────────────────┐
                │   Exact（精确）  │  ← 收割层：精准捕捉已验证的高转化词
                │   "精准收割"     │
                └─────────────────┘
```

每一层的作用：

| 层 | 类型 | 作用 | 优化重点 |
|---|---|---|---|
| 发现层 | **Auto** | 系统帮你找匹配你产品的搜索词 | 看 Search Term Report 找好词、否定烂词 |
| 验证层 | **Broad / Phrase** | 测试某个具体场景词（如"thick yoga mat"）效果如何 | 看这个词的转化和 ACOS |
| 收割层 | **Exact** | 把已验证能转化的词，用最精准的方式锁定 | 调出价、加 TOS 加价、稳定排名 |

### 为什么要按匹配类型分开建广告活动

新手常见错误：把 Broad、Phrase、Exact 全塞在一个广告活动里。问题：

- **预算混在一起**：Broad 把预算烧光，Exact 一毛钱花不出去。
- **没法分别优化**：调 Exact 出价时不小心影响 Broad。
- **数据看不清**：哪一层效果好完全没法判断。

正确做法：**至少建 3 个独立的 Campaign**（Auto / Broad / Exact 各一个），各自独立预算、独立优化。

### 命名规范（很重要）

广告活动一多就乱。养成命名习惯，比如：

```
[日期]_[SKU简码]_[类型]_[匹配类型]_[目的]
例：20260712_YM01_SP_Broad_测词
    20260712_YM01_SP_Exact_收割
    20260712_YM01_SP_Auto_发现
```

好的命名让你一眼看出：这个广告是给哪个 SKU、什么类型、什么阶段、什么目的。

### Auto 广告的四种匹配类型

Auto 广告其实在内部有 4 种匹配（亚马逊帮你分好类）：

| Auto 匹配类型 | 系统帮你匹配什么样的流量 |
|---|---|
| **Close Match（紧密匹配）** | 和你产品高度相关的搜索词（最有价值） |
| **Loose Match（宽泛匹配）** | 相关但不够精准的搜索词 |
| **Substitutes（同类商品）** | 和你产品同类的其他产品的详情页 |
| **Complements（互补商品）** | 经常一起买的其他产品详情页（如卖帐篷的，出现在睡垫详情页） |

> 优化 Auto 时，可以分别看这 4 类的表现，把表现差的（比如 Loose Match 高 ACOS）单独降出价或加否定。

### 什么时候把词"升级"到 Exact

升级条件（**两个都要满足**）：

1. 这个搜索词在 Auto / Broad 里跑了 **≥2 单**（不是偶然）
2. ACOS 可接受（低于或接近 BE ACOS）

> 关键：**不是看到一个词出 1 单就升级**。1 单可能是运气，2 单以上才说明这个词真有转化潜力。

### 反面教材：新品一上来就开 Exact

新手最容易犯的错：听说"Exact 精准、转化高"，新品一上就开一堆 Exact。

问题：你**根本不知道哪些词好**，凭感觉选的词可能：

- 根本没流量（你的产品在这个词下没排名）
- 转化极差（你以为买家这么搜，其实不是）
- 出价太高（被大卖挤死）

结果：广告烧不动或烧得很贵却没单。**正确顺序永远是 Auto / Broad 先跑数据，Exact 只装验证过的词**。

## 实操要点

- **每个 SKU 至少 3 个广告活动**：Auto、Broad、Exact（或 Auto + Phrase + Exact）。
- **Exact 只放验证过的词**：标准是 ≥2 单 + ACOS 可接受。
- **命名要规范**：否则 3 个月后你自己都看不懂。
- **Auto 的 4 种匹配类型分别看数据**：发现 Close Match 好就加预算；Loose Match 烂就降出价或加否定。
- **结构会随阶段变化**：新品 Auto / Broad 占比大；成熟期 Exact 占比应该最大（收割层应该是主力）。
