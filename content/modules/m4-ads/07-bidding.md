# 5.7 出价调整逻辑

> **模块5：广告体系**
> **本课目标**：搞懂出价策略、展示位置加价，学会用 Targeting 级数据按词调出价，而不是看整体平均瞎调。

## 这节课解决什么问题

"出价"是广告里最常被乱调的设置。新手要么不敢动、要么拍脑袋乱调。这节课讲清楚：什么时候该加价、什么时候该降价、按什么数据调、调多少。

## 核心知识

### 三种出价策略（Bidding Strategy）

广告活动层级可以选三种策略：

| 策略 | 系统怎么处理你的出价 | 适合什么时候 |
|---|---|---|
| **Dynamic Down（动态降出价 - 仅降低）** | 系统判断转化可能性低时，**降低**你的实际出价 | 大多数情况下的稳妥选择，避免在差流量上花全价 |
| **Dynamic Up/Down（动态升降出价）** | 系统判断高转化时**加价最多 100%**，低转化时**降价最多 100%** | 想冲排名、愿意为高转化流量付双倍 |
| **Fixed（固定出价）** | 出价不动，系统不调整 | 想要完全可控、精准测算 |

> 简单选择：
> - 新手 / 求稳 → **Dynamic Down（仅降低）**
> - 冲排名期 → **Dynamic Up/Down（升降）**
> - 精细测算 → **Fixed**

### 展示位置加价（Placement Multiplier）

即使出价定了，还可以对**不同展示位置**单独加价（百分比）：

| 位置 | 特点 | 加价方向 |
|---|---|---|
| **Top of Search（TOS，搜索结果顶部）** | 黄金位置，CTR 最高，但 CPC 也最贵 | 转化好、想冲排名时加价（如 +50%、+100%） |
| **Product Pages（产品详情页）** | CTR 天然偏低，但便宜 | 一般不优先加价 |
| **Rest of Search（其余位置）** | 中间位置，CTR 和 CPC 都中等 | 维持默认 |

> 关键理解：TOS 加价不是"额外加钱"，是"在这个位置上，把出价按比例放大"。比如基础出价 1 美金，TOS 加价 100%，那 TOS 位置上你最多出 2 美金。

### 什么时候加价 / 降价

| 情况 | 判断 | 动作 |
|---|---|---|
| ACOS < BE ACOS（赚钱且想要更多量） | 该词表现好，可以放量 | **加价 10% 左右**，或加 TOS 加价 |
| ACOS > BE ACOS（亏钱） | 该词太贵或转化不够 | **降价**（按比例降） |
| 词完全跑不出 Impressions | 出价太低，根本拿不到展示 | **加价**直到有展示 |
| CTR 很低（即使展示很多没人点） | 不是出价问题，是 Listing / 主图问题 | **别动出价**，先优化 Listing |

### 数据要看 Targeting 级，不是广告活动平均

**新手最大的错误**：看广告活动整体的 ACOS 是 15%，觉得不错，就不动了。

问题：**广告活动平均会把好词和烂词混在一起**。

举例：

```
广告活动整体 ACOS = 15%（看似不错）
  ├── 词 A：ACOS 5%（赚钱，应该加价放量）
  ├── 词 B：ACOS 80%（严重亏钱，应该降价或否定）
  └── 词 C：ACOS 20%（盈亏平衡）
```

如果只看整体 15%，你会错过"词 A 可以放量赚更多"和"词 B 在偷偷烧钱"。

> 永远在 **Targeting Report（投放报告）** 里按词看数据，再决定每个词的出价。这是 5.8 课要讲的四张报表之一。

### 调出价前需要多少数据

判断一个词的出价该不该调，**至少需要 20 次点击**（Targeting 级）。

为什么是 20：样本太小（如 3 次点击）的 ACOS 完全是偶然，调了也是瞎调。

### 调多少：两个简单公式

**情况 1：ACOS 太高（高于 BE ACOS）→ 降价**

```
降价幅度 ≈ (当前 ACOS - 目标 ACOS) / 当前 ACOS
例：当前 ACOS 40%，目标 ACOS 20%
    降幅 = (40 - 20) / 40 = 50% → 出价降一半
```

**情况 2：ACOS 很低（低于 BE ACOS，想要更多量）→ 加价**

```
加价幅度 ≈ 10%（小步加，观察一周再加）
```

> 调价是"小步快走"，不是"一次到位"。每次调完观察 3-7 天再决定下一步。

## 后台操作：在广告后台改出价、改策略、改位置加价

> 前面的内容讲的是"该怎么判断"，这一节讲"判断完了，在后台具体点哪里"。

### 操作一：调某个关键词的出价（按词调，不是按活动）

完整路径：

1. 登录卖家后台（Seller Central）→ 左上角主菜单（三条横线）→ **Advertising（广告）** → **Campaign Manager（广告活动管理）**。
2. 在广告活动列表里，**点击你要优化的那个广告活动的名字**（比如你的 Exact 广告 `20260712_YM01_SP_Exact_收割`）。
3. 进入广告活动详情页后，往下找到 **Ad groups（广告组）**，**点击广告组名字**进入广告组。
4. 广告组页面默认显示几个 Tab：**Keywords（关键词）**、**Search Terms（搜索词）**、**Negative keywords（否定关键词）** 等。**点开 Keywords 这个 Tab**。
5. 这里列出你投放的每个关键词，每一行后面都有一个 **Bid（出价）** 输入框。
6. **找到你要调的那个关键词那一行**，直接点击 Bid 框里的数字，改成新出价（比如从 0.80 改成 0.90）。
7. 改完**点输入框外面（失焦）或按 Tab**，页面通常会自动保存；个别版本需要点 **Save changes（保存更改）** 按钮。
8. 改完页面顶部会出现一个提示条确认已保存。

> 重要：调 ASIN 投放（商品投放）的出价，是在 **Product targeting（商品投放）** Tab 里，不是 Keywords Tab。步骤一样——找到那行，改 Bid。

### 操作二：改广告活动的出价策略（Dynamic Down / Up-Down / Fixed）

1. 进入 **Campaign Manager** → **点击广告活动名字**进入广告活动详情页。
2. 在广告活动名称旁边，有个 **Settings（设置 / 齿轮图标）** 或 **Edit campaign settings（编辑广告活动设置）**，点进去。
3. 在设置弹窗里找到 **Bidding strategy（出价策略）** 这一栏，是个三选一的单选项：
   - `Dynamic bids - down only`（动态出价 - 仅降低）= Dynamic Down
   - `Dynamic bids - up and down`（动态出价 - 提高和降低）= Dynamic Up/Down
   - `Fixed bids`（固定出价）= Fixed
4. 选你想要的策略 → 点 **Save（保存）**。

### 操作三：设置展示位置加价（Placement Multipliers）

展示位置加价是在**广告活动层级**设置的（不是广告组，也不是单个词）：

1. **Campaign Manager** → **点击广告活动名字**进入详情页。
2. 页面左侧或下方找到 **Placement（展示位置）** 这个区块（如果没看到，先进 **Settings / Edit campaign settings**，在里面找到 Placement 设置；新版后台有时把它放在广告活动主页的下半部分）。
3. Placement 区块有三个位置可以分别设置百分比加价：
   - **Top of Search（搜索结果顶部）**：输入框里填 `+50%`、`+100%`（最高 +900%）。
   - **Product Pages（商品详情页）**：同样填百分比。
   - **Rest of Search（其余位置）**：通常不用动。
4. **注意**：出价策略必须**不是 Fixed** 时，Placement 加价才生效（Fixed 模式下系统不接受位置加价）。
5. 填好百分比 → **Save（保存）**。

> 含义再确认一次：基础出价 1 美金，TOS 填 +100%，意思是 TOS 位置上你最多愿意出到 2 美金（1 × 2）。它不会让你在非 TOS 位置多花钱。

### 操作四：调广告活动预算（顺便讲，经常配合出价调）

1. **Campaign Manager** → 列表里直接能看到每个广告活动的 **Daily budget（日预算）** 一栏。
2. 点击预算数字 → 直接改成新数字 → 失焦自动保存。

## 实操要点

- **新手默认选 Dynamic Down**：等熟练后再用 Dynamic Up/Down 冲排名。
- **调价按词调，不按活动调**：永远在 Keywords Tab 里按词改 Bid，参考 Targeting Report 的词级数据。
- **数据不够不要调**：≥20 次点击才动手，否则是赌博。
- **小步调整**：每次 10%-20%，调完观察 3-7 天。
- **TOS 加价不是万能**：只有当你的主图 / 标题 / 价格在 TOS 位置有竞争力时，加价才划算。否则花双倍钱还是没人点。
- **后台改完确认已保存**：改出价后看页面顶部有没有绿色保存提示，避免改了没生效。

## 动手练习

某词的数据：30 次点击，Spend 30 美金，Sales 60 美金。你的 BE ACOS 是 20%，目标 ACOS 也是 20%。

1. 这个词的 ACOS 是多少？
2. 该加价还是降价？为什么？
3. 如果决定调价，按公式算应该调多少？
4. 调完后你会观察多久再决定下一步？
