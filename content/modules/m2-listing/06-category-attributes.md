# 3.7 类目与后台属性

> **模块3：Listing搭建基本功**
> **本课目标**：选对类目、填全后台属性，让你的产品在筛选和搜索里被正确露出

## 这节课解决什么问题

很多新手只盯着标题和五点，把后台属性随便填填就提交了。结果产品在筛选器里永远露不出来，明明买家筛选"红色、不锈钢"也搜不到你。这节课讲类目和后台属性为什么是 Listing 的隐形骨架。

## 核心知识

### 为什么类目很重要

类目（Category）决定了三件事：

| 影响 | 说明 |
|------|------|
| 必填属性 | 不同类目要填的字段不一样（服装要填尺码，电子要填电压） |
| 佣金费率 | 不同类目佣金不同（服装 17%，电子配件 8-15%） |
| 出现位置 | 浏览类目树时你出现在哪个节点，决定你能蹭到什么浏览流量 |

选错类目 = 你的产品出现在错的浏览路径里，买家在正确的类目树里翻不到你。

### 怎么选对类目

1. **搜你的核心词，看销量前 10 的产品挂哪个类目**：这是最直接的参考。
2. **选最具体（最深层）的那个节点**：能放 "Sports > Cycling > Bike Lights" 就别只放 "Sports"，越深节点竞争越小、流量越准。
3. **别跨类目蹭流量**：把保温杯放进"电子产品"类目蹭流量，会被亚马逊判定类目不符，移除 Listing。

### 后台属性是什么

后台属性（Attributes / Item Details）是创建 Listing 时那一长串字段：颜色、尺寸、材质、数量、目标人群、用途……新手最常见的错误是**只填必填项（标 * 号的），跳过推荐项**。

这是一个大坑。

### 为什么推荐属性也必须填

后台属性喂给亚马逊两套系统：

1. **A9 搜索**：属性帮 A9 理解你的产品到底是什么，匹配更精准的搜索词。
2. **筛选器**：买家在搜索结果左侧用筛选器勾选"红色""不锈钢""成人"时，只有填了对应属性的 Listing 才会出现。

举例：买家搜"water bottle"，然后左侧筛选"Material: Stainless Steel" + "Color: Black"。如果你没填材质或颜色属性，**即使你的产品就是黑色不锈钢水杯，也不会出现在筛选结果里**。

这就是为什么"少填一个属性 = 少一片流量"。

### 常见的关键属性

| 属性 | 为什么重要 |
|------|------------|
| 颜色（Color） | 筛选器高频项，影响变体分组 |
| 尺寸（Size） | 服装鞋包类必填，影响退货率 |
| 材质（Material） | 食品/母婴类买家必筛 |
| 数量（Item Quantity） | 决定单价比显示对不对 |
| 目标人群（Target Audience） | 男/女/儿童，影响精准曝光 |
| 用途（Use Case） | 室内/户外/运动，匹配场景搜索 |
| 品牌（Brand） | 必填，影响品牌筛选和品牌商店 |
| 节日/场合（Occasion） | 礼品类目的关键筛选 |

### 错填 / 漏填属性的危害

| 情况 | 后果 |
|------|------|
| 漏填材质 | 买家筛"不锈钢"时找不到你 |
| 错填颜色 | 买家收到货发现颜色不符 → 退货 + 差评 |
| 漏填目标人群 | 广告系统无法精准定向，ACOS 变高 |
| 跨类目乱填属性 | 类目不符被亚马逊移除 Listing |
| 数量填错 | 单价显示错乱，买家投诉 |

## 实操要点

1. **创建 Listing 前，先研究同类目的竞品**：看销量前 10 的产品，他们的属性栏都显示了什么，照着补全。
2. **必填项 + 推荐项全填**：宁可多填也不要漏，每个字段都是一条流量入口。
3. **属性值要和实物一致**：颜色、尺寸、材质填错会直接导致退货和差评，比漏填还严重。
4. **变体属性单独处理**：如果你有 3 个颜色，要用变体（Variation）功能正确分组，而不是建 3 个独立 Listing。
5. **上线后定期检查**：亚马逊会不定期新增属性字段，每隔几个月回后台看看有没有新的推荐项要补。

## 动手练习

拿你的产品做一遍属性盘点：

1. 在亚马逊搜你的核心词，找出 5 个销量好的竞品。
2. 看他们的详情页"产品信息"表格里都列了哪些属性，记下来。
3. 对照你准备填的属性清单，找出你漏掉了哪些。
4. 重点检查：颜色、材质、尺寸、数量、目标人群这 5 项填了没。
