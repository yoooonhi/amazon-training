# 3.10 竞品研究方法

> **入门3：Listing搭建基本功**
> **本课目标**：学会一套系统的竞品研究方法，从对手身上低成本学到什么有效、什么无效

## 这节课解决什么问题

新手做产品最怕「凭感觉」：标题怎么写？价格定多少？图片怎么拍？评论怎么攒？——这些问题的答案不全在你脑子里，**你的竞品已经用真金白银帮你测过了**。竞品研究就是花最少的钱、最低的风险，把市场里已经被验证有效的做法学过来。这节课讲一套完整的研究流程。

## 核心知识

### 为什么要研究竞品

| 你能学到 | 怎么学 |
|----------|--------|
| Listing 怎么写最好 | 看前 10 名的标题、五点结构 |
| 价格带在哪 | 看他们的售价和促销 |
| 评论策略 | 看他们评论数、评分、新增速度 |
| 广告策略 | 看谁在打广告、打什么词 |
| 市场机会 | 看哪里他们做得差，你可以更好 |

一句话：**竞品用钱试出来的经验，你免费就能看到。** 不研究竞品等于闭眼做生意。

### 第 1 步：找到真正的竞品

不是所有卖同类产品的都是你的竞品。真正的竞品是**和你抢同一批买家**的产品。

**怎么找**：

1. 在亚马逊搜索你的**主关键词**
2. 看搜索结果**第 1 页的前 10 个自然结果**（不算广告位）——这些是你的直接竞品
3. 看「Sponsored products related to this item」（相关广告商品）——这是广告竞品
4. 从中挑出 **5-7 个 ASIN** 作为长期监控对象

**挑选标准**：

- 同品类、同价位带（价格差不超过 50%）
- 销量稳定（不是一阵风爆款）
- 上架时间和你接近或更久（老链接有参考价值）

> 误区：别只盯着 BSR 第 1 名。第 1 名可能是巨头品牌，参考价值反而低。第 3-10 名的做法对新手更有借鉴意义。

### 第 2 步：拆解每个竞品的 Listing

用一个表格，逐项拆解每个竞品：

| 拆解维度 | 看什么 |
|----------|--------|
| **标题结构** | 主词放哪？品牌名？卖点？规格？ |
| **五点描述** | 第一条讲什么卖点？（第一条最重要）整体强调什么利益？ |
| **图片** | 几张？什么类型（信息图/场景图/对比图）？质量如何？ |
| **A+ 内容** | 有没有？用了哪些模块？ |
| **价格 + 促销** | 售价多少？有没有 coupon、deal？ |
| **评论** | 评论数？平均评分？差评主要抱怨什么？ |
| **Q&A 区** | 买家在问什么？（= 他们的疑虑 = 你的机会） |

> 重点关注 Q&A 和差评：**买家反复问的问题、差评反复吐槽的点，就是你 Listing 必须解决的痛点**。比如 5 个竞品的差评都在说「容易坏」，你的五点第 1 条就该强调「耐用」。

### 第 3 步：分析他们的关键词

光看 Listing 表面不够，要看他们**在哪些词上排名**——这暴露了他们的流量来源。

**工具方法**：用**反查 ASIN 工具**（Reverse ASIN）输入竞品 ASIN，看它排名的关键词：

| 工具 | 说明 |
|------|------|
| **Helium 10 Cerebro** | 主流，输入 ASIN 反查排名词 |
| **SellerSprite / 卖家精灵** | 国内常用，性价比高 |
| **Jungle Scout** | 也可反查 |
| **Brand Analytics** | 官方工具，品牌备案后免费（数据最准） |

**分析思路**：

1. 反查 **3-5 个竞品**，各导出他们的关键词清单
2. **找共同词**：3 个以上竞品都在排名的词 = **被验证的市场需求词**，你一定要用
3. **找机会词**：某个词搜索量高，但只有 1-2 个竞品在排名 = **竞争空隙**，可以抢
4. **找盲区词**：他们都没覆盖、但相关的长尾词 = 你的差异化机会

### 第 4 步：长期监控

竞品研究不是一次性的，市场在变，要**每周跟踪**：

| 监控指标 | 看什么变化 |
|----------|------------|
| **BSR 排名趋势** | 在涨还是跌？（涨=卖得好，跌=可能在掉量） |
| **价格变化** | 涨价？降价？上 deal？ |
| **评论增速** | 每周新增几条？（review velocity） |
| **Listing 改动** | 换图了？改标题了？加 A+ 了？ |
| **广告变化** | 广告位变多/变少？打新词了？ |

用一张电子表格或工具（Helium 10、SellerSprite 都有竞品追踪功能）记录，每周更新一次。

### 第 5 步：把学到的用起来

研究完不是写报告就完了，要落到行动：

| 发现 | 行动 |
|------|------|
| 3 个竞品的五点第 1 条都强调「易清洁」 | 你的五点第 1 条也必须讲「易清洁」（买家最在意） |
| 某竞品图片很糊、信息图缺失 | 你的机会——做更好的图片抢点击和转化 |
| 某关键词搜索量大但没竞品排名好 | 抢这个词，做标题/广告主攻 |
| 差评都在吐槽「电池不耐用」 | 你的卖点突出「长续航」，正面打击 |
| 竞品都上了 coupon | 你也考虑上 coupon 保持竞争力 |

### 重要提醒：数据是估算，别当绝对值

竞品研究工具给的是**估算数据**（销量估算用的是库存差值法，受补货、退货、Buy Box 切换影响）：

- **当趋势看**：排名在涨 / 跌、销量在升 / 降——这个方向性结论可信
- **别当精确数字**：「这个竞品月销 3427 件」可能实际是 2500-4500 之间的任何数

另外：**绝对不要照抄竞品的 Listing 文案**——有商标侵权和抄袭风险，亚马逊也会惩罚重复内容。学的是**结构和思路**，不是抄文字。

## 实操要点

1. **先锁定 5-7 个 ASIN**：范围太大研究不过来，太小看不出趋势，5-7 个刚好。
2. **重点看差评和 Q&A**：这是买家真实痛点的金矿，比看好评有用 10 倍。
3. **关键词要交叉验证**：单个竞品的词不够，至少 3 个竞品对比，共同词才是真需求。
4. **每周固定时间更新**：竞品研究是持续动作，不是一次性。每周花 30 分钟更新追踪表。
5. **数据看趋势不看绝对值**：工具的销量、搜索量都是估算，以方向性变化为准。
6. **学结构不抄文字**：标题怎么组织、五点先讲什么——这些思路可以学，但文案必须自己写。

## 动手练习

为你正在做（或准备做）的产品做一次竞品研究：

1. 在亚马逊搜你的主关键词，把第 1 页前 10 名的 ASIN 记下来，从中挑 5-7 个做长期监控。
2. 用表格拆解这 5-7 个竞品的：标题结构、五点第 1 条卖点、图片数量和类型、价格、评论数、平均评分。
3. 至少看 20 条差评，找出被吐槽最多的 3 个共性问题——这些就是你 Listing 要正面解决的痛点。
4. 如果有反查工具，挑 3 个竞品反查关键词，找出 5 个以上竞品都在排名的「共同词」（市场验证词）。
5. 写出 3 条「基于竞品研究的行动」：你要在标题/五点/图片上做什么不同的事。
