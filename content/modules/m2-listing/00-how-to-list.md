# 3.0 如何上架产品

> **入门3：Listing搭建基本功**
> **本课目标**：学会在亚马逊后台从0创建一个产品Listing的完整操作流程

## 这节课解决什么问题

前面学了Listing合规红线，接下来要知道在后台具体怎么操作——从点"添加产品"到Listing上线，每一步填什么。

## 核心知识

### 上架前的准备

上架前必须准备好这些材料：

| 准备项 | 说明 |
|---|---|
| 产品信息 | SKU、品牌名、产品描述、五点卖点 |
| UPC/EAN条码 | 没有品牌备案需要UPC；有品牌备案可申请GTIN豁免 |
| 产品图片 | 主图+至少6张副图，按合规要求准备好 |
| 类目确认 | 知道产品属于哪个类目（选错类目会影响费用和流量） |
| 合规文件 | 如需CE/CPC等证书，提前准备好 |
| 价格和库存 | 售价、是否FBA发货 |

### 后台操作步骤

#### 第1步：进入添加产品页面

Seller Central → **Catalog** → **Add Products**

两个选项：
- **添加已有产品**：如果别人已经建了同款，你可以跟卖（不建议，容易侵权）
- **添加新产品**：你是第一个上架的 → 选这个 → **I'm adding a product not sold on Amazon**

#### 第2步：选择类目

选择最匹配你产品的类目。类目选择影响：
- 需要填哪些后台属性
- 佣金比例（不同类目佣金不同，通常8-15%）
- 买家在浏览时在哪里找到你

::: tip 选类目的技巧
用关键词搜索亚马逊前台，看同类产品大多在哪个类目。选错类目会导致流量不对、属性要求不对。
:::

#### 第3步：填写基本信息

| 字段 | 填什么 |
|---|---|
| Product ID | UPC/EAN（有品牌备案可豁免） |
| Brand Name | 你的品牌名（需与品牌备案一致，或填Generic） |
| Product Name | 标题（**2026年新规：最多75字符**，入门3.2会详细讲怎么写。另新增125字符商品亮点字段） |
| Manufacturer | 制造商 |
| Item Type | 产品类型关键词 |

#### 第4步：填写五点描述和描述

| 字段 | 说明 |
|---|---|
| Bullet Point 1-5 | 五条核心卖点，每条最多500字符 |
| Product Description | 产品描述（有A+后可以用图文代替） |

五点怎么写在入门3.3会详细讲，这里先知道填在哪。

#### 第5步：填写价格和数量

| 字段 | 说明 |
|---|---|
| Price | 你的售价（US站不含税，EU站含VAT） |
| Quantity | FBM填库存数量；FBA填0（库存由亚马逊仓库管理） |
| Fulfillment | 选择FBA或FBM |

#### 第6步：上传图片

- 主图：白底产品图，至少1000×1000px（建议1600+）
- 副图：最多6张，可以是信息图/生活场景/尺寸图/对比图等

图片规格在入门3.6详细讲。

#### 第7步：填写后台属性

类目选定后，亚马逊会要求填一组属性。分两类：
- **必填属性**：不填无法上架
- **推荐属性**：建议全填，填得越多系统越能准确识别你的产品

常见属性：颜色、尺寸、材质、数量、适用人群、用途等。

#### 第8步：填写Search Terms

在**Keywords**标签页找到**Search Terms**字段：
- 最多250字节（UTF-8）
- 全小写，空格分隔
- 不重复标题里已有的词
- 不用竞品品牌名

Search Terms怎么写在入门3.5详细讲。

#### 第9步：提交保存

点击 **Save and finish**。亚马逊会审核你的Listing：
- 大部分产品几分钟到几小时上线
- 某些类目（如健康、美容、儿童产品）需要额外审核，可能需要1-3天

#### 第10步：上架后检查

Listing上线后，去前台搜索你的产品，检查：
- 标题显示是否完整（移动端可能截断）
- 图片加载是否正常
- 价格是否正确
- Buy Box是否赢得（FBA产品通常自动获得）
- 能不能搜到（用你的关键词搜）

### 常见上架问题

| 问题 | 原因 | 解决 |
|---|---|---|
| 提交后显示"需要审批" | 该类目需要分类审核 | 提交审核材料（发票/证书）等审批 |
| 图片上传失败 | 尺寸不够或格式不对 | 确保JPEG/TIFF，至少1000×1000px |
| UPC报错"无效" | UPC不是正规渠道购买的 | 用GS1正规条码，或申请GTIN豁免 |
| 搜索不到产品 | 刚上线，索引需要时间 | 等24小时再搜；检查类目和关键词是否正确 |
| 变体创建失败 | 父子ASIN关系设置错误 | 检查variation theme和子ASIN属性是否一致 |

### 变体上架

如果一个产品有多个颜色/尺寸：
1. 先创建父ASIN（Parent ASIN，不可售）
2. 在父ASIN下添加子ASIN（Child ASIN，每个可售）
3. 设置variation theme（如Color-Size）
4. 每个子ASIN有自己的SKU、价格、库存

**注意**：不要把不相关的产品硬凑成变体（如把不同功能的产品做成颜色变体），亚马逊会判定变体滥用。

## 实操要点

- 第一次上架最好在老板/管理员指导下操作，不要自己闷头提交
- 上架前把所有内容（标题/五点/Search Terms/图片）先在文档里写好，再往后台填
- 填完每个标签页先点Save保存，不要全填完一次性提交（防止网络中断丢失）
- 上架后一定要去前台搜一遍，确认买家视角看到的是对的

## 动手练习

拿一个真实产品（或让老板给你一个新品），把上架需要的信息整理成一份文档：
1. 确定类目
2. 写好标题
3. 写好五点
4. 写好Search Terms
5. 列出需要填的属性和值
6. 准备好图片清单

然后在老板指导下实际上架一遍。
