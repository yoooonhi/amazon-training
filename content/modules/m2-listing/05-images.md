# 3.6 主图与副图体系

> **模块3：Listing搭建基本功**
> **本课目标**：搭一套合规且高转化的图片体系（1 张主图 + 最多 6 张副图）

## 这节课解决什么问题

买家在搜索结果页第一眼看到的就是图，点进详情页主要也是看图。图片质量直接决定转化率。但亚马逊对主图有严格规定，踩了就被强制下架图。这节课教你既合规又好看的图片怎么规划。

## 核心知识

### 图片为什么是转化的核心

数据上：买家在详情页停留时间 80% 花在看图，文字（标题+五点+描述）只占 20%。一张差的图，主图点击率掉一半；一套差的副图，转化率掉一半。

所以图片不是"配套装饰"，是 Listing 的**主战场**。

### 主图（Main Image）的硬规则

主图是搜索结果页和详情页第一张图，规则最严：

| 规则 | 要求 |
|------|------|
| 背景 | **纯白**（RGB 255, 255, 255） |
| 产品占比 | 占图片面积 **≥85%** |
| 不能有文字 | 不能有标题、卖点、标签 |
| 不能有边框/水印/badge | 一根线、一个 logo 角标都不行 |
| 不能有道具/场景 | 主图只能展示产品本身，不能放使用场景 |
| 最小尺寸 | **≥1000px**（一边或两边） |
| 推荐尺寸 | **≥1600px**，开启放大镜功能（zoom） |
| 格式 | JPEG / TIFF / PNG |

主图踩雷后果：亚马逊会强制把你的主图替换成副图，甚至直接屏蔽 Listing 直到整改。

### 副图（Secondary Images）体系

每个 Listing 最多 6 张副图（加上主图共 7 张，部分类目允许视频和更多图）。每张副图有固定任务：

| 顺序 | 类型 | 作用 |
|------|------|------|
| **副图 1** | 尺寸/规格信息图 | 标注长宽高、容量、重量，买家最想知道的硬指标 |
| **副图 2** | 使用场景图 | 产品在真实环境里被使用的样子（厨房、户外、办公） |
| **副图 3** | 材质/细节图 | 拉近拍材质、做工、关键部件 |
| **副图 4** | 对比图 | 和旧版/竞品对比优势（不点名竞品） |
| **副图 5** | 包装/配件图 | 盒子里有什么，配件全家福 |
| **副图 6** | 卖点强化图 | 把五点描述里最重要的卖点再用图讲一遍 |

副图允许加文字（不像主图），所以信息图（infographic）是副图的主力：在产品图上标注尺寸、特性、用法。

### 顺序为什么重要

手机端默认只显示前 4-5 张图的缩略图，后面要滑动才看到。所以**前 3 张副图必须解决买家最主要的疑虑**：多大、怎么用、什么材质。次要的（对比、包装）放后面。

### A+ 内容（品牌卖家专属）

完成品牌注册的卖家可以加 A+ 内容（也叫 EBC），出现在五点描述下方、评论上方。它的核心模块：

| 模块 | 作用 |
|------|------|
| 品牌故事 | 讲品牌来历、价值观，建立信任 |
| 对比表 | 用表格对比自家不同型号，帮买家选 |
| 交叉销售 | 展示店铺其他产品，提高客单价 |
| 大图 + 文案 | 把核心卖点用图文并茂的方式再讲一遍 |

A+ 的价值：让详情页更专业、停留时间更长、转化率通常提升 5-15%。但前提是品牌注册 + 主图副图已经做好，A+ 是锦上添花，不是雪中送炭。

### 好图 vs 坏图

✅ **好主图**：纯白背景、产品占满画面 85% 以上、清晰锐利、无任何文字水印。

❌ **坏主图 1**：背景是渐变色或场景 → 不合规，会被下架。

❌ **坏主图 2**：产品只占画面 30%，周围大片留白 → 不符合 85% 规则。

❌ **坏主图 3**：图上写着"Best Seller 2024"或贴了 discount badge → 违规。

✅ **好副图**：信息图把尺寸用箭头标清楚、场景图真实自然、对比图直观。

❌ **坏副图**：6 张图全是同角度的产品照，没有信息增量 → 浪费位置，转化低。

## 实操要点

1. **主图找专业拍**：纯白背景 + 85% 占比这两条硬指标，自己手机很难拍达标，建议找产品摄影服务。
2. **副图先做信息图**：尺寸信息图通常是转化提升最大的一张，优先做。
3. **图片尺寸做到 1600px 以上**：开启放大镜，买家能看清细节，转化率更高。
4. **文件名用关键词**：上传前把图片文件名改成包含关键词的英文（如 `wireless-earbuds-noise-cancelling.jpg`），有小幅 SEO 加成。
5. **A+ 内容上线后再做**：没品牌注册先专注 7 张图，别分心。
6. **定期换图测试**：上线 1-2 个月后，把转化最差的副图换掉重做，持续优化。

## 动手练习

为你的产品规划一套 7 张图（写在纸上即可，不用真拍）：

1. 主图：列出拍摄要求（背景、占比、角度）。
2. 副图 1-6：每张写出类型（尺寸图/场景图/材质图/对比图/包装图/卖点图）和要标注的关键信息。
3. 自查主图有没有违反任何一条硬规则（背景纯白？无文字？85% 占比？）。
4. 思考：手机端前 3 张副图，能不能让买家不用看文字就知道这产品适不适合他？
