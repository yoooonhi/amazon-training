# 1.7 EU合规基础

> **入门1：平台认知与账号安全**
> **本课目标**：了解做欧洲站（EU）必须知道的 5 个合规维度，知道每个合规要求"是什么、不做到底会怎样"。

## 这节课解决什么问题

欧洲站的合规比美国站复杂得多——除了亚马逊的规则，还要遵守欧盟各国的法律。很多新手以为"产品卖得好就行"，结果产品刚上架就被下架，甚至面临高额罚款。这节课帮你建立欧洲合规的"全景认知"，知道有哪些坑要提前避开。

## 核心知识

### 欧洲合规的 5 大维度

```
┌─────────────────────────────────────────────────┐
│            欧洲站合规 5 大维度                    │
├─────────────┬─────────────┬─────────────────────┤
│  1. 税务    │  2. 环保    │  3. 产品安全        │
│  (VAT)      │  (EPR等)    │  (GPSR/CE等)        │
├─────────────┴─────────────┴─────────────────────┤
│   4. 标签（多语言）  │   5. 平台（类目审核）    │
└─────────────────────────────────────────────────┘
```

这节课只讲概念，让你知道"这是什么"，不展开操作细节。

### 维度1：税务合规 —— VAT（增值税）

- **是什么**：在欧洲卖货必须缴纳增值税（Value Added Tax），类似国内的增值税。
- **关键点**：
  - 在哪个国家有库存（包括亚马逊 FBA 仓），就要在哪个国家注册 VAT 税号并申报。
  - 用了德国 FBA 仓 → 要注册德国 VAT；用了法国 FBA 仓 → 要注册法国 VAT。
- **不做的后果**：账号被冻结、FBA 库存被扣押、补缴税款 + 罚款。

### 维度2：环保合规 —— EPR / LUCID / WEEE

这是新手最容易忽略、但罚款最狠的一块。

| 法规 | 适用国家 | 管什么 | 一句话理解 |
|---|---|---|---|
| **EPR**（生产者责任延伸） | 欧盟各国 | 包装、电子电气、电池等的回收 | 你卖的东西用完扔了，谁负责回收处理？你得出钱 |
| **LUCID**（包装法） | 德国 | 所有产品的**包装** | 只要卖到德国，包装就要注册 LUCID |
| **WEEE**（电子电气） | 欧盟 | 电子电气产品的回收 | 卖电子产品必须注册 |

- **不做的后果**：Listing 强制下架、高额罚款（德国 LUCID 罚款可达 10 万欧元）、账号冻结。
- **常见踩坑**：卖家以为"我卖的是产品不是包装"——错，产品的外包装、快递箱都算。

### 维度3：产品安全合规 —— GPSR / CE / Responsible Person

这是 2024 年后变得非常严格的领域。

| 要求 | 是什么 | 适用产品 |
|---|---|---|
| **GPSR**（通用产品安全法规） | 欧盟新的通用安全法规，要求所有非食品消费品都有"责任人" | 所有非食品消费品 |
| **Responsible Person（欧盟授权代表）** | 在欧盟境内指定的合规联系人，代表你承担安全责任 | 所有需要 GPSR 的产品 |
| **CE 认证** | 产品符合欧盟安全/健康/环保标准的标志 | 电子产品、玩具、儿童用品、医疗器械等 |
| **EU 1223/2009（化妆品法规）** | 化妆品的成分、标签、安全评估要求 | 化妆品、护肤品 |

- **关键点**：
  - 卖到欧盟必须有**欧盟授权代表**（Responsible Person），可以是服务商代理。
  - 电子/玩具/儿童产品必须有 **CE 标志** + 对应的技术文件。
  - 化妆品需要做**CPNP 通报**（化妆品产品通报门户）。

### 维度4：标签合规 —— 多语言警告语

- **是什么**：产品销往哪个国家，警告语、说明书就要用对应语言。
- **例子**：
  - 卖到德国 → 德语警告语
  - 卖到法国 → 法语警告语
  - 卖到意大利、西班牙 → 意语、西语
- **不做的后果**：产品被海关扣留、Listing 被投诉下架。

### 维度5：平台合规 —— 类目审核

- 亚马逊对某些敏感类目（如医疗器械、儿童玩具、食品、化妆品、电池）要求卖家**提交资质材料**才能上架。
- 即使法律层面合规，平台审核没过也卖不了。

### 5 个维度的关系

不是"满足一个就行"，而是**都要满足**。比如卖一个电子玩具到德国：
- VAT（税务）✓
- EPR/LUCID（包装环保）✓
- CE 认证 + GPSR + 欧盟代表（产品安全）✓
- 德语警告语（标签）✓
- 玩具类目审核（平台）✓

缺任何一个，这个产品就卖不了，或卖了被罚。

## 上架前合规自检清单

把这张清单存下来，每次上架欧洲产品前过一遍：

| 维度 | 自检项 | ☑ |
|---|---|---|
| 税务 | 目标国家的 VAT 税号已注册并生效 | ☐ |
| 环保 | 德国 LUCID 包装注册已完成 | ☐ |
| 环保 | 电子/电池产品的 WEEE 已注册 | ☐ |
| 安全 | 欧盟授权代表（Responsible Person）已指定 | ☐ |
| 安全 | 需要的产品认证（CE/化妆品 CPNP）已办理 | ☐ |
| 标签 | 目标国家语言的警告语/说明书已准备 | ☐ |
| 平台 | 类目审核所需资质材料已准备 | ☐ |

任何一项打不了勾，就**先不要上架**。

## 实操要点

- **欧洲合规要"前置"**：不要等产品上架了才想起来办合规——很多认证需要几周甚至几个月，等你想补的时候，Listing 早就被下架了。
- **找专业服务商**：VAT、EPR、CE、欧盟代表这些，普通人搞不定，找靠谱的合规服务商代办，比自己摸索快也安全。
- **不同国家要求不同**：德国、法国、西班牙、意大利各有各的细则，不要假设"一个国家合规了全欧盟就合规"。
- **合规成本要算进利润**：欧洲合规费用不低（VAT 申报费 + EPR 注册费 + CE 测试费 + 欧盟代表年费），选品时要算清楚。
- **关注政策更新**：欧盟法规每年都在变（如 2024 年 GPSR 上线），要定期跟进。

## 动手练习

假设你要把一款"带电池的儿童玩具"卖到德国，对照本课的"上架前合规自检清单"，列出：
1. 你需要办理哪些合规事项？（至少 5 项）
2. 每一项大概需要多久、找谁办？
3. 如果其中两项办不下来，会发生什么？

完成后，你就明白为什么欧洲合规不能临时抱佛脚了。
