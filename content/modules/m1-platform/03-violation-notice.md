# 1.3 收到违规通知怎么办

> **入门1：平台认知与账号安全**
> **本课目标**：学会收到违规通知后"前 30 分钟"的标准处理流程，避免因为慌乱把事情搞得更糟。

## 这节课解决什么问题

收到违规通知那一刻，新手的本能反应是"赶紧申诉、赶紧删 Listing、赶紧认错"。但这些"赶紧"的动作，往往是把事情弄更糟的开始。这节课教你收到通知后**前 30 分钟该做什么、不该做什么**。

## 核心知识

### 前 30 分钟标准流程

收到违规通知（Performance Notification / 账户状况警告）后，按下面这个时间表行动：

| 时间 | 你要做的事 |
|---|---|
| **0-5 分钟** | 通读通知全文，搞清楚到底违反了哪条政策 |
| **5-10 分钟** | 截图、保存通知原文（防止通知消失或被覆盖） |
| **10-15 分钟** | 判断严重程度：是单个 Listing 被下架，还是整个账号有风险？ |
| **15-20 分钟** | 收集证据（采购发票、授权书、检测报告、合规证书等） |
| **20-30 分钟** | 决定是否需要求助（上级、服务商、品牌方） |

### 第一步：搞清楚到底违反了什么

通知里通常会写明：违反了哪条政策（如"二手当新品""涉嫌侵权""商品安全投诉"）。**先读完通知，再动手**，不要看个标题就开始申诉。

常见的违规类型：

| 类型 | 典型例子 | 严重程度 |
|---|---|---|
| Listing 不合规 | 标题用了别人的品牌词、写了"治疗""100%"等违禁词 | 中（可改） |
| 知识产权侵权 | 被投诉商标/专利/版权侵权 | 高（可能封号） |
| 商品真实性问题 | 买家投诉"二手当新品""货不对板" | 高（影响 ODR） |
| 商品安全投诉 | 买家说产品起火、有伤害风险 | 极高 |
| 操纵评论 | 涉嫌刷单、买好评、返现好评 | 极高（直接封号） |

### 第二步：截图保存

通知可能在后台只显示一段时间，或被新通知覆盖。**第一时间截图保存**（包括通知标题、正文、涉及的 ASIN、政策编号）。后面写申诉、找服务商、内部复盘都要用。

### 第三步：判断严重程度

- **单个 Listing 被下架**：账号本身没风险，先把这个 Listing 的问题解决，不影响其他产品。
- **账号级别警告**（账户状况页面出现红点）：必须重视，否则可能升级为暂停。
- **直接暂停**：账号已经不能用，必须立刻写 POA 申诉。

### 第四步：收集证据

根据违规类型，准备对应的证据材料：

| 违规类型 | 可能需要的证据 |
|---|---|
| 商品真实性 / 二手当新品 | 供应商发票（含 ASIN、数量、供应商盖章） |
| 商标侵权 | 品牌授权书、商标注册证 |
| 商品安全 | 检测报告、合规认证（CE/FCC/UL 等） |
| 专利侵权 | 专利不侵权分析（这个通常需要律师） |

### 第五步：决定是否求助

- 涉及法律（专利侵权、商标纠纷）→ 找知识产权律师或服务商
- 涉及账号暂停、申诉多次被拒 → 找有经验的同事/服务商
- 普通的 Listing 改一下就好 → 自己处理

### POA（行动计划）三段式

申诉信的标准格式叫 **POA（Plan of Action）**，必须包含三部分：

```
1. 根本原因（Root Cause）
   —— 到底为什么会出问题？不是甩锅，是诚实分析自己哪里没做好。

2. 整改措施（Corrective Actions）
   —— 已经做了什么来修复当前问题？（如下架了违规 Listing、培训了员工）

3. 预防措施（Preventive Measures）
   —— 以后怎么保证不再发生？（如建立了上架前的合规检查流程）
```

**关键点**：亚马逊看的是"你是不是真的理解了问题，并有系统性的预防方案"，而不是"你道歉得有多诚恳"。

## 绝对不要做的 4 件事

| ❌ 错误动作 | 为什么不能做 |
|---|---|
| 没读懂通知就提交申诉 | 答非所问，浪费申诉机会，还可能暴露更多问题 |
| 反复提交同一份申诉 | 每次被拒，下次通过的难度更大；多次被拒可能直接封号 |
| 立刻删除违规 Listing | 删了等于"销毁现场"，申诉时没有证据可查，反而显得心虚 |
| 自己乱写 POA 凑数 | POA 是结构化的，乱写会让亚马逊认为你根本没认真对待 |

## 实操要点

- **冷静 24 小时**：收到通知后，除非是紧急的账号暂停，否则先冷静半天再行动，慌乱中容易做错决定。
- **保留所有沟通记录**：和亚马逊、买家、供应商的所有相关邮件、消息都存档。
- **POA 要具体**：写"我们会加强培训"是废话，写"我们建立了上架前 3 人复核清单，已执行 2 周"才是亚马逊想看的。
- **申诉被拒不要立刻重交**：先想清楚上次哪里没说服亚马逊，修改后再交。

## 动手练习

假设你收到这样一条通知：*"Your listing has been removed due to a customer complaint about the product being used."*（你的 Listing 因买家投诉产品被使用过而被下架）

按照本课的流程，在纸上写下：
1. 这是哪种类型的违规？（提示：商品真实性/二手当新品）
2. 你需要收集哪些证据？（提示：发票）
3. POA 三段式里，"根本原因"你会怎么写？
