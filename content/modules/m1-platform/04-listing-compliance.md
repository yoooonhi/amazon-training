# 1.4 Listing合规红线

> **入门1：平台认知与账号安全**
> **本课目标**：识别 Listing 里最容易触发审核或下架的"红线词"和"红线图片"，学会在写文案时主动避开。

## 这节课解决什么问题

Listing 被下架、被审核，90% 是因为**用词不当**或**图片不合规**。很多新手不是故意违规，而是不知道某些词会触发系统审核。这节课帮你建立一张"避坑词表"，从源头减少 Listing 出问题的概率。

## 核心知识

### 5 类 Listing 合规红线

#### 1. 商标/品牌名侵权

- **绝对不能**：在没有授权的情况下，在标题、关键词、描述里写别人的品牌名或变体（如卖通用蓝牙耳机却写"AirPods"）。
- **常见踩坑**：.compatibles、.fits、.for XX（品牌）——亚马逊会判定你在蹭别人品牌流量。
- **正确做法**：只写自己的品牌（已备案的），或写通用描述词（如"无线蓝牙耳机"）。

#### 2. 医疗/健康功效宣称

| ❌ 违禁词（触发审核） | ✅ 安全替代 |
|---|---|
| 治疗、治愈、修复（treat/cure/heal） | 舒缓、帮助维护（soothe/supports） |
| 预防疾病、降血压、降血糖 | 不提任何疾病名 |
| 抗菌、杀菌（antibacterial） | （除非有 EPA 认证，否则完全不用） |
| 处方、药用、临床验证 | 不写 |

一旦写了这类词，产品会被判定为"医疗器械/药品"，需要走 MDR（医疗器械法规）审核，绝大多数普通产品根本过不了。

#### 3. 绝对化用语

| ❌ 违禁词 | 为什么不能写 |
|---|---|
| 最/第一（best/first/#1） | 广告法 + 亚马逊禁止夸大宣传 |
| 100%、永久、终身（100%/permanent/lifetime） | 无法证明，易被投诉虚假宣传 |
| 唯一、独家（only/exclusive） | 同上 |

#### 4. 安全性宣称（无认证）

| ❌ 违禁词 | 触发什么 |
|---|---|
| 无毒（non-toxic） | 需要毒理学检测报告 |
| 阻燃（fireproof/fire-resistant） | 需要阻燃测试认证 |
| 儿童安全（child-safe） | 触发儿童用品类目审核 |
| 食品级（food-grade） | 需要食品接触材料认证 |

#### 5. 敏感类目触发词

| ❌ 违禁词 | 触发什么 |
|---|---|
| 有机（organic） | 需要有机认证（USDA Organic 等） |
| 天然（natural） | 部分类目需审核 |
| BPA Free | 需要检测报告 |
| 医用级、食品级 | 触发对应类目审核 |

### 主图规则（硬性要求）

主图是 Listing 的"门面"，亚马逊对它的要求非常严格：

| 要求 | 说明 |
|---|---|
| 纯白背景 | RGB(255,255,255)，不能有任何底色 |
| 产品占比 ≥ 85% | 不能有大片留白，产品要撑满 |
| 无文字/水印/徽章 | 不能写"促销""包邮""New"等 |
| 无边框 | 不能加装饰边框 |
| 单一产品 | 主图只能展示一个产品（套装除外） |
| 真实展示 | 不能用过度 PS、不能展示产品没有的功能 |

### 不要照抄竞品 Listing

- **版权风险**：竞品的图片、文案、五点描述都是原创内容，照抄属于版权侵权。
- **会被识别**：亚马逊系统会比对 Listing 内容，雷同度过高直接判定抄袭。
- **正确做法**：参考竞品的卖点结构，但用自己的语言和图片重写。

## 高风险词替换速查表

把这张表存下来，写 Listing 时对照检查：

| 风险词 | 安全替代 |
|---|---|
| 治疗/治愈（treat/cure） | 舒缓/帮助（soothe/support） |
| 100% | 不写绝对值，或写"高达 X%" |
| 最好的/第一 | 删掉，或写"高品质" |
| 抗菌/杀菌 | 删掉（除非有 EPA 注册号） |
| 无毒/食品级 | 删掉（除非有检测报告） |
| 有机 | 删掉（除非有有机认证） |
| 阻燃 | 删掉（除非有阻燃认证） |
| 终身/永久 | 写具体时长（如"12 个月"） |

## 实操要点

- **上架前必做**：让文案过一遍"红线词自查"，可以用上面的速查表，也可以用合规检查工具。
- **不要心存侥幸**：亚马逊的审核系统是机器+人工，你以为"打擦边球没人发现"，其实系统早就标记了。
- **中英文都要查**：如果你做欧洲站，德语/法语/西语/意语的翻译里也可能带出违禁词。
- **改 Listing 有风险**：每次改标题、五点、主图，都可能触发重新审核。改之前备份原版，改完观察 48 小时销量和排名。
- **A+ 页面同样适用**：A+ 内容（图文版详情）也受同样的合规约束，不要以为只有 Listing 五要素才查。

## 动手练习

挑一个你正在卖（或准备卖）的产品的 Listing 标题和五点描述，用本课的"高风险词替换速查表"逐字检查：
1. 找出所有可能的违禁词
2. 写出安全替代
3. 检查主图是否符合纯白背景、无文字、产品占比 ≥ 85%

如果找到了违禁词，先别急着改——记下来，确认后再批量修改。
