# 1.0 亚马逊基础知识和专业术语

> **入门1：平台认知与账号安全**
> **本课目标**：搞清楚亚马逊到底是什么平台、掌握最常用的专业术语，后面所有课都建立在这些概念上

## 这节课解决什么问题

新人刚接触亚马逊，满屏幕的缩写和术语——SKU、ASIN、FBA、FNSKU、Buy Box、ACOS……看不懂就没法往下学。这节课把最基础的概念一次讲清楚。

## 核心知识

### 亚马逊是什么

亚马逊既是**电商平台**（像淘宝/京东，卖家在上面卖东西给消费者），也是**物流服务商**（帮卖家存货、发货），还是**广告平台**（卖家可以投广告获取流量）。

你作为卖家，需要在这三个角色之间打交道：
- 平台规则 → 合规、Listing、账号健康
- 物流服务 → FBA发货、库存管理
- 广告平台 → SP/SB/SD广告

### 必须掌握的术语表

#### 产品标识

| 术语 | 全称 | 含义 |
|---|---|---|
| **SKU** | Stock Keeping Unit | 卖家自己给产品的编号，你自己定，比如 WATERBOTTLE-32OZ-BLK |
| **ASIN** | Amazon Standard Identification Number | 亚马逊给每个产品的唯一编号，10位字母数字，如 B08XYZ1234 |
| **FNSKU** | Fulfillment Network SKU | 亚马逊物流给的条码编号，贴在产品包装上，用于FBA仓库识别 |
| **UPC/EAN** | Universal Product Code | 商品条形码，上架时通常需要（有品牌备案可用GTIN豁免） |

#### 物流相关

| 术语 | 含义 |
|---|---|
| **FBA** | Fulfillment by Amazon——你把货发到亚马逊仓库，亚马逊负责存货、拣货、包装、发货、退货 |
| **FBM** | Fulfillment by Merchant——你自己存货自己发货 |
| **FBA Fee** | 亚马逊收的物流费，包括配送费（按件）和仓储费（按体积/时间） |
| **Inbound** | 入库——你发往亚马逊仓库的货在途中或正在被接收 |
| **Sellable Inventory** | 可售库存——买家能买到的库存 |
| **Reserve/Transfer** | 保留/调拨中的库存——暂时不能卖 |
| **Buy Box** | 购物车——产品页面右侧的"Add to Cart"按钮，赢得Buy Box才能被买家直接购买 |

#### 账号相关

| 术语 | 含义 |
|---|---|
| **Seller Central (SC)** | 卖家后台——你每天操作的管理界面 |
| **Brand Registry** | 品牌备案——注册后可以用A+内容、品牌广告、品牌保护等功能 |
| **AHR** | Account Health Rating——账号健康评级（绿/黄/红） |
| **ODR** | Order Defect Rate——订单缺陷率，红线 <1% |
| **A-to-Z** | A-to-Z Guarantee Claim——买家向亚马逊投诉订单问题的索赔 |
| **Vine** | 亚马逊官方测评计划，免费送产品给测评人换取评价 |
| **Prime** | 会员标识，FBA产品通常自动获得Prime标志，享有快速配送 |

#### Listing相关

| 术语 | 含义 |
|---|---|
| **Listing** | 产品详情页——标题、图片、五点、描述、价格的组合 |
| **Title (Item Name)** | 产品标题——2026年新规限制**75字符**，是搜索权重最高的字段 |
| **Item Highlights** | 商品亮点——2026年新增字段，**125字符**以内，用逗号分隔的核心卖点，显示在标题下方 |
| **Bullet Points** | 五点描述——产品详情页的五条核心卖点 |
| **Search Terms** | 后台搜索词——买家看不到，但影响你的产品被搜到 |
| **A+ Content** | 增强内容——品牌备案后可用，图文并茂的产品描述模块 |
| **Variation** | 变体——同一产品的不同颜色/尺寸/款式的组合 |
| **Parent ASIN** | 父ASIN——变体组的总ASIN，本身不可售 |
| **Child ASIN** | 子ASIN——变体组里的每个具体可售款式 |

#### 广告相关

| 术语 | 含义 |
|---|---|
| **SP** | Sponsored Products——商品推广广告，最常用，出现在搜索结果和产品页面 |
| **SB** | Sponsored Brands——品牌推广广告，需要品牌备案 |
| **SD** | Sponsored Display——展示广告，可追踪离开亚马逊的买家 |
| **ACOS** | Advertising Cost of Sales——广告花费/广告销售额，衡量广告效率 |
| **TACOS** | Total ACOS——广告花费/总销售额（含自然单），衡量广告依赖度 |
| **CPC** | Cost Per Click——每次点击成本 |
| **CTR** | Click-Through Rate——点击率=点击/曝光 |
| **TOS** | Top of Search——搜索结果第一页顶部位置，最贵的广告位 |
| **Auto Campaign** | 自动广告——亚马逊帮你选关键词 |
| **Manual Campaign** | 手动广告——你自己选关键词或ASIN投放 |

#### 费用相关

| 术语 | 含义 |
|---|---|
| **Referral Fee** | 销售佣金——亚马逊按类目收，通常12-15%，从售价中扣 |
| **VAT** | 增值税——EU站点售价含VAT，各站点税率不同 |
| **Closing Fee** | 单件交易费——每笔订单的固定费用（通常$0.30-$1.00） |

### 亚马逊后台基本导航

Seller Central 的主要菜单：

| 菜单 | 你在这里做什么 |
|---|---|
| **Catalog** | 管理产品Listing（上架、编辑、查看） |
| **Inventory** | 管理库存（FBA库存、发货计划、库存报告） |
| **Pricing** | 管理价格（调价、促销、Coupon） |
| **Orders** | 管理订单（查看订单、退货、买家消息） |
| **Advertising** | 管理广告（Campaign Manager、报表） |
| **Stores** | 品牌店铺（需品牌备案） |
| **Brands** | 品牌工具（Brand Analytics、Vine、A+） |
| **Performance** | 账号健康、绩效指标、政策合规 |

## 实操要点

- 术语不用硬背，用到的时候回来查这张表就行
- 最容易混淆的：SKU（你自己的编号）vs ASIN（亚马逊的编号）vs FNSKU（物流条码）
- 最容易搞错的：ACOS（只看广告）vs TACOS（看整体）——后面广告单元会反复用

## 动手练习

打开亚马逊卖家后台（或让老板给你看看），找到以下信息：
1. 找一个产品，看清它的 SKU、ASIN、FNSKU 分别是什么
2. 在 Performance 菜单里找到 Account Health，看看当前是什么颜色
3. 在 Advertising 菜单里找到 Campaign Manager，看看有没有正在跑的广告
