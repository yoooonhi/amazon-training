# 6.3 调价是原子操作

> **入门6：定价与利润**
> **本课目标**：搞清楚"改价"不是改一个数字那么简单，而是一次必须把整条利润链重算的"原子操作"。

## 这节课解决什么问题

新手改价经常是这样的：听说竞品降价了，自己也跟着降 €1，"先降了再说，利润回头再算"。结果降完才发现：利润率掉到 5%、广告开始亏钱、库存只够 10 天还降价（加速断货）——一个动作引发了三个新问题。

问题出在：**改价不是改一个数字，是触发了一连串连锁反应**。VAT 变了、佣金变了、可能的 FBA 费变了、净收益变了、利润率变了、BE ACOS 变了、广告安全线也变了。这些必须**一起重算**，不能"改了再说"。

这节课告诉你：改价前必须重算什么，什么情况绝对不能改，以及改完怎么检查广告会不会从赚钱变亏钱。

## 核心知识

### 什么是"原子操作"

"原子操作"的意思是：**这件事必须作为一个整体一次性完成，不能拆成"先做一半，另一半以后再说"**。

改价就是原子操作。改一个价格，你必须**在改的当下**就把整条利润链在新价格上重新算一遍，确认每一项都还成立，然后才能动手。不能"先改价格，明天再算利润"。

### 改价后，下面这些全部要重算

| 项目 | 为什么会变 | 怎么算 |
|---|---|---|
| VAT 金额 | 按新售价算 | 新售价 − 新售价÷(1+税率) |
| 佣金 | 基数是含税售价 | 新含税售价 × 15% |
| FBA 配送费 | 跨尺寸/重量段会变 | 查新尺寸段对应的费率 |
| 数字服务费 | 按含税售价算 | 新含税售价 × 费率 |
| 净收益 | 上面几项都变了 | 售价 − VAT − 佣金 − FBA − 数字服务费 |
| 利润率 | 净收益变了 | 净利润 ÷ 售价 |
| BE ACOS | 利润结构变了 | 重新算盈亏平衡广告成本比 |
| 广告安全线 | BE ACOS 变了 | 看当前 ACOS 还在不在安全区 |

**任何一个没重算，你都可能在不知不觉中把一个赚钱的产品改成亏钱的。**

### 改完价，立刻检查广告会不会"由赚变亏"

这是最容易被忽略的一步。BE ACOS（盈亏平衡 ACOS）是跟着利润率走的——**利润率下降，BE ACOS 也下降**，意味着广告能容忍的 ACOS 变低了。

举个例子：

```
改价前：利润率 18%，BE ACOS = 18%（当前广告 ACOS 12%，安全）
改价后：利润率 12%，BE ACOS = 12%（当前广告 ACOS 还是 12%，刚好踩线！）
```

如果改价后 BE ACOS 跌到比当前广告 ACOS 还低，说明**广告从赚钱变成亏钱了**，必须立刻去做广告复盘（同步调广告）——降出价、砍亏钱关键词、收紧预算，把 ACOS 拉回新的安全线以内。

**判断口诀**：改完价，新 BE ACOS < 当前广告 ACOS → 立刻做广告复盘。

### 改价的三条红线（碰线就不能改）

有些情况，不管竞品怎么动，你都不能跟着改：

1. **改价后利润率 < 10%**：这是基本生存线。利润率低于 10%，一个退货、一次广告波动就把利润吃光，等于白干。
2. **新价格低于 SC（供货价）最低价**：如果这个产品有供货价底线（比如跟供应商/品牌方有约定），低于最低价会违约或被锁价。
3. **库存 < 14 天还在降价**：库存本来就快断货了，降价只会让断货来得更快。断货掉排名，恢复要花数倍广告费——这时候应该稳价甚至提价，让库存撑到下一批货到。

## 实操要点

- **改价前先建一张"新价格利润表"**：把新价格下的 VAT、佣金、FBA、净收益、利润率、BE ACOS 全部算出来，确认每一项都OK 再动手。
- **重点看 BE ACOS 和当前 ACOS 的关系**：BE ACOS < 当前 ACOS → 先别改价，先想好广告怎么调。
- **库存低于 14 天不降价**：宁可少卖点，也不能加速断货。
- **利润率 10% 是底线**：低于这个数，宁可不动价格，去找别的办法（降成本、提转化、优化广告）。
- **改价后 24-72 小时盯数据**：看真实利润、广告 ACOS、销量变化是否符合预期，不符合就复盘（参考入门6的决策复盘）。

## 动手练习

你有一个产品，当前售价 €11.99（DE 19% VAT），利润率 16%，BE ACOS 16%，当前广告 ACOS 13%。现在竞品降价，老板让你也降 €1，新售价 €10.99。

请算：
1. 新售价下的 VAT 是多少？
2. 新佣金是多少？
3. 如果净收益下降后，新利润率掉到 11%，那新 BE ACOS 大约是多少？（粗算：BE ACOS ≈ 利润率，所以约 11%）
4. 当前广告 ACOS 13% 和新 BE ACOS 11% 比——广告还安全吗？

如果你算出来"不安全"，那你的回答应该是："先别急着改价，改了广告会亏，我先把广告 ACOS 调到 11% 以下再改价。"这就是"原子操作"的思维。
