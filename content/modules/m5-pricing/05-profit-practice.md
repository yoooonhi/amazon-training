# 6.5 利润手算实操

> **入门6：定价与利润**
> **本课目标**：拿一个真实的 SKU，从含税售价一路手算到净利润、利润率、BE ACOS，验证你是不是真的把前 4 节课学通了。

## 这节课解决什么问题

前面 4 节课讲了利润链、VAT、佣金基数、原子操作、促销成本——但"听懂"和"会算"是两回事。很多人理论能背，一拿到真实数字就手忙脚乱：VAT 怎么剥？广告怎么摊到每单？BE ACOS 怎么算？分母用含税还是未税？

这节课是**入门 5 的毕业考试**：给你一组真实数据，你必须从售价一路算到净利润、利润率、BE ACOS，并判断当前广告是否安全。**算对到 1 个百分点以内，才算过关。**

如果你卡在某一步，说明对应的那节课没学透，回去重学再回来。

## 核心知识

### 题目：这个 SKU 赚不赚钱？

**已知条件（一个德国站的 SKU，最近 30 天数据）：**

| 项目 | 数值 |
|---|---|
| 售价（含 19% VAT） | €11.99 |
| 佣金率 | 15% |
| FBA 配送费 | €2.50 |
| 数字服务费率 | 2% |
| 产品成本 | €2.00 / 件 |
| 头程运费（分摊） | €0.80 / 件 |
| 退货成本（按 5% 退货率分摊） | €0.30 / 件 |
| 30 天广告花费 | €120 |
| 30 天广告销售额 | €800 |
| 30 天总销售额 | €1200 |

**要算出来的：**

1. 未税价、VAT 金额
2. 佣金（注意基数）
3. 数字服务费
4. 净收益（Net Proceeds）
5. 扣完成本后的"广告前利润 / 件"
6. 销量、广告花费分摊到每单
7. 净利润 / 件、总净利润
8. 利润率
9. BE ACOS
10. 当前 ACOS，和 BE ACOS 比，广告安不安全

### 一步步算

**第 1 步：剥 VAT**

```
未税价 = €11.99 ÷ 1.19 = €10.08
VAT 金额 = €11.99 − €10.08 = €1.91
```

**第 2 步：佣金（按含税价，这是新手最容易算错的一步）**

```
佣金 = €11.99 × 15% = €1.80
```

（不是 €10.08 × 15% = €1.51——按未税价算会少算 €0.29）

**第 3 步：数字服务费（按含税价）**

```
数字服务费 = €11.99 × 2% = €0.24
```

**第 4 步：净收益（Net Proceeds）**

```
净收益 = 售价 − VAT − 佣金 − FBA − 数字服务费
      = 11.99 − 1.91 − 1.80 − 2.50 − 0.24
      = €5.54
```

这 €5.54 是你"用来覆盖成本和广告"的钱。

**第 5 步：广告前利润 / 件（扣完成本和头程、退货）**

```
广告前利润 = 净收益 − 产品成本 − 头程 − 退货
          = 5.54 − 2.00 − 0.80 − 0.30
          = €2.44 / 件
```

这 €2.44 是"如果不投广告，每单净赚多少"。但广告是必须的，所以继续往下扣。

**第 6 步：销量 + 广告分摊到每单**

```
销量 = 总销售额 ÷ 售价 = €1200 ÷ €11.99 ≈ 100 件
广告花费 / 件 = €120 ÷ 100 = €1.20 / 件
```

**第 7 步：净利润 / 件、总净利润**

```
净利润 / 件 = 广告前利润 − 广告分摊 = 2.44 − 1.20 = €1.24 / 件
总净利润 = €1.24 × 100 = €124
```

**这个 SKU 每件净赚约 €1.24，30 天总净赚约 €124。**

**第 8 步：利润率**

```
利润率 = 净利润 ÷ 售价 = €1.24 ÷ €11.99 ≈ 10.3%
```

（如果按未税价算 = €1.24 ÷ €10.08 ≈ 12.3%，口径不同，但全程要统一）

**结论**：利润率 10.3%，刚踩在 10% 生存线上，属于"赚是赚，但很薄"的状态。

**第 9 步：BE ACOS（盈亏平衡 ACOS）**

```
BE ACOS = 广告前利润 ÷ 售价 = €2.44 ÷ €11.99 ≈ 20.3%
```

含义：如果广告 ACOS 低于 20.3%，广告在赚钱；高于 20.3%，广告在亏钱。

**第 10 步：当前 ACOS vs BE ACOS**

```
当前 ACOS = 广告花费 ÷ 广告销售额 = €120 ÷ €800 = 15%
BE ACOS = 20.3%
当前 ACOS 15% < BE ACOS 20.3% → 广告在赚钱，安全
安全垫 = 20.3% − 15% = 6.3 个百分点
```

**最终结论**：这个 SKU 是赚钱的，每件净赚约 €1.24，利润率约 10.3%（偏薄），广告 ACOS 15% 低于 BE ACOS 20.3%，广告安全，但安全垫只有 5 个百分点，经不起太大波动。

## 实操要点

- **每一步都要写出算式**，不要跳步、不要心算。跳步是算错的根源。
- **VAT 用除法剥**：÷(1+税率)，不是乘、不是减。
- **佣金和数字服务费都用含税价**做基数。
- **广告分摊到每单**：总花费 ÷ 销量。销量 = 总销售额 ÷ 售价。
- **BE ACOS 用"广告前利润"做分子**，不是净利润——因为 BE ACOS 回答的是"广告最多能花到多少才不亏"，所以分子必须是没扣广告的利润。
- **全程统一口径**：要么都用含税，要么都用未税。上面这套用的是含税售价做分母，所以全程含税。
- **算完要会解读**：利润率 10.3% 是薄利、广告安全垫 5pp 不算厚——这种产品要盯紧，一有波动（涨价、退货率上升、广告ACOS上升）就可能转亏。

## 动手练习

**这是入门 5 的毕业练习，必须独立完成，算对到 1 个百分点以内。**

换个产品自己算一遍。已知一个法国站（FR，VAT 20%）的 SKU：

| 项目 | 数值 |
|---|---|
| 售价（含 20% VAT） | €14.99 |
| 佣金率 | 15% |
| FBA 配送费 | €2.80 |
| 数字服务费率 | 2% |
| 产品成本 | €2.50 / 件 |
| 头程运费 | €0.90 / 件 |
| 退货成本 | €0.35 / 件 |
| 30 天广告花费 | €150 |
| 30 天广告销售额 | €900 |
| 30 天总销售额 | €1500 |

请独立算出：未税价、VAT、佣金、数字服务费、净收益、广告前利润、销量、广告/件、净利润/件、利润率、BE ACOS、当前 ACOS、广告是否安全。

算完对照关键数（自己算完再看）：
- 未税价 €12.49，VAT €2.50
- 佣金 €2.25（含税价 €14.99 × 15%）
- 数字服务费 €0.30
- 净收益 €7.14（= 14.99 − 2.50 − 2.25 − 2.80 − 0.30）
- 广告前利润 €3.39（= 7.14 − 2.50 − 0.90 − 0.35）
- 销量约 100 件，广告分摊 €1.50 / 件
- 净利润 / 件 €1.89，利润率约 12.6%
- BE ACOS 约 22.6%，当前 ACOS 16.7%——**广告安全**（当前 < BE，安全垫约 6 个百分点）

注意：这个产品利润率 12.6% 比例题还高一点，广告也安全。但它和例题一样属于"薄利 + 安全垫不大"的产品，一有波动（退货率上升、广告 ACOS 上升、被迫降价）就可能转亏——这就是为什么利润要持续盯。

如果你算的每一项都在 ±1 个百分点以内，恭喜，入门 5 你毕业了。
