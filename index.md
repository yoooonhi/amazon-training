## 怎么用这个站

1. **按顺序学**——入门1是底线（合规与账号安全），必须先过。然后按 入门2→3→4→5→6→7 的顺序走。
2. **学完一个就在真实产品上用**——每个单元最后一节是实操课，拿真实产品练。
3. **不赶进度**——没学懂就别往下走。每个单元有验收清单，全过了再进下一个。
4. **做自测题**——每课底部有2-3道题，答错了回去重读。

## 入门课 · 7个单元

| 单元 | 主题 | 学完能做什么 |
|---|---|---|
| **1** | 平台认知与账号安全 | 知道什么绝对不能做，账号健康怎么维护 |
| **2** | 选品与采购 | 判断一个产品值不值得做，找供应商、算利润 |
| **3** | Listing搭建基本功 | 从0搭一个合格的Listing，品牌备案+A+，竞品研究 |
| **4** | 库存与FBA物流 | 算补货量、判断货件状态、预防断货 |
| **5** | 广告体系 | 看懂广告数据、做优化建议、新品广告策略、Vine评论 |
| **6** | 定价与利润 | 手算利润链、判断调价影响、Coupon/Deal计算 |
| **7** | 日常运营与判断力 | 每天怎么开工、排优先级、跟卖防御、退货处理、决策复盘 |

## 学习顺序建议

```
入门1（合规底线）→ 必须先过，不碰产品先学规则
    ↓
入门2（选品）→ 学完就知道卖什么、怎么找货
    ↓
入门3（Listing）→ 学完就拿真实产品搭Listing
    ↓
入门4（库存）→ 学完就跟进一个SKU的补货
    ↓
入门5（广告）→ 最长，学完就看真实广告数据做优化
    ↓
入门6（利润）→ 学完就手算一个SKU的真实利润
    ↓
入门7（日常节奏）→ 把前面学的串起来每天用
```

## 三个绝不能犯的错

1. **在没确认数据是否真实/新鲜的情况下就下决策**——Vine订单不是真实销量，过期数据不能用来调价
2. **高不可逆操作（清仓/删货件/大改Listing）没确认就执行**
3. **把不相关的决策混在一起做**——广告问题归广告、定价问题归定价，不要"一起调了吧"
