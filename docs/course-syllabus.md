### 🟢 入门：从零搭建运营基本功（60 节，免费试听）

**学完能做到**：理解亚马逊运营全貌，能独立完成一个产品从选品到上架到发货的基础操作。

| 单元 | 主题 | 核心能力 |
|------|------|---------|
| **入门1 平台认知与账号安全**（8节）| 平台规则、账号健康、合规红线 | 不踩雷，收到违规通知知道怎么处理 |
| **入门2 选品与采购**（6节）| 市场需求分析、竞争评估、利润预筛、找供应商 | 用数据筛掉烂品，找到能做的款 |
| **入门3 Listing搭建**（12节）| A9算法、标题五点、Search Terms、图片、A+、竞品研究 | 独立搭建一套合格 Listing |
| **入门4 库存与FBA**（7节）| FBA逻辑、费用、周转、补货、断货预防 | 能建发货计划，不断货 |
| **入门5 广告体系**（11节）| 广告类型、匹配、指标、ACOS、结构、新品策略 | 能搭起一个广告活动并读懂报表 |
| **入门6 定价与利润**（5节）| 利润链、VAT佣金、调价、Coupon/Deal | 把一笔账从售价算到净利 |
| **入门7 日常运营**（7节）| 每日流程、优先级、数据可信度、跟卖防御、退货处理 | 有标准化的每日工作节奏 |

**5 节实操课**：选品实操 / Listing实操 / 发货实操 / 广告实操 / 利润手算实操

### 🔵 初级：让在售产品卖得更好（21 节，基础会员）

**学完能做到**：对一个已经在卖的产品，系统性地优化广告、Listing、库存、定价，把业绩做上去。

| 模块 | 主题 | 核心交付物 |
|------|------|-----------|
| **初级1 广告优化实战**（6节）| 搜索词五分类、否定词池、分时调价、ACOS压降闭环、SB/SD | 🔑 **否定词池表** + **ACOS诊断树** + **周度优化清单** |
| **初级2 Listing持续优化**（6节）| A/B测试、排名监控、关键词迭代、图片CTR、评论管理 | 🔑 **Listing体检表**（六维度打分） |
| **初级3 库存精细化**（4节）| 补货量模型、备货资金预算、滞销预警、清货决策 | 🔑 **补货计划表**（16列）+ **滞销五级阶梯** |
| **初级4 定价与促销**（5节）| 竞品跟踪、促销节奏、价格弹性、Buy Box防御 | 🔑 **月度促销作战表** + **促销真实成本算法** |

**4 节实操课**：周度优化清单 / Listing体检 / 补货计划 / 月度促销表

---

### 🟡 中级：从单品到多SKU操盘（21 节，进阶会员）

**学完能做到**：不再只盯一个产品，能规划产品线、管理多个 SKU、搭建品牌基础、用数据找增长。

| 模块 | 主题 | 核心能力 |
|------|------|---------|
| **中级1 新品冷启动**（7节）| 选款验证、0-60天作战日历、冷启动四阶段、排名冲刺、首批评论 | 一个新品从 0 做到稳定出单 |
| **中级2 多SKU与产品线**（5节）| 引流/利润/形象款矩阵、新老品配比、SKU精简、生命周期 | 管理一整条产品线而不是单个SKU |
| **中级3 品牌化基础**（5节）| 品牌备案深度、A+高级版、旗舰店、Brand Analytics | 搭建品牌基础设施 |
| **中级4 数据分析**（4节）| Business Reports深读、第三方工具、增长诊断、自建看板 | 用数据定位问题、找到增长点 |

**3 节实操课**：新品0-60天作战计划 / 产品线健康度盘点 / 品牌基础搭建

---

### 🟠 高级：品牌保护与危机求生（19 节，进阶会员）

**学完能做到**：保护品牌不被跟卖假货侵蚀、扩张到多站点、在账号出事时能自救——这是高门槛的"救命级"能力。

| 模块 | 主题 | 核心能力 |
|------|------|---------|
| **高级1 品牌保护**（5节）| Transparency、Project Zero、跟卖打击SOP、品牌资产 | 从源头杜绝跟卖和假货 |
| **高级2 全球扩张**（6节）| 站点差异、FBA/FBM选择、跨境物流、本地化、税务合规 | 把生意从一个站点扩张到多个 |
| **高级3 合规与申诉**（5节）| 账号冻结申诉、侵权处理、视频验证、评论违规 | 🔑 **账号被封时能用POA申诉书自救** |
| **高级4 高阶广告**（4节）| DSP广告、SD再营销、品牌广告组合、全漏斗策略 | 用全漏斗打法把流量效率榨干 |

**4 节实操课**：品牌防御体系 / 新站点扩张评估 / POA申诉书撰写 / 全漏斗流量设计

---

### 🔴 进阶：战略级运营与团队化（20 节，进阶会员）

**学完能做到**：从"运营"升级到"操盘手"——做品类战略、管财务现金流、建团队SOP、应对黑天鹅。

| 模块 | 主题 | 核心能力 |
|------|------|---------|
| **进阶1 团队与SOP**（5节）| 岗位分工、KPI设计、SOP体系、新人30/60/90天培训 | 带团队而不是自己干 |
| **进阶2 财务现金流**（5节）| 资金预算、周转管理、利润盘汇总、现金流预警 | 算清全盘真实利润，不断现金流 |
| **进阶3 战略选品**（5节）| 品类卡位、产品矩阵、护城河四维、1/3年路线图 | 做品类战略而不是单品买卖 |
| **进阶4 风险管理**（5节）| 账号风险体系、库存资金风险、政策应对、黑天鹅预案 | 建立风险预警，不被突发事件打垮 |

**4 节实操课**：团队管理框架 / 财务监控表 / 品类战略总纲 / 风险应急预案
